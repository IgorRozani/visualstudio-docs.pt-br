# <a name="jsgetextensionallowed-function"></a>Função JsGetExtensionAllowed
Retorna um valor que indica se um objeto é ou não extensível.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
STDAPI_(JsErrorCode) JsGetExtensionAllowed(  
   _In_ JsValueRef object,  
   _Out_ bool *value  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `object`  
 O objeto a ser testado.  
  
 `value`  
 Se o objeto é ou não extensível.  
  
## <a name="return-value"></a>Valor de retorno  
 O código `JsNoError` se a operação foi bem-sucedida, caso contrário, um código de falha.  
  
## <a name="remarks"></a>Comentários  
 Exige um contexto de script ativo.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
