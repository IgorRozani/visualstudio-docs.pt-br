# <a name="jsconstructobject-function"></a>Função JsConstructObject
Invoca uma função como um construtor.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
STDAPI_(JsErrorCode) JsConstructObject(  
   _In_ JsValueRef function,  
   _In_reads_(argumentCount) JsValueRef *arguments,  
   _In_ unsigned short argumentCount,  
   _Out_opt_ JsValueRef *result  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `function`  
 A função a invocar como um construtor.  
  
 `arguments`  
 Os argumentos da chamada.  
  
 `argumentCount`  
 O número de argumentos sendo passados para a função.  
  
 `result`  
 O valor retornado ao invocar a função.  
  
## <a name="return-value"></a>Valor de retorno  
 O código `JsNoError` se a operação foi bem-sucedida, caso contrário, um código de falha.  
  
## <a name="remarks"></a>Comentários  
 Exige um contexto de script ativo.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
