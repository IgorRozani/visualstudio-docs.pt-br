# <a name="jspropertyidref-typedef"></a>Typedef JsPropertyIdRef
Um identificador de propriedade.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>Comentários  
 Os identificadores de propriedade são usados para fazer referência às propriedades de objetos JavaScript em vez de usar cadeias de caracteres.  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
