# <a name="jssourcecontext-typedef"></a>Typedef JsSourceContext
Um cookie que identifica um script para fins de depuração.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
typedef DWORD_PTR JsSourceContext;  
```  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
