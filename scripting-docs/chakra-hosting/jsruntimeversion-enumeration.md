# <a name="jsruntimeversion-enumeration"></a>Enumeração JsRuntimeVersion
A versão secundária do tempo de execução.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
enum JsRuntimeVersion;  
```  
  
## <a name="members"></a>Membros  
  
### <a name="values"></a>Valores  
  
|Nome|Descrição|  
|----------|-----------------|  
|`JsRuntimeVersion10`|Crie o tempo de execução com a versão IE10.|  
|`JsRuntimeVersion11`|Crie o tempo de execução com a versão IE11.|  
|`JsRuntimeVersionEdge`|Crie o tempo de execução com a versão mais recente presente no computador em tempo de execução.|  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** jsrt.h
