# <a name="mathtan-function-javascript"></a>Função Math.tan (JavaScript)
Retorna a tangente de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório `radians` argumento é uma expressão numérica que contém um ângulo, medido em radianos.  
  
 O valor de retorno é a tangente do argumento numérico de `radians`.
