# <a name="subtraction-assignment-operator---javascript"></a>Operador de atribuição de subtração (-=) (JavaScript)
Subtrai o valor de uma expressão do valor de uma variável e atribui o resultado à variável.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parâmetros  
 *resultado*  
 Qualquer variável numérica.  
  
 *expressão*  
 Qualquer expressão numérica.  
  
## <a name="remarks"></a>Comentários  
 Usando o `-=` operador é exatamente o mesmo que fazer o seguinte:  
  
```JavaScript  
result = result - expression  
```
