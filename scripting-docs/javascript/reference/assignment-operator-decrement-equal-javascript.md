# <a name="assignment-operator--javascript"></a>Operador de atribuição (=) (JavaScript)
Atribui um valor a uma variável.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parâmetros  
 `result`  
 Qualquer variável.  
  
 `expression`  
 Qualquer expressão numérica.  
  
## <a name="remarks"></a>Comentários  
 O = operador se comporta como outros operadores para expressões que contêm tenham um valor. Isso significa que você pode encadear os operadores de atribuição da seguinte maneira: `j = k = l = 0`. Nesse caso `j`, `k`, e `l` igual a zero.
