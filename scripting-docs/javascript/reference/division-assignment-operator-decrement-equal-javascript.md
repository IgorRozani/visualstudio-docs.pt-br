# <a name="division-assignment-operator---javascript"></a>Operador de atribuição de divisão (-) (JavaScript)
Divide o valor de uma variável pelo valor de uma expressão e atribui o resultado à variável.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>Parâmetros  
 `result`  
 Qualquer variável numérica.  
  
 `expression`  
 Qualquer expressão numérica.  
  
## <a name="remarks"></a>Comentários  
 Usando o `/=` operador é o mesmo que especificar: `result = result / expression`.
