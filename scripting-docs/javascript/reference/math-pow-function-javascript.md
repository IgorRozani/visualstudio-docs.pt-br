# <a name="mathpow-function-javascript"></a>Função Math.pow (JavaScript)
Retorna o valor de uma expressão de base para uma potência especificada.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Parâmetros  
 *base*  
 Necessário. O valor base da expressão.  
  
 *expoente*  
 Necessário. O valor exponencial da expressão.  
  
## <a name="example"></a>Exemplo  
 No exemplo a seguir, uma expressão numérica igual a baseexponent retorna 1000.  
  
```JavaScript  
Math.pow(10,3);  
```
