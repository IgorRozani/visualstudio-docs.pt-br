# <a name="mathceil-function-javascript"></a>Função Math.ceil (JavaScript)
Retorna o menor inteiro maior ou igual ao seu argumento numérico.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Comentários  
 O argumento `number` necessário é uma expressão numérica.  
  
 O valor de retorno é um valor inteiro igual ao menor número inteiro maior ou igual ao seu argumento numérico.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
