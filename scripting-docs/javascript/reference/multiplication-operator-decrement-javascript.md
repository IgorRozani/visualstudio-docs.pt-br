# <a name="multiplication-operator--javascript"></a>Operador de multiplicação (*) (JavaScript)
Multiplica o valor de duas expressões.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
result = number1*number2  
```  
  
## <a name="parameters"></a>Parâmetros  
 *resultado*  
 Qualquer variável.  
  
 `number1`  
 Qualquer expressão.  
  
 `number2`  
 Qualquer expressão.
