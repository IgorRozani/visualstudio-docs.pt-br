# <a name="mathcbrt-function-javascript"></a>Função Math.cbrt (JavaScript)
Retorna a raiz cúbica de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Comentários  
 Se `number` for `NaN`, o resultado será `NaN`.
