# <a name="modulus-operator--javascript"></a>Operador de módulo (JavaScript)
Divide o valor de uma expressão numérica, o valor de outra expressão numérica e produz o restante.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>Arguments  
 `result`  
 Qualquer variável.  
  
 `expression1`  
 Qualquer expressão numérica.  
  
 `expression2`  
 Qualquer expressão numérica.
