# <a name="mathasin-function-javascript"></a>Função Math.asin (JavaScript)
Retorna o arco seno de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório `number` argumento é uma expressão numérica para o qual o arco seno é necessária.  
  
 O valor de retorno é o arco seno do `number` argumento, em radianos.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
