# <a name="tostring-method-map-javascript"></a>Método toString (Map) (JavaScript)
Retorna uma representação de cadeia de caracteres de um mapa.  
  
## <a name="syntax"></a>Sintaxe  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `mapObj`  
 Necessário. Um objeto `Map`.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/valor de retorno  
 A representação de cadeia de caracteres do mapa.
