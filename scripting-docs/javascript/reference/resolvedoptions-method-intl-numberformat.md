# <a name="resolvedoptions-method-intlnumberformat"></a>Método resolvedOptions (Intl.NumberFormat)
Retorna um objeto que contém as propriedades e os valores de `NumberFormat` objeto formatador.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `numberFormatObj`  
 O `NumberFormat` objeto para recuperar informações do.  
  
## <a name="remarks"></a>Comentários  
 As propriedades do objeto retornado correspondem às propriedades computadas do `NumberFormat` objeto.
