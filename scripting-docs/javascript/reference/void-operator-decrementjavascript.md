# <a name="void-operator-javascript"></a>Operador void (JavaScript)
Impede que uma expressão retorne um valor.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Comentários  
 O `expression` argumento é qualquer [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] expressão.  
  
 O `void` operador avalia a expressão e retorna `undefined`. É útil em situações em que uma expressão deve ser avaliada, mas você não deseja que os resultados visíveis para o restante do script.
