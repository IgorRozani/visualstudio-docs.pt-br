# <a name="mathlog1p-function-javascript"></a>Função Math.log1p (JavaScript)
Retorna o logaritmo natural de 1 + um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
Math.log1p(x)   
```  
  
## <a name="return-value"></a>Valor de retorno  
 Se `x` for `NaN`, essa função retornará `NaN`. Se `x` for menor que zero (0), essa função retornará `NaN`.
