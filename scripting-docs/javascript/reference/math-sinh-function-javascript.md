# <a name="mathsinh-function-javascript"></a>Função Math.sinh (JavaScript)
Retorna o seno hiperbólico inverso de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Comentários  
 O valor retornado é o mesmo que `(exp(x) - exp(-x))/2`. Se x for `NaN`, o resultado é `NaN`.
