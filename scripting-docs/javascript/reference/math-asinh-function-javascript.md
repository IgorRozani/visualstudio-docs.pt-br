# <a name="mathasinh-function-javascript"></a>Função Math.asinh (JavaScript)
Retorna o seno hiperbólico inverso de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>Comentários  
 Se `x` for `NaN`, o resultado é `NaN`.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
