# <a name="mathtrunc-function-javascript"></a>Função Math.trunc (JavaScript)
Retorna a parte inteira de um número, removendo os dígitos fracionários.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>Comentários  
 O argumento `number` obrigatório é uma expressão numérica para a qual o inteiro truncado é necessário.  
  
 Se `number` for `NaN`, `NaN` é retornado.
