# <a name="mathrandom-function-javascript"></a>Função Math.random (JavaScript)
Retorna um número pseudorrandômico entre 0 e 1.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Comentários  
 O número pseudoaleatórios gerado é de 0 (inclusivo) a 1 (exclusivo), ou seja, o número retornado pode ser zero, mas sempre será menor do que um. O gerador de número aleatório será propagado automaticamente quando [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] é carregado pela primeira vez.
