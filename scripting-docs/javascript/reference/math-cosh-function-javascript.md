# <a name="mathcosh-function-javascript"></a>Função Math.cosh (JavaScript)
Retorna o cosseno hiperbólico de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Comentários  
 O valor retornado é o mesmo que `(exp(x) + exp(-x))/2`. Se `x` for `NaN`, o resultado é `NaN`.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
