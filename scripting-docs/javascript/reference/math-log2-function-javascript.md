# <a name="mathlog2-function-javascript"></a>Função Math.log2 (JavaScript)
Retorna o logaritmo de base 2 de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
Math.log2(x)   
```  
  
## <a name="return-value"></a>Valor de retorno  
 Se `x` for `NaN`, essa função retornará `NaN`. Se `x` for menor que zero (0), essa função retornará `NaN`.
