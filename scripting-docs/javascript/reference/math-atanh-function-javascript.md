# <a name="mathatanh-function-javascript"></a>Função Math.atanh (JavaScript)
Retorna a tangente hiperbólica inversa de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Comentários  
 Se `x` for `NaN`, o resultado é `NaN`.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
