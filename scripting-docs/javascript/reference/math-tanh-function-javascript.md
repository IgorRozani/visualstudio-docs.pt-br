# <a name="mathtanh-function-javascript"></a>Função Math.tanh (JavaScript)
Retorna a tangente hiperbólica de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>Comentários  
 O valor retornado é o mesmo que `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. Se `x` for `NaN`, o resultado é `NaN`.
