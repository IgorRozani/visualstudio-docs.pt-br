# <a name="mathfloor-function-javascript"></a>Função Math.floor (JavaScript)
Retorna o maior inteiro menor ou igual ao seu argumento numérico.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório *número* argumento é uma expressão numérica.  
  
 O valor de retorno é um valor inteiro igual para o maior inteiro menor ou igual ao seu argumento numérico.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
