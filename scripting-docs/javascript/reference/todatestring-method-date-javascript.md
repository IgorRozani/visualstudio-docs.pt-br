# <a name="todatestring-method-date-javascript"></a>Método toDateString (Date) (JavaScript)
Retorna uma data como um valor de cadeia de caracteres.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório *objDate* referência é um `Date` objeto.  
  
 O `toDateString` facilmente método retorna um valor de cadeia de caracteres que contém a data, no fuso horário atual, em um conveniente, ler o formato.
