# <a name="resolvedoptions-method-intldatetimeformat"></a>Método resolvedOptions (Intl.DateTimeFormat)
Retorna um objeto que contém as propriedades e os valores do objeto formatador de data/hora.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `dateTimeFormatObj`  
 O `DateTimeFormat` objeto para recuperar informações do.  
  
## <a name="remarks"></a>Comentários  
 As propriedades do objeto retornado correspondem às propriedades computadas do `DateTimeFormat` objeto.
