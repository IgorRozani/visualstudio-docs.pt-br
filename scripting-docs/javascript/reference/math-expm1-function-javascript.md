# <a name="mathexpm1-function-javascript"></a>Função Math.expm1 (JavaScript)
Retorna o resultado da subtração de 1 de e (a base de logaritmos naturais) elevado a uma potência.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Comentários  
 O argumento necessário `number` é uma expressão numérica que representa o poder do e.  
  
 O valor retornado é um número. A constante e é o número de Euler, aproximadamente igual a 2,71828, e `number` é o argumento fornecido.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
