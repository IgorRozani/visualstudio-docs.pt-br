# <a name="debugdebuggerenabled-property"></a>Propriedade Debug.debuggerEnabled
Determina se a depuração está habilitada para o contexto do script. Depuração pode ser habilitada ou desabilitada ou não um depurador anexado.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
