# <a name="infinity-constant-javascript"></a>Constante de infinito (JavaScript)
Um número que é maior que o maior número de ponto flutuante. Retorna um valor inicial de `Number.POSITIVE_INFINITY`. Negativo infinito (-infinito) é menor do que o flutuante menor número de ponto.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Comentários  
 O `Infinity` constante é um membro do `Global` de objeto e fica disponível quando o mecanismo de script é inicializado.
