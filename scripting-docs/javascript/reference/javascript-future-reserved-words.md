# <a name="javascript-future-reserved-words"></a>JavaScript Palavras reservadas futuras
Em JavaScript, palavras-chave reservadas futuras não devem ser usadas como identificadores, mesmo que eles tenham nenhum significado especial na versão atual.  
  
 Para obter uma lista de palavras reservadas em JavaScript, consulte [palavras reservadas JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Palavras reservadas futuras  
 A seguir é palavras reservadas futuras:  
  
1.  classe  
  
2.  enum  
  
3.  export  
  
4.  estende  
  
5.  import  
  
6.  Super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Palavras reservadas futuras no modo estrito  
 A seguir é palavras reservadas futuras somente no modo estrito. Para obter mais informações sobre o modo estrito, consulte [diretiva use strict](../../javascript/reference/use-strict-directive.md).  
  
1.  implementa  
  
2.  interface  
  
3.  pacote  
  
4.  particulares  
  
5.  protegidos  
  
6.  públicos  
  
7.  static  
  
8.  yield
