# <a name="clear-method-weakmap-javascript"></a>Método clear (WeakMap) (JavaScript)
Remove todos os elementos de um objeto `WeakMap`.  
  
## <a name="syntax"></a>Sintaxe  
  
```JavaScript  
weakmapObj.clear()  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `weakmapObj`  
 Necessário. O `WeakMap` objeto para limpar.
