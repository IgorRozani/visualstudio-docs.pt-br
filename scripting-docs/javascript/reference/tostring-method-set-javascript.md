# <a name="tostring-method-set-javascript"></a>Método toString (Set) (JavaScript)
Retorna uma representação de cadeia de caracteres de um conjunto.  
  
## <a name="syntax"></a>Sintaxe  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `setObj`  
 Necessário. Um objeto `Set`.  
  
## <a name="property-valuereturn-value"></a>Valor da propriedade/valor de retorno  
 A representação de cadeia de caracteres do conjunto.
