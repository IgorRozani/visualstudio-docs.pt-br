# <a name="mathatan-function-javascript"></a>Função Math.atan (JavaScript)
Retorna o arco tangente de um número.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório `number` argumento é uma expressão numérica para o qual o arco tangente é necessária.  
  
 O valor de retorno é o arco tangente do `number` argumento, em radianos.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
