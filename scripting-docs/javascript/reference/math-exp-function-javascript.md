# <a name="mathexp-function-javascript"></a>Função Math.exp (JavaScript)
Retorna e (a base dos logaritmos naturais) elevado a uma potência.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Comentários  
 Obrigatório `number` argumento é uma expressão numérica que representa a potência de *e*.  
  
 O valor retornado é um número. A constante *e* é de Euler, aproximadamente igual a 2,71828, número e `number` é o argumento fornecido.  
  
 **Aplica-se a**: [objeto Math](../../javascript/reference/math-object-javascript.md)
