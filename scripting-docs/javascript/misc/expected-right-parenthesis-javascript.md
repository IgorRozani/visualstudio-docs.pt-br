# <a name="expected-3939-javascript"></a>Esperado &#39;) &#39; (JavaScript)
Você tentou incluir uma expressão dentro de um conjunto de parênteses, mas não inclui o parêntese de fechamento. Alguma expressão deve ser colocado em um conjunto de abertura e fechamento de parênteses. Observe o uso de parênteses no exemplo a seguir.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione parênteses à direita para a expressão de avaliação.
