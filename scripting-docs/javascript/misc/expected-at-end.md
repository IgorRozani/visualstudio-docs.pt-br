# <a name="expected-39end39"></a>Esperado &#39; @end&#39;
Você tentou criar um bloco compilado condicionalmente de código, mas não inclui o  **@end**  instrução no final. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]instruções podem ser compiladas condicionalmente, colocando-os em uma  **@if/ @end**  bloco.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar correspondente  **@end**  instrução.
