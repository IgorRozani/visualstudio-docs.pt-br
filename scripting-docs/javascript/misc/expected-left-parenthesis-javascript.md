# <a name="expected-3939-javascript"></a>Esperado &#39; (&#39; (JavaScript)
Você tentou incluir uma expressão dentro de um conjunto de parênteses, mas não inclui o parêntese de abertura. Algumas expressões devem ser colocados em um conjunto de abertura e fechamento de parênteses. Observe o uso de parênteses no exemplo a seguir.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o parêntese esquerdo para a expressão de avaliação.
