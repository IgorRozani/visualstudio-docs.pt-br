# <a name="visual-studio-subscriptions"></a>Assinaturas do Visual Studio
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>Ferramentas de desenvolvedor, serviços de nuvem, software, suporte e treinamento

As Assinaturas do Visual Studio são o conjunto mais abrangente de recursos para você criar, implantar e gerenciar grandes aplicativos em suas plataformas e dispositivos preferidos, incluindo Windows, iOS, Android e Linux.

Se for um assinante ou gerenciar assinaturas para sua empresa, você encontrará informações para ajudá-lo a aproveitar ao máximo as assinaturas do Visual Studio.
