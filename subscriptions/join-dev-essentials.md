# <a name="how-to-join-visual-studio-dev-essentials"></a>Como ingressar no Visual Studio Dev Essentials

O Visual Studio Dev Essentials fornece tudo o que você precisa para compilar e implantar seu aplicativo em qualquer plataforma. Com ferramentas de ponta, o poder da nuvem, treinamento e suporte, é o nosso programa gratuito para desenvolvedores mais abrangente de todos os tempos.

Visite [https://www.visualstudio.com/dev-essentials/](https://www.visualstudio.com/dev-essentials/) para saber mais sobre tudo o que está incluído em uma associação ao Visual Studio Dev Essentials. 

Quando estiver pronto para se registrar, basta clicar no botão [Registrar ou acessar agora](https://my.visualstudio.com/Benefits?wt.mc_id=o~msft~vscom~devessentials-hero~mt689&campaign=o~msft~vscom~devessentials-hero~mt689). 

E isso é tudo.
