# Nenhuma atividade de thread a ser mostrada (exibi&#231;&#227;o de threads)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Essa área exibe dados sobre os threads reexibidos no intervalo de tempo atualmente visível.  
  
 Se nenhuma informação será visível, verifique as seguintes configurações:  
  
-   É o nível de zoom alto?  Tente reduzir ou rolar para tornar mais atividade do thread no intervalo.  
  
-   Muitos threads são ocultados?  Nesse caso, tente que mostra todos os threads  
  
-   Se **Apenas Meu Código** estiver selecionado, você só pode exibir dados em seu código.  Tente desmarcar a instalação para verificar se há alguma atividade do thread do sistema.  
  
-   Verifique se a redução de ruído é definida como um limite baixo.
