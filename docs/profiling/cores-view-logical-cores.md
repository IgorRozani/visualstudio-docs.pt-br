# N&#250;cleos l&#243;gicos da exibi&#231;&#227;o de n&#250;cleos
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Cada nome de canal representa um dos cores lógicas existentes no sistema analisado.  A nomeação conta acima de 0 ao número total de cores lógicas.  Por exemplo, um sistema de quatro núcleo contará de núcleo lógico 0 ao núcleo lógico 3.
