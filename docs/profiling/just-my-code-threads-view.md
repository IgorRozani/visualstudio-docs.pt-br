# Apenas meu c&#243;digo (exibi&#231;&#227;o de threads)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Se você selecionar essa opção, você filtrará a pilha de chamadas para mostrar somente seu código mais um nível de funções chamados.  
  
 Ativando essa opção, você poderá reduzir significativamente a complexidade da pilha de chamadas e talvez facilite para diagnosticar um problema específico.  
  
 Em alguns casos, a seleção dessa opção pode filtrar a chamada de bloqueio.  Se você precisar de detalhes completos da pilha de chamadas fazer a determinação, desmarque esta opção para exibir a pilha de chamada inteira.
