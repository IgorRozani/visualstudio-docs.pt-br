# Caixa de di&#225;logo Aviso da Carga de Projeto para &lt;Projectname&gt;
Essa caixa de diálogo aparecerá se as configurações, para projetos abertos no IDE, forem carregados incorretamente.  
  
 **Corrija os erros especificados e recarregue o projeto**  
 Fornece informações sobre a complicação como o caminho ou o nome de arquivo inválido.  Para resolver erros, remova ou substitua quaisquer referências inválidas de páginas de propriedades do seu projeto.  Salve o projeto e recarregue\-o no IDE.  
  
 **Não exibir esta caixa de diálogo novamente na sessão**  
 Quando selecionado, o aviso de carregamento de projeto da caixa de diálogo \<Projectname\> só voltará a aparecer na próxima vez que você abrir o IDE.
