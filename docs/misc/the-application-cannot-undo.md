# O aplicativo n&#227;o pode desfazer.
Esse erro geralmente ocorre quando foi tentada uma operação de desfazer para um item que tenha um Desfazer vinculado a outro arquivo. Se uma ação tiver sido feita desde a modificação vinculada, o Visual Studio não pode reverter a alteração.  
  
### Para corrigir este erro  
  
1.  Reverter a alteração manualmente.
