# Erro MSB3110 (MSBuild)
**Msb3110 \(\): O Assembly '\< assembly \>' tem incompatíveis filename de identidade '\< AssemblyIdentity \>', esperada: '\< arquivo \>'.**  
  
 Esse erro é gerado quando o nome do arquivo assembly obtido da identidade do assembly não corresponde ao nome do arquivo real.  
  
### Para corrigir este erro  
  
-   Corrija o nome do arquivo de assembly para que corresponda a identidade do assembly.
