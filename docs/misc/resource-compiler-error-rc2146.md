# RC2146 de erro do compilador de recurso
VÍRGULA ausente na instrução de LINGUAGEM  
  
 A vírgula separa o idioma principal e secundária linguagem está ausente.  
  
 O **idioma** instrução deve usar a sintaxe a seguir:  
  
 **Idioma** *primary\_language\_ID*,*secondary\_language\_ID*
