# Marca de fim &lt; / tagName &gt; esperado
A marca de fim em um literal de elemento XML não coincide com a marca de início correspondente.  
  
 **ID do erro:** BC31150  
  
### Para corrigir este erro  
  
-   Modifique o marca XML para coincidir com a marca XML de início para o literal de elemento XML final.
