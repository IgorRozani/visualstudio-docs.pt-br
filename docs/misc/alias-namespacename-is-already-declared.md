# Alias &#39;&lt; nomenamespace &gt;&#39; j&#225; est&#225; declarado
Você tentou usar `Imports` para importar um namespace que já está declarado.  
  
 **ID do erro:** BC30572  
  
### Para corrigir este erro  
  
-   Verifique se o namespace que você está tentando importar.
