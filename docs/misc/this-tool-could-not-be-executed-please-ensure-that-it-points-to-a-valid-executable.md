# N&#227;o foi poss&#237;vel executar esta ferramenta. Verifique se ela aponta para um execut&#225;vel v&#225;lido.
Esse erro ocorre geralmente quando você tentar iniciar uma ferramenta externa e a ferramenta não executa.  Este erro pode ser causado porque o arquivo especificado não é um executável ou um arquivo em lotes válido.  
  
### Para corrigir este erro  
  
1.  Verifique se a ferramenta externa é um executável ou um arquivo em lotes válida usando a caixa de diálogo **ExternoFerramentas** disponível no menu de **Ferramentas** .
