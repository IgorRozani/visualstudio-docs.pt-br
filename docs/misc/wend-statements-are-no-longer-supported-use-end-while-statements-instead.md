# N&#227;o h&#225; suporte para instru&#231;&#245;es &#39;Wend&#39;; Use instru&#231;&#245;es &#39;End While&#39; em vez disso
O `End While` Construir substitui o `Wend` palavra\-chave.  
  
 **ID do erro:** BC30809  
  
### Para corrigir este erro  
  
-   Use `End While` para encerrar um `While` bloco.
