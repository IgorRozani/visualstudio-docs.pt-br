# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Reflection.CustomAttributeFormatException
A exceção que é lançada quando o formato binário de um atributo personalizado é inválido.  
  
## Comentários  
 Isso pode ser resultado de um bug em uma ferramenta usada para criar o tipo de dados gerando o erro.
