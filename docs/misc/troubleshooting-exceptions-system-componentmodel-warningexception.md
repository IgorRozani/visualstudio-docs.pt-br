# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ComponentModel.WarningException
Especifica uma exceção que é tratada como um aviso em vez de um erro.  
  
## Comentários  
 Se essa exceção não for detectada, ele é apresentado ao usuário como uma mensagem de aviso.
