# Lista usada mais recentemente pode conter itens de &lt; n&#250;mero &gt;.
Listas MRU podem conter até vinte e quatro itens. Esse erro geralmente ocorre quando um número menor que ou maior que 24 foi inserido no **listas usadas mais recentemente contêm itens \< número \>** caixa no painel geral das opções de ambiente na caixa de diálogo Opções.  
  
### Para corrigir este erro  
  
1.  Insira um número entre uma e vinte e quatro na caixa.
