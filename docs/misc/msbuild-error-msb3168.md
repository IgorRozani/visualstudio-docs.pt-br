# Erro MSB3168 (MSBuild)
**MSB3168: Item duplicado '\< pacote \>' será ignorado.**  
  
 Este aviso ocorre se você tiver especificado que o bootstrapper deve instalar dois pacotes de bootstrapper idênticos. Nesse caso, o instalador instalará apenas uma instância.  
  
### Para corrigir este erro  
  
-   Para obter mais informações sobre a causa e as possíveis resoluções de erro MSBuild, consulte [Recursos adicionais](../msbuild/additional-msbuild-resources.md).
