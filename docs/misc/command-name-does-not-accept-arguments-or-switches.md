# Comando &lt; nome &gt; n&#227;o aceita argumentos nem op&#231;&#245;es.
Esse erro geralmente ocorre quando um argumento ou opção foi inserido com um comando ou um alias que não usa argumentos nem opções. Se um alias foi inserido, o nome completo do comando é exibido no texto da mensagem de erro, não o nome do alias. Para obter uma lista de comandos válidos que use argumentos e opções, consulte [Comandos do Visual Studio](../ide/reference/visual-studio-commands.md).  
  
### Para corrigir este erro  
  
1.  Digite novamente o comando e os argumentos ou insira um novo comando.
