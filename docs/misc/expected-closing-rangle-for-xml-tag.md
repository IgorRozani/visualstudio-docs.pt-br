# Fechamento esperado &#39; &gt;&#39; para a marca XML
A marca XML para um literal XML está faltando o fechamento necessários ' \>' caracteres.  
  
 **ID do erro:** BC31166  
  
### Para corrigir este erro  
  
-   Adicionar o fechamento ' \>' caracteres ao final da marca XML para o literal XML.
