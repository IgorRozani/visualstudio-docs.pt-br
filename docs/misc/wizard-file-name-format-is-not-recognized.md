# Formato de arquivo &lt; nome &gt; do assistente n&#227;o &#233; reconhecido.
O assistente não pode ser aberto. Esse erro geralmente é causado por um arquivo corrompido do assistente ou um assistente que não coincide com o número de versão do Visual Studio.  
  
### Para corrigir este erro  
  
1.  Verifique se o arquivo do assistente tem o mesmo número de versão da versão do Visual Studio listado como o valor de VSWIZARD, como `VSWIZARD 7.0`.  
  
     – ou –  
  
     Reinstale o Visual Studio para reparar o arquivo corrompido.
