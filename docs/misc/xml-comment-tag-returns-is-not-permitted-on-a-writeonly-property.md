# Marca de coment&#225;rio XML &#39;returns&#39; n&#227;o &#233; permitida em uma propriedade &#39;WriteOnly&#39;
Marca de comentário XML 'returns' não é permitida em uma propriedade 'WriteOnly'. O comentário XML será ignorado.  
  
 Um comentário XML para uma declaração de propriedade somente gravação não pode conter uma marca \< retorna \>.  
  
 **ID do erro:** BC42313  
  
### Para corrigir este erro  
  
-   Remova a marca \< retorna \> sem suporte.
