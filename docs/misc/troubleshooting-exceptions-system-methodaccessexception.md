# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.MethodAccessException
Um <xref:System.MethodAccessException> exceção é lançada quando há uma tentativa inválida de acessar um método particular ou protegido dentro uma classe.  
  
## Dicas associadas  
 **Se o nível de acesso de um método em uma biblioteca de classes for alterado, recompile todos os assemblies que fazem referência a biblioteca.**  
 Essa exceção ocorre normalmente quando o chamador não tem permissão de acesso ao membro.
