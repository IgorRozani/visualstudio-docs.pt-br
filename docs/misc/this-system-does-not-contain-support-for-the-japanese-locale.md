# Este sistema n&#227;o tem suporte para a localidade japon&#234;s
Você está tentando usar o `VbStrConv.Japanese` membro de enumeração, mas seu sistema não dá suporte para a localidade japonês.  
  
### Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade japonês ou executar o aplicativo em um computador que ofereça suporte a localidade.
