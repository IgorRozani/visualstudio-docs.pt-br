# O suplemento selecionado n&#227;o foi confirmado como &#39;Seguro para Linha de Comando&#39; e talvez exija alguma interven&#231;&#227;o do usu&#225;rio (poss&#237;vel interface do usu&#225;rio).
Esse erro geralmente ocorre quando um suplemento foi selecionado para ser usado no prompt de comando \(prompt do MS\-DOS\), mas o suplemento não lista em si como seguro para linha de comando. O suplemento pode exibir a interface do usuário que poderia interferir no uso da linha de comando.  
  
### Para corrigir este erro  
  
1.  Do **ferramentas** menu, escolha **Gerenciador de suplementos**.  
  
2.  Em **o Gerenciador de suplementos** caixa de diálogo, desmarque **linha de comando** para o suplemento.
