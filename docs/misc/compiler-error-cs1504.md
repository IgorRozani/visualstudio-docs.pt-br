# CS1504 de erro do compilador
Não foi possível abrir o arquivo de origem 'arquivo' \('razão'\)  
  
 Um arquivo de origem não pode ser aberto ou lido pelo compilador. O arquivo pode estar bloqueado por outro aplicativo ou pode haver algum outro problema de sistema operacional. A mensagem inclui o motivo do sistema operacional porque o compilador não pôde abrir ou ler o arquivo.
