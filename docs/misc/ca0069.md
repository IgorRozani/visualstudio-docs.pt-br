# CA0069
UsingAlternatePlatform.  *FrameworkVersion1* não pôde ser localizado.  Usando *FrameworkVersion2* em vez disso.  Para obter os melhores resultados da análise satisfaça garantem que o .NET Framework correto está instalado.
