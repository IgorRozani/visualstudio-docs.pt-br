# O campo &#39;&lt; fieldname &gt;&#39; &#233; de um tipo sem suporte
Um campo está declarado como um tipo que não pode ser representado por [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
 **ID do erro:** BC30656  
  
### Para corrigir este erro  
  
1.  Defina o campo como um tipo com suporte.  
  
2.  Desenvolva um novo tipo que descreve os dados no campo.
