# Instru&#231;&#245;es de loop n&#227;o s&#227;o v&#225;lidas na janela imediata
`Loop` instruções não podem ser usadas os **imediato** janela.  
  
 **ID do erro:** BC30708  
  
### Para corrigir este erro  
  
-   Remover `Loop` instruções do código de depuração.
