# Nome da guia &lt; nome &gt; j&#225; existe.
Cada guia na caixa de ferramentas deve ter um nome exclusivo. Esse erro geralmente ocorre quando o nome que você inseriu para a guia é o mesmo que uma guia existente na caixa de ferramentas.  
  
### Para corrigir este erro  
  
1.  Insira um nome exclusivo para a guia.
