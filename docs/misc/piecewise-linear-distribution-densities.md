# piecewise_linear_distribution::densities
Retorna as densidades armazenadas para a distribuição.  
  
## Sintaxe  
  
```  
std::vector<double> densities() const;  
```  
  
## Comentários  
 A função de membro retorna os valores de densidade armazenados.  
  
## Requisitos  
 **Cabeçalho:** \< random \>  
  
 **Namespace:** std
