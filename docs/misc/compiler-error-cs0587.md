# CS0587 de erro do compilador
Erro interno do compilador: etapa 'estágio'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se esse não for o caso, tente [Fale conosco](../ide/talk-to-us.md).
