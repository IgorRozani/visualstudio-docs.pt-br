# CS1555 de erro do compilador
Não foi possível encontrar o 'class' especificado para o método Main  
  
 Uma classe especificada para o [\/principal](/dotnet/csharp/language-reference/compiler-options/main-compiler-option) opção de compilador, mas o nome da classe não foi encontrada no código\-fonte.
