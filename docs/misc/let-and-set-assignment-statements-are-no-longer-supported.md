# &#39;Let&#39; e &#39;Set&#39; n&#227;o h&#225; suporte para instru&#231;&#245;es de atribui&#231;&#227;o
`Let` e `Set` não podem ser usados em atribuições.  
  
 **ID do erro:** BC30807  
  
### Para corrigir este erro  
  
-   Omitir `Let` e `Set` das instruções de atribuição; por exemplo:  
  
     `X = 2`
