# N&#227;o foi poss&#237;vel acessar o Registro do sistema.
Esse erro geralmente ocorre quando o sistema operacional não é possível localizar o arquivo de registro, o arquivo está corrompido ou falha na tentativa de ler ou gravar o arquivo.  
  
### Para corrigir este erro  
  
1.  Reinstale o Visual Studio.
