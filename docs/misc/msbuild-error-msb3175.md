# Erro MSB3175 (MSBuild)
**MSB3175: Valor inválido para '\< arquivo \>' do item '\< tarefas de compilação \>'.**  
  
 Esse aviso é gerado quando o processo de compilação não reconhece o valor de várias propriedades de item de tarefa baseada na enumeração como `AssemblyType`, `DependencyType`, `FileType`, ou `TargetZone`. Aplica\-se a manifestos de aplicativo e implantação manifestos.
