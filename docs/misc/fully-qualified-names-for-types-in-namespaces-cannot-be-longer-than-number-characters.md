# Nomes totalmente qualificados para tipos em namespaces n&#227;o podem ter mais de &lt; n&#250;mero &gt; caracteres
O nome totalmente qualificado do tipo não pode ser maior que o número especificado de caracteres.  
  
 **ID do erro:** BC30031  
  
### Para corrigir este erro  
  
-   Diminua o nome do tipo.
