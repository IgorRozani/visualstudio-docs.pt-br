# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Workflow.Activities.Rules.RuleSetValidationException
Um <xref:System.Workflow.Activities.Rules.RuleSetValidationException> exceção é lançada quando o processamento não pode continuar porque um conjunto de regras não pode ser validado. A exceção é lançada pelo <xref:System.Workflow.Activities.Rules.RuleEngine> construtor se o <xref:System.Workflow.Activities.Rules.RuleSet> usado tem erros de validação.
