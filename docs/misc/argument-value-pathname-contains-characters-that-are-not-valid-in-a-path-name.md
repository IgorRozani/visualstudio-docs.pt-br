# Argumento valor &#39;&lt; caminho &gt;&#39; cont&#233;m caracteres que n&#227;o s&#227;o v&#225;lidos em um nome de caminho
Um nome de caminho contém um ou mais caracteres que não são válidos.  
  
### Para corrigir este erro  
  
-   Remova o caractere do nome do caminho.
