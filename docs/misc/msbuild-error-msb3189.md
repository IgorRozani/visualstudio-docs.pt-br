# Erro MSB3189 (MSBuild)
**MSB3189: O local de atualização para este aplicativo é um caminho local.**  
  
 Este aviso só se aplica a manifestos de implantação.  
  
### Para corrigir este erro  
  
-   Certifique\-se de que o local de atualização é uma URL válida.
