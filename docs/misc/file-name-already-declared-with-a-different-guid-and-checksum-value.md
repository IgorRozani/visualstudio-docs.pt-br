# Nome de arquivo j&#225; declarado com um valor GUID e soma de verifica&#231;&#227;o diferente
Foi feita uma tentativa para declarar um arquivo, mas o arquivo já existe com um valor GUID e soma de verificação diferente.  
  
 **ID do erro:** BC42034  
  
### Para corrigir este erro  
  
-   Verifique o nome do arquivo e o caminho para certificar\-se de que cada um está correto.  
  
-   Exclua o arquivo supérfluo se ele não for necessário.
