# CS2011 de erro do compilador
Não é possível abrir arquivo 'arquivo' de resposta  
  
 Foi especificado um arquivo de resposta em uma compilação, mas o compilador não pôde localizar e abrir o arquivo.  
  
 Para obter mais informações sobre arquivos de resposta, consulte [@ \(especificar arquivo de resposta\)](/dotnet/csharp/language-reference/compiler-options/response-file-compiler-option).
