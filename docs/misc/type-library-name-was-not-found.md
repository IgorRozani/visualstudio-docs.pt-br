# Biblioteca de tipos &lt; nome &gt; n&#227;o foi encontrada.
Visual Studio não pode executar sem o DTE.olb e vsext.tlb bibliotecas de tipo. Esse erro geralmente ocorre quando esses dois arquivos foram removidos ou excluídos.  
  
### Para corrigir este erro  
  
1.  Reinstale o Visual Studio.
