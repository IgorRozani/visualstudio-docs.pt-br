# O comando permite no m&#225;ximo argumentos &lt; n&#250;mero &gt;.
Esse erro geralmente ocorre quando há muitos argumentos foram inseridos para um comando ou porque foi usada uma combinação incorreta de argumentos.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
