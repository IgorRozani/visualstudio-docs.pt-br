# CA0055
Não foi possível carregar file:path\<\>  
  
 A ferramenta de análise de código não pôde carregar o arquivo de destino que foi especificado para a análise.  Isso ocorre se o arquivo não for encontrado ou acesso ao arquivo esteve negado.  Em geral, isso ocorre se o usuário optou para substituir a propriedade do assembly de entrada em [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], ou invocou fxcopcmd.exe manualmente.
