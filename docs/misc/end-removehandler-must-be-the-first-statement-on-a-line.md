# &#39;End RemoveHandler&#39; deve ser a primeira instru&#231;&#227;o em uma linha
Um `End RemoveHandler` instrução segue um separador de declaração dois\-pontos \(:\).`End RemoveHandler` deve ser a única declaração na sua linha de código.  
  
 **ID do erro:** BC31119  
  
### Para corrigir este erro  
  
-   Separe as várias declarações em linhas diferentes.
