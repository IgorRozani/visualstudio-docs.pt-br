# PRJ0048 de erro de compila&#231;&#227;o de projeto
Não foi possível iniciar uma compilação porque uma compilação já está em andamento.  
  
 Você não pode iniciar uma compilação se outra compilação está em andamento. Aguarde a compilação atual concluir ou cancelar a compilação
