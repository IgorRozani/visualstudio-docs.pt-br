# &#39;Nothing&#39; n&#227;o pode ser avaliada
O `Nothing` palavra\-chave foi usada na janela de depuração.  
  
 **ID do erro:** BC30701  
  
### Para corrigir este erro  
  
-   Não use `Nothing` durante a depuração.
