# CA0502
Idioma sem suporte  
  
 Esse erro ocorre quando o usuário tentar analisar um projeto ASP.NET que contém uma ou mais páginas ou classes escritas em um idioma diferente do c\# ou Visual Basic. Para corrigir esse erro, remova o arquivo do projeto ou desativar a análise de código.
