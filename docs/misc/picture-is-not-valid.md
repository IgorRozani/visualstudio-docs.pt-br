# Imagem n&#227;o &#233; v&#225;lida
Você tentou atribuir um formato gráfico diferente de um bitmap, ícone ou metarquivo do Windows para o `Picture` propriedade de um formulário ou controle.  
  
### Para corrigir este erro  
  
1.  Certifique\-se de que o arquivo que você está tentando carregar para o `Picture` propriedade é um arquivo de gráficos válido com suporte pelo [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)].
