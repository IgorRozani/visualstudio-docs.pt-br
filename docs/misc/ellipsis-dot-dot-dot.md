# Retic&#234;ncias (...)
O operador de reticências \(`...`\) tem muitos usos em C e C\+\+, dependendo do contexto.  Usado originalmente no C como um declarador abstrato para listas de parâmetros de função variável, também é usado na programação de macros de pré\-processador na forma de macros variadic.  No C\+\+, `...` é usado em blocos `catch` de tratamento de exceções. No C\+\+11, é usado para modelos variadic.
