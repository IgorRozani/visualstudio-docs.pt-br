# Caixa de di&#225;logo Apagar Propriedades Substitu&#237;das
> [!NOTE]
>  Essa caixa de diálogo não tem suporte no [!INCLUDE[vs_dev10_long](../code-quality/includes/vs_dev10_long_md.md)].  
  
 Quando você adiciona uma folha de propriedades definida pelo usuário \(arquivo .props\) à configuração de um projeto, tal configuração herda todas as propriedades definidas na folha de propriedades por padrão.  Use esta caixa de diálogo para limpar propriedades herdadas indesejadas para uma configuração de um projeto.
