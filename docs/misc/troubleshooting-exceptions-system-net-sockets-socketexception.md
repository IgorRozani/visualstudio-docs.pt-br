# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Net.Sockets.SocketException
Um <xref:System.Net.Sockets.SocketException> exceção é lançada pela <xref:System.Net.Sockets.Socket> e <xref:System.Net.Dns> classes quando ocorre um erro com a rede.  
  
## Dicas associadas  
 **Verifique a propriedade Errorcode para determinar por que ocorreu o erro de soquete.**  
 O construtor padrão para o <xref:System.Net.Sockets.SocketException> classe conjuntos de <xref:System.Net.Sockets.SocketException.ErrorCode%2A> propriedade para o último erro de soquete do sistema operacional que ocorreu.
