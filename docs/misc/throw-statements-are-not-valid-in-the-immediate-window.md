# Instru&#231;&#245;es &#39;Throw&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Throw` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30725  
  
### Para corrigir este erro  
  
-   Remover `Throw` instruções do código de depuração.
