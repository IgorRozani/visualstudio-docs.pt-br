# &#39;&lt; nome &gt;&#39; n&#227;o pode expor o tipo &lt; type1 &gt; no &lt; especificador &gt;&lt; type2 &gt; &#39;&lt; typename &gt;&#39; por meio de &lt; nome &gt;
Você tentou expor um tipo privado fora da classe pública `MyClass`.  
  
 **ID do erro:** BC30508  
  
### Para corrigir este erro  
  
-   Declare o tipo `Public`.
