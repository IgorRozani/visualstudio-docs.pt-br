# Compilador CS2029 de aviso (n&#237;vel 1)
Valor inválido para '\/define'; 'identifier' não é um identificador válido  
  
 Este aviso ocorre se o valor que é usado no [\/define](/dotnet/csharp/language-reference/compiler-options/define-compiler-option) opção tem caracteres inválidos.  
  
 Esse aviso não pode ser suprimido pelo [\/nowarn](/dotnet/csharp/language-reference/compiler-options/nowarn-compiler-option) opção.
