# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IO.PathTooLongException
Um <xref:System.IO.PathTooLongException> exceção é lançada quando um nome de caminho ou nome de arquivo é maior que o comprimento máximo definido pelo sistema.  
  
## Dicas associadas  
 **Verifique se que o caminho não é maior que o máximo definido pelo sistema.**  
 Nas plataformas baseadas em Windows, caminhos devem ter menos de 248 caracteres e nomes de arquivo devem ter menos de 260 caracteres.
