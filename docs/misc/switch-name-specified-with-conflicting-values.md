# Op&#231;&#227;o &lt; nome &gt; especificada com valores conflitantes.
Esse erro ocorre quando você insere um comutador duas vezes na janela de comando e o valor especificado para a opção é diferente em cada instância.  
  
### Para corrigir este erro  
  
-   Digite novamente o comando, mas usar cada opção apenas uma vez.
