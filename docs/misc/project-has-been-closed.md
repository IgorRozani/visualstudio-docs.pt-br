# Projeto foi fechado
Ocorreu um erro de acesso do modelo de código dentro de [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador.  
  
 **ID do erro:** BC32301  
  
### Para corrigir este erro  
  
1.  Compile o programa novamente para ver se o erro persiste.  
  
2.  Se o erro persistir, reinstale o [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador.  
  
3.  Se o erro persistir após a reinstalação, colete informações sobre as circunstâncias e notifique o Microsoft Product Support Services.
