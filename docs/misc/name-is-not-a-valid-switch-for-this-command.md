# &lt; nome &gt; n&#227;o &#233; uma op&#231;&#227;o v&#225;lida para este comando.
Esse erro geralmente ocorre quando uma chave incorreta ou combinação de opções de caractere único foi inserida para um comando.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
