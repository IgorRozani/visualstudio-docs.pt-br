# Um arquivo inv&#225;lido foi especificado para /resetsettings: &lt;filename&gt;. /resetsettings ser&#225; ignorado.
Este erro geralmente ocorre quando o arquivo de configurações não podem ser lido, não existe ou contém XML mal formado.  
  
### Para corrigir este erro  
  
1.  Verifique se o nome do arquivo está correto.  
  
2.  Verifique se o arquivo está no formato correto.  
  
3.  Verifique se o arquivo contém XML formado corretamente.
