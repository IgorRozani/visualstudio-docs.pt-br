# Erro MSB2007 (MSBuild)
**Visual Studio não pode localizar o arquivo de projeto "{0}".**  
  
 [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] não foi possível localizar o arquivo de projeto especificado.  
  
### Para corrigir este erro  
  
1.  Verifique a ortografia do nome do arquivo de projeto.
