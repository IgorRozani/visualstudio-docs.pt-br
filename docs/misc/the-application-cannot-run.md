# N&#227;o &#233; poss&#237;vel executar o aplicativo.
Não há memória suficiente para Visual Studio executar.  
  
### Para corrigir este erro  
  
1.  Verifique se o computador atende os requisitos mínimos para executar o Visual Studio.  
  
2.  Feche outros programas em execução na máquina para liberar memória.  
  
     \- ou \-  
  
     Reinstale o Visual Studio.
