# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ServiceModel.PoisonMessageException
Um <xref:System.ServiceModel.PoisonMessageException> exceção é lançada quando uma mensagem é considerada "veneno". Uma mensagem é considerada veneno se várias tentativas de entregar a mensagem de falham.
