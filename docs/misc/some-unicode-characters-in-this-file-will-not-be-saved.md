# Alguns caracteres Unicode neste arquivo n&#227;o ser&#227;o salvos.
Esse erro geralmente ocorre quando um arquivo contém caracteres que só podem ser salvo, aplicando uma página de código Unicode.  
  
### Para salvar o arquivo com caracteres Unicode  
  
1.  Clique em **Cancelar**.  
  
2.  Do **arquivo** menu, escolha **Opções de salvamento avançadas**.  
  
3.  No **Opções de salvamento avançadas** caixa de diálogo, escolha a codificação apropriada para seu arquivo.
