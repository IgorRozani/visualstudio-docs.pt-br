# Diret&#243;rio ou arquivo especificado em Examinar n&#227;o est&#225; dispon&#237;vel ou n&#227;o existe.
Esse erro geralmente ocorre quando o nome de caminho ou arquivo especificado na lista de pesquisa não pode ser encontrado.  
  
### Para corrigir este erro  
  
1.  Digite o caminho e o nome novamente.  
  
     – ou –  
  
2.  Navegue até o arquivo correto ou pasta usando a lista de pesquisa e **itens disponíveis** caixa.
