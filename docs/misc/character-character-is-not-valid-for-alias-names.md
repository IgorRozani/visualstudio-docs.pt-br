# Caractere &lt; caractere &gt; n&#227;o &#233; v&#225;lido para nomes de alias.
Esse erro geralmente ocorre quando um caractere incorreto for inserido como parte de um nome de alias. Nomes de alias não pode conter espaços em branco, \=,, \/, ou ".  
  
### Para corrigir este erro  
  
1.  Digite novamente o nome do alias, omitindo o caractere incorreto.
