# Valor da propriedade n&#227;o &#233; v&#225;lido. (Visual Studio) (2)
Esse erro geralmente ocorre quando um valor inválido inserido para uma propriedade. Por exemplo, inserir um formato incorreto para um valor inteiro ou muito grande de um valor para um valor curto.  
  
### Para corrigir este erro  
  
1.  Insira um valor válido para a propriedade.  
  
    > [!TIP]
    >  Pesquise a documentação online para saber quais valores são válidos para a propriedade.
