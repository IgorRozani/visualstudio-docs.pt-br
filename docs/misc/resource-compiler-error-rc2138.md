# RC2138 de erro do compilador de recurso
comentário de fechamento não correspondente ' \/ \*'  
  
 Uma seqüência de comentário de abertura \(`/*`\) não foi correspondida com uma seqüência de comentário de fechamento \(`*/`\) Este erro pode ser causado pelo aninhamento de comentários.
