# &#39;&lt; nome &gt;&#39; n&#227;o pode expor o tipo delegado subjacente &#39;&lt; delegatetype &gt;&#39; do evento que est&#225; implementando para &lt; specifier1 &gt; &#39;&lt; type1 &gt;&#39; por meio de &lt; specifier2 &gt; &#39;&lt; type2 &gt;&#39;
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30914  
  
### Para corrigir este erro  
  
-   Declare o tipo `Public` ou use outro tipo.
