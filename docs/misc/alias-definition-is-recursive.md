# A defini&#231;&#227;o do alias &#233; recursiva.
Esse erro geralmente ocorre quando um alias foi definido que direta ou indiretamente faz referência a mesmo.  
  
### Para corrigir este erro  
  
1.  Altere a definição do alias e tente novamente.  
  
     – ou –  
  
2.  Examine a lista atual de aliases e suas definições digitando `>alias` no **comando** janela e tente novamente.
