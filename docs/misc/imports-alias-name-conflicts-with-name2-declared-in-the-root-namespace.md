# Importa&#231;&#245;es alias &#39;&lt; nome &gt;&#39; est&#225; em conflito com &#39;&lt; nome2 &gt;&#39; declarado no namespace raiz
O nome de alias usado no `Imports` instrução está em conflito com outro nome declarado no namespace raiz.  
  
 **ID do erro:** BC31403  
  
### Para corrigir este erro  
  
-   Escolha um nome exclusivo para o alias usado pelo `Imports` instrução.
