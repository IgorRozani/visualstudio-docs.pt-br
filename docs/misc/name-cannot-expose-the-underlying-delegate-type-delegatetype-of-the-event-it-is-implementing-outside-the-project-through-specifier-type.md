# &#39;&lt; nome &gt;&#39; n&#227;o pode expor o tipo delegado subjacente &#39;&lt; delegatetype &gt;&#39; do evento que est&#225; implementando fora do projeto por meio do especificador de &lt; &gt; &#39;&lt; type &gt;&#39;
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30915  
  
### Para corrigir este erro  
  
-   Declare o tipo `Public`.  
  
     \- ou \-  
  
-   Use outro tipo.
