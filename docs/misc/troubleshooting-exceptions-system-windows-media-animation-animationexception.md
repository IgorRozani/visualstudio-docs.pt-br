# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Windows.Media.Animation.AnimationException
Um <xref:System.Windows.Media.Animation.AnimationException> exceção é lançada quando ocorre um erro enquanto uma propriedade está sendo animada.  
  
## Comentários  
 Essa classe não é usada para erros relacionados à temporização, como a criação de uma animação com um negativo <xref:System.Windows.Media.Animation.Timeline.Duration%2A> ou inválido <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> e <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> as configurações.
