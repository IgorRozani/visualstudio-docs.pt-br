# Falha ao carregar o pacote &lt; nome &gt;.
Não foi possível abrir o pacote. Esse erro geralmente ocorre quando o pacote não foi registrado corretamente, o pacote está faltando uma DLL, a DLL está corrompida ou está faltando uma DLL que requer o pacote de outro componente.  
  
### Para corrigir este erro  
  
1.  Reinstale o produto que fornece o pacote.
