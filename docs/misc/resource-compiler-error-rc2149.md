# RC2149 de erro do compilador de recurso
constante numérica esperada na tabela de cadeia de caracteres  
  
 Uma constante numérica, definido em um `#define` instrução, deve seguir imediatamente o **começar** palavra\-chave em uma **STRINGTABLE** instrução.
