# CA0503
A propriedade é substituída.  Use a propriedade de substituição  
  
 CA0503 de aviso será gerada quando uma propriedade substituído para análise de código contém um valor no arquivo de configuração de criação ou do projeto.  Para resolver esse aviso, use a propriedade especificada que o substitui.
