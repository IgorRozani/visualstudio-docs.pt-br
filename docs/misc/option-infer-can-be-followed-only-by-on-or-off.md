# &#39;Option Infer&#39; pode ser seguido somente por &#39;On&#39; ou &#39;Off&#39;
`Option Infer` instruções devem especificar um `On` ou `Off`. Outros valores não são válidos.  
  
 **ID do erro:** BC30979  
  
### Para corrigir este erro  
  
-   Especifique o `Option Infer On` ou `Option Infer Off`.
