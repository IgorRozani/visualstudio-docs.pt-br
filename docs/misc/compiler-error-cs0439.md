# CS0439 de erro do compilador
Declaração de alias externo deve preceder todos os outros elementos definidos no namespace  
  
 Esse erro ocorre quando um `extern` declaração vem depois de algo, como um `using` declaração, no mesmo namespace. O `extern` declarações devem vir antes de todos os outros elementos do namespace.  
  
## Exemplo  
 O exemplo a seguir gera CS0439:  
  
```  
// CS0439.cs using System; extern alias MyType;   // CS0439 // To resolve the error, make the extern alias the first line in the file. public class Test { public static void Main() { } }  
```
