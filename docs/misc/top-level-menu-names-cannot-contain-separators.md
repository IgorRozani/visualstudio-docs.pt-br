# Nomes de menus de n&#237;vel superior n&#227;o podem conter separadores.
Esse erro geralmente ocorre quando é feita uma tentativa de arrastar texto que contém um caractere separador, como um traço \(\-\), em um nome de menu. Menus de nível superior não podem conter Este caractere, que é reservado para separar os conjuntos de itens de menu.  
  
### Para corrigir este erro  
  
1.  Insira o texto sem usar o caractere separador.
