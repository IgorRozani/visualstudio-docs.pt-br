# &#39;&lt; especificador &gt;&#39; n&#227;o &#233; v&#225;lido em uma declara&#231;&#227;o WithEvents
Um `Dim` instrução contém uma palavra\-chave, como `ReadOnly`, que é inválido em combinação com o `WithEvents` palavra\-chave.  
  
 **ID do erro:** BC30234  
  
### Para corrigir este erro  
  
1.  Remova a palavra\-chave inválida do `Dim` instrução.
