# Pol&#237;tica de coment&#225;rios do Visual Studio
Agradecemos seus comentários. As equipes de desenvolvimento usam seus comentários para aprimorar a documentação. Enquanto estivermos examinando seus comentários, poderemos enviar que email para solicitar esclarecimentos ou comentários em uma solução. Não podemos usar seu endereço de email para nenhuma outra finalidade e será excluído após o encerramento da revisão. Para obter mais informações sobre as políticas de privacidade da Microsoft, consulte [http:\/\/privacy.microsoft.com\/default.aspx](http://privacy.microsoft.com/default.aspx).
