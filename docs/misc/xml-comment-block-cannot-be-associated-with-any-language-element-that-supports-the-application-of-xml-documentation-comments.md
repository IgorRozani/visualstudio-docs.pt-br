# Bloco de coment&#225;rio XML n&#227;o pode ser associado a qualquer elemento de linguagem que ofere&#231;a suporte a aplica&#231;&#227;o de coment&#225;rios de documenta&#231;&#227;o XML
Bloco de comentário XML não pode ser associado a qualquer elemento de linguagem válido. O comentário XML será ignorado. Esse erro também ocorre se uma marca XML no bloco de comentário não tem nenhum elemento de linguagem correspondente.  
  
 **ID do erro:** BC42312  
  
### Para corrigir este erro  
  
-   Localizar e remova a marca XML estranhas ou fornecer o elemento de linguagem que ele descreve.
