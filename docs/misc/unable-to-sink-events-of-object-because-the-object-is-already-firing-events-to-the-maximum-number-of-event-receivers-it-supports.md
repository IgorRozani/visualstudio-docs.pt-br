# N&#227;o &#233; poss&#237;vel coletar eventos de objeto porque o objeto j&#225; est&#225; acionando eventos para o n&#250;mero m&#225;ximo de receptores de evento que ele suporta
O número máximo de receptores de eventos suportado pelo objeto foi excedido.  
  
### Para corrigir o erro  
  
-   Reduza o número de receptores de eventos.
