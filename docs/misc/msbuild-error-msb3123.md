# Erro MSB3123 (MSBuild)
**MSB3123: O número de associações de arquivo excede o limite de limite de \< \>.**  
  
 Cada arquivo é permitido somente um número fixo de extensões de nome de arquivo associado. Esse erro ocorre quando você tenta associar extensões de nome de arquivo mais que o número permitido para o sistema operacional de destino.  
  
### Para corrigir este erro  
  
-   Limite o número de associações de arquivo para o número especificado.
