# O nome de arquivo &lt;name&gt; n&#227;o pode conter nomes de dispositivos de caracteres.
Esse erro geralmente ocorre quando um nome de arquivo é inserido que é realmente um nome de dispositivo de caracteres.  Os nomes de dispositivo de caractere incluem, mas não estão limitados a, CON, PRN, AUX, CLOCK$, NUL, LPT1, LPT2, COM1, COM2, COM3, e COM4.  
  
### Para corrigir este erro  
  
1.  Digite um nome de arquivo diferente.
