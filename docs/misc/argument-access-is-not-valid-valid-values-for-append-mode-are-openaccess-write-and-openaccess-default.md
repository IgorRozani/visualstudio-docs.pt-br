# O argumento &#39;Access&#39; n&#227;o &#233; v&#225;lido. Os valores v&#225;lidos para o modo Append s&#227;o &#39;Write&#39; e &#39;OpenAccess. Default&#39;
Um valor de acesso não é válido no modo de acréscimo.  
  
### Para corrigir este erro  
  
1.  Altere o valor para `OpenAccess.Write` ou `OpenAccess.Default`.
