# Sistema operacional n&#227;o preenche os requisitos para objeto com identifica&#231;&#227;o de classe &lt; n&#250;mero &gt;.
Esse erro geralmente ocorre quando um sistema operacional incompatível ou foi detectada uma versão incompatível do sistema operacional.  
  
### Para corrigir este erro  
  
1.  Verifique os requisitos de sistema para o aplicativo.  
  
2.  Instalar um dos sistemas operacionais compatíveis e tente novamente.
