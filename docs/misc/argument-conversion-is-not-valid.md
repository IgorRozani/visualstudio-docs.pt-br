# O argumento &#39;Conversion&#39; n&#227;o &#233; v&#225;lido
Um valor para o `VbStrConv` enumeração não é válida, possivelmente porque ele é menor que zero. Valores de enumeração devem ser maior ou igual a zero.  
  
### Para corrigir este erro  
  
1.  Certifique\-se de que o valor é maior que ou igual a zero.
