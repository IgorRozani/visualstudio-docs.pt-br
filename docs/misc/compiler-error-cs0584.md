# CS0584 de erro do compilador
Erro interno do compilador: símbolo de estágio estágio' 'symbol'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se não for o caso, tente [Fale conosco](../ide/talk-to-us.md).
