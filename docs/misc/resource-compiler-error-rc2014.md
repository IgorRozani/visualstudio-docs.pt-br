# RC2014 de erro do compilador de recurso
comando do pré\-processador deve começar como primeiro sem espaço em branco  
  
 Caracteres de espaço em branco não apareceram antes do sinal de número \(**\#**\) de uma diretiva de pré\-processador na mesma linha.
