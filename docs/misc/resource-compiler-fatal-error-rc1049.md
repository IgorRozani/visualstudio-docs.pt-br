# RC1049 de Erro Fatal de compilador de recurso
argumento numérico inválido 'string'  
  
 O compilador de recurso espera um argumento numérico, mas recebeu a cadeia de caracteres fornecida.  
  
 Esse erro pode ser causado por fornecendo um número hexadecimal sem o prefixo \\x necessárias ou por um número de ponto flutuante formado incorretamente.
