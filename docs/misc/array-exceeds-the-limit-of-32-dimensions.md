# Matriz excede o limite de 32 dimens&#245;es
Matrizes não podem ter mais de 32 dimensões.  
  
 **ID do erro:** BC30052  
  
### Para corrigir este erro  
  
-   Reduza o número de dimensões na declaração de matriz.
