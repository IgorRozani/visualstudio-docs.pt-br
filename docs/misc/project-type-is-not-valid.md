# Tipo de projeto inv&#225;lido.
Esse erro geralmente ocorre quando um valor incorreto foi inserido para o argumento *templatename* para o **Adicionar novo projeto** comando.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o **Adicionar novo projeto** de comando e tente novamente.
