# Suplemento &lt; nome &gt; causada uma viola&#231;&#227;o de acesso. Remova-o do Gerenciador de suplementos?
Esse suplemento falhou e pode estar causando problemas no ambiente de desenvolvimento integrado.  
  
### Para corrigir este erro  
  
1.  Escolha Sim para remover o suplemento da lista de suplementos disponíveis.  
  
     \- ou \-  
  
     Escolha não.  
  
    > [!NOTE]
    >  Entre em contato com o autor do suplemento para relatar o problema.
