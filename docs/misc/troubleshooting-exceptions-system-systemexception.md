# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.SystemException
Define a classe base para exceções predefinidas no `System` namespace.  
  
## Comentários  
 Essa classe é fornecida como um meio de diferenciar entre as exceções definidas pelo sistema e as exceções definidas por aplicativos.
