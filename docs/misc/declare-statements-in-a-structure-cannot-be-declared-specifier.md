# Instru&#231;&#245;es &#39;Declare&#39; em uma estrutura n&#227;o podem ser declaradas &#39;&lt; especificador &gt;&#39;
Um `Declare` declaração tem um especificador que não é válido dentro de um `Structure` declaração.  
  
 **ID do erro:** BC30791  
  
### Para corrigir este erro  
  
-   Remova o especificador.
