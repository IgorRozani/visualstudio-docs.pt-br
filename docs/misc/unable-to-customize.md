# N&#227;o &#233; poss&#237;vel personalizar.
Esse erro ocorre quando você tenta personalizar menus ou barras de ferramentas ao depurar um aplicativo.  
  
### Para corrigir este erro  
  
-   Aguarde até que você concluiu a depuração para fazer personalizações da barra de ferramentas ou menu.  
  
     – ou –  
  
     Parar a depuração e faça as personalizações da barra de ferramentas ou menu.
