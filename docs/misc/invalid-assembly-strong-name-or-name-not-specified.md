# Nome forte do assembly inv&#225;lido ou nome n&#227;o especificado
A propriedade do projeto no modelo de objeto do sistema de projeto que permite que você especifique um contêiner keyname ou a chave para projetos que serão compilados em assemblies com nome forte \(assinatura de assembly\) estava em branco. A compilação não terá êxito se esta mensagem é exibida.  
  
### Para corrigir este erro  
  
1.  Verifique se um nome de contêiner de chave ou um arquivo de chave é especificado no código.
