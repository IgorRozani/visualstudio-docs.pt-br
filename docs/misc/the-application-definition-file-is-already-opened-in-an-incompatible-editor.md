# O arquivo de defini&#231;&#227;o do aplicativo j&#225; est&#225; aberto em um editor incompat&#237;vel
**O arquivo de definição de aplicativo já está aberto em um editor incompatível. Feche o outro editor e recarregue a página de propriedades do projeto.**  
  
 Esse erro ocorre quando você tenta abrir o arquivo de definição de aplicativo e ele já está aberto em outro editor.  
  
### Para corrigir este erro  
  
1.  Feche o outro editor.  
  
2.  Feche o Designer de projeto.  
  
3.  Sobre o **projeto** menu, clique em **propriedades** reabrir o Designer de projeto.
