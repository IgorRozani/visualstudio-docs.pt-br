# Membros em um m&#243;dulo n&#227;o podem ser declarados &#39;&lt; especificador &gt;&#39;
Você usou um especificador que não é válido em um membro em uma `Module` instrução. Módulos nunca podem ser instanciados, não oferecem suporte a herança e não podem implementar interfaces.  
  
 **ID do erro:** BC30436  
  
### Para corrigir este erro  
  
-   Remova o especificador.
