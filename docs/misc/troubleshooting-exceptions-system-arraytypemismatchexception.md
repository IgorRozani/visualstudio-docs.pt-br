# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ArrayTypeMismatchException
Um <xref:System.ArrayTypeMismatchException> exceção é lançada quando o sistema não pode converter o elemento para o tipo declarado para a matriz.  
  
## Dicas associadas  
 **Verifique se que o tipo de objeto pode ser convertido para o tipo de matriz.**  
 Uma conversão deve ser possível entre os dois tipos.  
  
 Para obter mais informações, os usuários do Visual Basic poderão ver [Matrizes](/dotnet/visual-basic/programming-guide/language-features/arrays/index).
