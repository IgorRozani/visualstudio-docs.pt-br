# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Windows.Automation.ElementNotAvailableException
Um <xref:System.Windows.Automation.ElementNotAvailableException> exceção contém informações sobre a exceção gerada quando é feita uma tentativa de acessar uma interface do usuário <xref:System.Windows.Automation.AutomationElement> corresponde a uma parte da interface do usuário que não está mais disponível.  
  
## Comentários  
 Essa exceção pode ser gerada se o elemento está em uma caixa de diálogo que foi fechada ou em um aplicativo que foi fechado ou encerrado.
