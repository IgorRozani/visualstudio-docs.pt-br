# Erro MSB3041 (MSBuild)
**Não é possível criar um nome de recurso de manifesto para "'\< arquivo \>'".**  
  
 O `CreateManifestResourceName` tarefa não é possível criar um nome de recurso de manifesto para o arquivo especificado.  
  
### Para corrigir este erro  
  
-   Para obter mais informações sobre a causa e as possíveis resoluções de erro MSBuild, consulte [Recursos adicionais](../msbuild/additional-msbuild-resources.md).
