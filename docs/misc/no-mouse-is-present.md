# Nenhum mouse est&#225; presente
Uma das propriedades do `My.Computer.Mouse` objeto foi chamado, mas o computador tem não mouse ou porta de mouse instalado.  
  
### Para corrigir este erro  
  
-   Adicionar um `Try...Catch` em torno da chamada para a propriedade do `My.Computer.Mouse` objeto.  
  
     – ou –  
  
-   Instale um mouse no computador.
