# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Drawing.Printing.InvalidPrinterException
Um <xref:System.Drawing.Printing.InvalidPrinterException> exceção é lançada quando é feita uma tentativa de acessar uma impressora usando configurações de impressora inválidas.  
  
## Dicas associadas  
 **Verifique se a impressora existe.**  
 A causa mais comum de configurações inválidas de impressora está fazendo referência a uma impressora inexistente. Para obter mais informações, consulte <xref:System.Drawing.Printing>.  
  
 **Certifique\-se de que uma impressora padrão foi instalada.**  
 Se nenhuma impressora foi especificada, certifique\-se de que uma impressora padrão foi instalada. Para obter mais informações, consulte <xref:System.Drawing.Printing.PrintDocument.PrinterSettings%2A>.
