# Formato de arquivo n&#227;o &#233; v&#225;lido
Seja você tentou carregar um arquivo em um componente, mas o formato dos dados no arquivo era incompatível com o componente, ou você tentou salvar dados do componente em um arquivo, mas o formato dos dados era incompatível com o formato do arquivo.  
  
### Para corrigir este erro  
  
1.  Consulte a documentação do componente determinar o formato adequado para dados de arquivo de disco.  
  
2.  Consulte a documentação do componente determinar se ele oferece suporte para conversão de um formato para outro.
