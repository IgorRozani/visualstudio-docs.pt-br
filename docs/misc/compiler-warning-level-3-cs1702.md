# Compilador CS1702 de aviso (n&#237;vel 3)
Supondo que a referência de assembly "Nome do Assembly \#1" corresponde a "Nome do Assembly \#2", talvez seja necessário fornecer a diretiva de tempo de execução  
  
 As referências de duas assembly têm diferentes de compilação e\/ou números de revisão, então não será unificar automaticamente. Talvez seja necessário fornecer a diretiva de tempo de execução para forçar a Unificação, usando diretivas no arquivo. config do aplicativo.
