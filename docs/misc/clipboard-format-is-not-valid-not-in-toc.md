# Formato inv&#225;lido de &#225;rea de transfer&#234;ncia.
A área de transferência aceita apenas dados em certos formatos especificados. Esse erro geralmente ocorre quando você tentar colocar os dados de um componente na área de transferência, mas os dados em um formato incompatível com a área de transferência.  
  
### Para corrigir este erro  
  
1.  Consulte a documentação do componente determinar se você pode usar a área de transferência para transferir dados do componente.  
  
     – ou –  
  
2.  Selecione outro item e tente novamente.
