# N&#227;o foi poss&#237;vel carregar o DLL &lt; nome &gt; do sistema.
Um arquivo. dll fornecido pelo sistema operacional, como DDEML. DLL, VERSÃO. DLL ou WINSPOOL. Não foi possível encontrar o DRV. Esse erro geralmente ocorre se uma das seguintes opções for verdadeira: o arquivo não está no caminho correto, a DLL foi corrompida ou excluída ou não há memória suficiente.  
  
### Para corrigir este erro  
  
1.  Verifique se a DLL está no caminho do sistema Windows.  
  
     – ou –  
  
     Recarregue a DLL.  
  
     – ou –  
  
     Tente liberar memória fechando outros aplicativos abertos.
