# V&#237;rgula ou &#39;)&#39; esperado (lista de par&#226;metros de procedimento)
Uma lista de parâmetros contém um elemento inesperado. Use vírgulas \(`,`\) para separar parâmetros na lista e use um parêntese de fechamento \(`)`\) para finalizar a lista.  
  
 **ID do erro:** BC30213  
  
### Para corrigir este erro  
  
-   Adicione uma vírgula e continue a lista de parâmetros de tipo ou adicionar um fechamento de parêntese de abertura para finalizar a lista.
