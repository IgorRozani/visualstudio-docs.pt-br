# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: Microsoft.Office.Tools.ControlNameAlreadyExistsException
Um <xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> exceção é lançada quando você tenta criar programaticamente um controle com o mesmo nome de outro controle no seu projeto.  
  
## Dicas associadas  
 **Use um nome diferente para um dos controles.**  
 Nomes de controle devem ser exclusivos.
