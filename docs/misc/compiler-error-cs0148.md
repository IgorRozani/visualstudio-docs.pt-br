# CS0148 de erro do compilador
O delegado 'delegate' não tem um construtor válido  
  
 Importado e usado um programa gerenciado \(um que usa o .NET Framework common language runtime\) que foi criado com outro compilador. Esse compilador permitido um malformados [Delegar](/dotnet/csharp/language-reference/keywords/delegate) construtor. Para obter mais informações, consulte [Delegados](/dotnet/csharp/programming-guide/delegates/index).
