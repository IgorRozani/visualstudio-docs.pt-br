# Erro MSB1022 (MSBuild)
**Arquivo de resposta não existe.**  
  
 O arquivo de resposta não existe no local especificado.  
  
### Para corrigir este erro  
  
1.  Verifique se o arquivo de resposta existe no diretório especificado, se você tiver especificado o local absoluto ou relativo do arquivo de resposta.
