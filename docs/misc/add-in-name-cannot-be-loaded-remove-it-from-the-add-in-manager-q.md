# Suplemento &lt; nome &gt; n&#227;o pode ser carregado. Remova-o do Gerenciador de suplementos?
O suplemento selecionado não foi encontrado. Esse erro normalmente ocorre quando o suplemento selecionado não tem um arquivo. dll ou outro componente necessário.  
  
### Para corrigir este erro  
  
1.  Reinstale o suplemento.
