# Erro MSB3173 (MSBuild)
**MSB3173: Não é possível gravar o manifesto '\< arquivo \>'. '\< erro \>'**  
  
 Esse erro é gerado quando o processo de compilação encontra um problema geral ao gravar um arquivo de manifesto.  
  
### Para corrigir este erro  
  
-   Para obter mais informações sobre a causa e as possíveis resoluções de erro MSBuild, consulte [Recursos adicionais](../msbuild/additional-msbuild-resources.md).
