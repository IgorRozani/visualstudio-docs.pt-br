# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ServiceModel.Persistence.InstanceNotFoundException
Um <xref:System.ServiceModel.Persistence.InstanceNotFoundException> exceção nas seguintes circunstâncias:  
  
-   Uma operação é executada em uma instância de serviço durável que foi marcada para conclusão.  
  
-   Um provedor de persistência criado por um <xref:System.ServiceModel.Persistence.SqlPersistenceProviderFactory> tenta bloquear, desbloquear ou processar dados de estado que não foi encontrados no banco de dados.
