# O componente n&#227;o est&#225; registrado corretamente.
Esse erro geralmente ocorre quando o registro está corrompido, o sistema não pode acessar o registro ou um item foi gravado no registro com um formato incorreto.  
  
### Para corrigir este erro  
  
1.  Reinstale o componente.
