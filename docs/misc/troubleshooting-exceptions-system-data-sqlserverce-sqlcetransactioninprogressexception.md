# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Data.SqlServerCe.SqlCeTransactionInProgressException
Um `System.Data.SqlServerCe.SqlCeTransactionInProgressException` exceção ocorre quando um aplicativo tenta modificar um banco de dados enquanto outra transação está em andamento. Para obter mais informações, consulte [SQL Server Compact 3.5 Class Library](http://go.microsoft.com/fwlink/?LinkID=102595).
