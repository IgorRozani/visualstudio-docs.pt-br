# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IdentityModel.Selectors.IdentityValidationException
Um <xref:System.IdentityModel.Selectors.IdentityValidationException> exceção é lançada quando o certificado do destinatário não é válido. Isso pode acontecer se o certificado não se liga a para o certificado raiz com êxito.
