# &lt; specifier1 &gt;&lt; tipo &gt; n&#227;o pode herdar de &lt; specifier2 &gt;&lt; tipo &gt; porque ele expande o acesso da base de &lt; tipo &gt;
Você tentou fazer uma classe pública herdar de uma classe base marcada como `Private` ou `Friend`.  
  
 **ID do erro:** BC30509  
  
### Para corrigir este erro  
  
-   Declare a classe base `Public` ou declara a classe herdeira `Private` ou `Friend`.
