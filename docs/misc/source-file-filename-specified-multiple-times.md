# &lt; Filename &gt; do arquivo de origem especificado v&#225;rias vezes
Um arquivo foi especificado várias vezes.  
  
 **ID do erro:** BC2002  
  
### Para corrigir este erro  
  
-   Remova especificadores de arquivo redundantes.
