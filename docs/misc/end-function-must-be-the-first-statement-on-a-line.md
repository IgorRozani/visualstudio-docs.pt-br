# &#39;End Function&#39; deve ser a primeira instru&#231;&#227;o em uma linha
Um `End Function` instrução segue um separador de declaração dois\-pontos \(:\).`End Function` deve ser a única declaração na sua linha de código.  
  
 **ID do erro:** BC32032  
  
### Para corrigir este erro  
  
-   Separe as várias declarações em linhas diferentes.
