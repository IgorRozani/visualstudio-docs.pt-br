# Especificadores de fun&#231;&#227;o
Você pode usar o [embutido](../misc/inline-inline-forceinline.md) e [virtual](/visual-cpp/cpp/virtual-specifier) palavras\-chave como especificadores em declarações de função. Esse uso de **virtual** difere de seu uso no especificador de classe base de uma definição de classe.
