# O arquivo op&#231;&#245;es de configura&#231;&#245;es de usu&#225;rio para este projeto est&#225; com a se&#231;&#227;o &#39;se&#231;&#227;o&#39; faltando.
O nó compilação está ausente no the.vbproj.user ou. csproj.user arquivo.  
  
 A seção de compilação inclui configurações de depuração. Se essa seção estiver ausente, as configurações de depuração não será carregadas e levará em valores padrão.  
  
 A situação é provavelmente causada editando manualmente o arquivo de projeto.  
  
 O sistema de projeto reescreve automaticamente o arquivo de projeto por usuário quando o projeto é fechado.  
  
 Este é um aviso informativo.
