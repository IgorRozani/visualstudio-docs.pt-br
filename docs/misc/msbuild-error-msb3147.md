# Erro MSB3147 (MSBuild)
**MSB3147: Não foi possível localizar o arquivo necessário '\< arquivo \>' em '\< caminho \>'.**  
  
 Esse erro ocorre quando o bootstrapper não é possível localizar setup.bin no disco.
