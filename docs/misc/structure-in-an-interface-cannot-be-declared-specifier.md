# Estrutura em uma interface n&#227;o pode ser declarada como &#39;&lt; especificador &gt;&#39;
Um modificador inválido foi especificado com uma estrutura declarada em uma interface.  
  
 **ID do erro:** BC31071  
  
### Para corrigir este erro  
  
-   Remova modificadores como `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, ou `Overridable`.
