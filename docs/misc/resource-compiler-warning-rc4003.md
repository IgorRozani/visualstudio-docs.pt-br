# RC4003 de aviso do compilador de recurso
Não há parâmetros reais para 'Identificador' de macro  
  
 O número de parâmetros reais especificado com o identificador fornecido era menor que o número de parâmetros formais na definição de macro do identificador.  
  
 Quando um parâmetro formal é referenciado na definição e o parâmetro real correspondente não foi fornecido, texto vazio é substituído na expansão de macro.
