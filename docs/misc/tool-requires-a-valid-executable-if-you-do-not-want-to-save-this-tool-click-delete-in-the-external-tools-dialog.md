# A ferramenta exige um execut&#225;vel v&#225;lido. Se n&#227;o quiser salvar a ferramenta, clique em Excluir, na caixa de di&#225;logo Ferramentas Externas.
Esse erro geralmente ocorre quando uma ferramenta foi adicionada à caixa de diálogo externo de ferramentas, mas nenhum executável foi especificado.  
  
### Para corrigir este erro  
  
1.  Se você pretende salvar a ferramenta, clique **OK** e digite um nome de arquivo na caixa de **Comando** para a ferramenta ou procurar por um arquivo escolhendo o botão de reticências.  
  
2.  Se você não perto salvar a ferramenta, clique **OK** e clique em **Excluir** na caixa de diálogo **Ferramentas Externas** .
