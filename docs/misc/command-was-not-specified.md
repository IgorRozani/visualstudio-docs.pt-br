# O comando n&#227;o foi especificado.
Esse erro geralmente ocorre quando uma cadeia de caracteres vazia foi inserida para o **comando** janela ou o **Localizar\/comando** caixa.  
  
### Para corrigir este erro  
  
1.  Insira um comando válido. Para obter mais informações, consulte [Comandos do Visual Studio](../ide/reference/visual-studio-commands.md).
