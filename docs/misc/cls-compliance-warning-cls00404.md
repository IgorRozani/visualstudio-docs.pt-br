# CLS00404 de aviso de conformidade CLS
Identificadores de membro devem seguir o anexo 7 do relatório técnico do padrão Unicode 3.0  
  
 Certifique\-se de que os identificadores de membro sigam anexo 7 do relatório técnico do padrão Unicode 3.0.
