# Instru&#231;&#245;es &#39;Using&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Using` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30186  
  
### Para corrigir este erro  
  
-   Remover o `Using` instrução do **imediato** janela.
