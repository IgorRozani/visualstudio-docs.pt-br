# O aplicativo requer um. dll para identifica&#231;&#227;o de classe &lt; n&#250;mero &gt;.
Visual Studio não pôde localizar a DLL para a classe que você especificou. Este é um erro de sistema operacional.  
  
### Para corrigir este erro  
  
1.  Reinstalar o Visual Studio ou o aplicativo afetado e tente novamente.
