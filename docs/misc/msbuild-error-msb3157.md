# Erro MSB3157 (MSBuild)
**Msb3157 \(\): Não pode corresponder a cultura '\< culture \>' do item '\< pacote \>'. Usando a cultura '\< culture \>'.**  
  
 Esse aviso é gerado quando o produto especificado não é possível localizar um arquivo de manifesto de pacote \(Package\) que usa a cultura especificada.  
  
### Para corrigir este erro  
  
-   Forneça o arquivo de manifesto do pacote apropriado.
