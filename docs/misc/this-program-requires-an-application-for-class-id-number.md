# Este programa requer um aplicativo para a ID de classe &lt; n&#250;mero &gt;.
Visual Studio não pôde localizar o aplicativo que você está tentando usar. Essa mensagem é um erro de sistema operacional.  
  
### Para corrigir este erro  
  
1.  Reinstalar o aplicativo afetado e tente novamente.  
  
     – ou –  
  
     Reinstalar o Visual Studio e tente novamente.
