# Foi especificado um conjunto de caracteresouvazio.
Esse erro ocorre geralmente quando foi selecionada a opção de expressões regulares e uma seqüência de caracteres contendo sintaxe incompleta foi especificada para os padrões de  **conjunto de caracteres \[de** ou  **caractere fora do conjunto \[^**.  Por exemplo, `[}`.  
  
### Para corrigir este erro  
  
1.  Leia o tópico expressões regulares para obter informações sobre a sintaxe correta para o padrão e digite a seqüência de caracteres novamente.
