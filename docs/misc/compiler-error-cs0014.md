# CS0014 de erro do compilador
Arquivo necessário 'arquivo' não pôde ser encontrado  
  
 Um arquivo é necessário pelo compilador, mas não está no sistema. Certifique\-se de que o caminho está correto. Se o arquivo for um [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] do sistema de arquivos, talvez seja necessário reparar sua instalação ou então remova e reinstale o [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] completamente.
