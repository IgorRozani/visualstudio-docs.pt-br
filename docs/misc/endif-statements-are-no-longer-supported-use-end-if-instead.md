# N&#227;o h&#225; suporte para instru&#231;&#245;es &#39;EndIf&#39;; Use &#39;End If&#39; em vez disso
O `EndIf` instrução não é mais suportada.  
  
 **ID do erro:** BC30826  
  
### Para corrigir este erro  
  
-   Use `End If` para encerrar `If` instruções.
