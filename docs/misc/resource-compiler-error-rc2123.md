# RC2123 de erro do compilador de recurso
TÉRMINO esperado no menu  
  
 O **final** palavra\-chave deve vir no final de um **MENU** instrução. Verifique se você não tem qualquer aspas abertas ou um par incompatível de **BEGIN** e **final** instruções.
