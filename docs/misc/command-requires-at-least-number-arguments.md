# O comando requer pelo menos argumentos &lt; n&#250;mero &gt;.
Esse erro geralmente ocorre quando poucos argumentos foram inseridos para um comando.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
