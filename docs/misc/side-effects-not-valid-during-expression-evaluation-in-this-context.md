# Efeitos colaterais inv&#225;lidos durante a avalia&#231;&#227;o da express&#227;o neste contexto
Instruções que alteram o estado de um objeto não são permitidas neste contexto.  
  
 **ID do erro:** BC30700  
  
### Para corrigir este erro  
  
-   Altere seu código para que as instruções sejam avaliadas sem alterar o estado dos outros itens.
