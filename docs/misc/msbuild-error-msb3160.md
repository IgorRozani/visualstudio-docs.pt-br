# Erro MSB3160 (MSBuild)
**MSB3160: Aviso de validação de Xml no arquivo '\< arquivo \>': '\< erro \>'**  
  
 Esse aviso é gerado quando o arquivo Package. XML ou do produto não são compatíveis com o esquema XML. A última parte desse aviso é uma mensagem de aviso específico.  
  
### Para corrigir este erro  
  
-   Atualize o arquivo XML para resolver o erro de validação.
