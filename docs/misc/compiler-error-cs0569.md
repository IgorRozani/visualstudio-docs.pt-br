# CS0569 de erro do compilador
'method2': não é possível substituir 'method1' porque não há suporte de linguagem  
  
 Esse erro ocorre quando você derivar de uma classe base que foi escrita em outra linguagem e o compilador não suporta o método que você está tentando substituir.
