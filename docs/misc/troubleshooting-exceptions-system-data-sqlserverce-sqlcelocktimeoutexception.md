# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Data.SqlServerCe.SqlCeLockTimeoutException
Um `System.Data.SqlServerCe.SqlCeLockTimeoutException` exceção será lançada se o intervalo de bloqueio é atingido e SQL Server Compact expira enquanto aguarda um bloqueio. O intervalo de bloqueio padrão é 2000 milissegundos. Para obter mais informações, consulte [SQL Server Compact 3.5 Class Library](http://go.microsoft.com/fwlink/?LinkID=102595).
