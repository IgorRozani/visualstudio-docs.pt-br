# Divis&#227;o por zero.
Divisão por zero não é possível matematicamente. Esse erro geralmente ocorre quando o valor de uma expressão que está sendo usada como um divisor for zero.  
  
### Para corrigir este erro  
  
1.  Verifique se a expressão de ortografia, lógica, ou caso de erros.
