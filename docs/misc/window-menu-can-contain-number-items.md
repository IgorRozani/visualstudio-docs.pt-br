# Menu janela pode conter itens de &lt; n&#250;mero &gt;.
Lista do Windows no menu janela pode exibir somente nomes de janela até vinte e quatro. Esse erro geralmente ocorre quando um número menor que ou maior que 24 foi inserido no **\< número \> itens mostrados no menu janela** caixa no painel geral das opções de ambiente na caixa de diálogo Opções.  
  
### Para corrigir este erro  
  
1.  Insira um número entre uma e vinte e quatro na caixa.
