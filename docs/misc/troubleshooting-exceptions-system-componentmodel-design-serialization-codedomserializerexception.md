# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ComponentModel.Design.Serialization.CodeDomSerializerException
A exceção que é lançada quando o serializador CodeDOM encontra um erro.  
  
## Comentários  
 Esta exceção permite que o serializador retornar informações sobre o local e o tipo do erro.
