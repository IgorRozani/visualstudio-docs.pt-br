# RC1107 de Erro Fatal de compilador de recurso
Uso inválido; Use RC \/? Para obter ajuda  
  
 Uma opção de compilador de recurso inválida foi especificada. As opções válidas são \/r, \/v, \/d, \/fo, \/l, \/i, \/x e \/c.
