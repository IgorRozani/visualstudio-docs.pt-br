# &#39;MyClass&#39; n&#227;o pode ser usado fora de uma classe
Você usou o `MyClass` palavra\-chave fora de uma estrutura de classe.  
  
 **ID do erro:** BC30470  
  
### Para corrigir este erro  
  
-   Mova o `MyClass` declaração à sua estrutura de classe.
