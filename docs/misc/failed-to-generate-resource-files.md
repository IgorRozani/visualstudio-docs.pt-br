# Falha ao gerar os arquivos de recursos
Houve uma falha geral do sistema de projeto para ler um arquivo. resx como parte do processo de criação de arquivos. Resources binários que o Gerenciador de recursos do .NET possa entender.  
  
 Esse erro geralmente é causado por um arquivo. resx incorreto.  
  
 Esse aviso não causará a falha da compilação. No entanto, o arquivo. resx para o qual esse erro é exibido não será incluído como um recurso de manifesto de assembly no projeto.
