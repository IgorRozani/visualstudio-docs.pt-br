# N&#227;o foi especificado um destino adequado para a localiza&#231;&#227;o ou ocorreu outro erro de localiza&#231;&#227;o.
Esse erro geralmente ocorre quando foi tentada uma pesquisa com o escopo de pesquisa definido para o documento atual, mas nenhum arquivo está aberto.  
  
### Para corrigir este erro  
  
1.  Abrir um arquivo e tente a pesquisa novamente ou alterar o escopo da pesquisa.
