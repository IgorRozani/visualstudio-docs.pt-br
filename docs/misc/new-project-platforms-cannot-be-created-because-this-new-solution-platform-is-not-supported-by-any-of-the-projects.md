# N&#227;o &#233; poss&#237;vel criar novas plataformas de projeto porque nenhum dos projetos oferece suporte para a nova plataforma de solu&#231;&#227;o.
Esse erro ocorre quando você seleciona **criar novas plataformas de projeto**, mas nenhum dos projetos oferece suporte a nova plataforma de solução.  
  
### Para corrigir este erro  
  
-   Desmarque **criar novas plataformas de projeto** e clique em **OK**.  
  
     – ou –  
  
     Desmarque **criar novas plataformas de projeto** e selecione um nome de plataforma diferente na lista suspensa e, em seguida, clique em **OK**.
