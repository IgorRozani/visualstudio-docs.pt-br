# O argumento &#39;Access&#39; n&#227;o &#233; v&#225;lido. Os valores v&#225;lidos para o modo de entrada s&#227;o &#39;Read&#39; e &#39;OpenAccess. Default&#39;
Um valor de acesso não é válido em `Input` modo.  
  
### Para corrigir este erro  
  
-   Altere o valor para `OpenAccess.Read` ou `OpenAccess.Default`.
