# N&#227;o &#233; poss&#237;vel encontrar o ponto de entrada &#39;&lt; nome &gt;&#39; na DLL &#39;&lt; nomedoarquivo &gt;&#39;: &lt; erro &gt;
O nome do procedimento não pôde ser encontrado no DLL especificado.  
  
 **ID do erro:** BC31014  
  
### Para corrigir este erro  
  
1.  Verifique se o nome do procedimento está escrito corretamente e se as maiúsculas e minúsculas corresponde à configuração da função na DLL.
