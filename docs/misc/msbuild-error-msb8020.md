# Erro de MSBuild MSB8020
Ferramentas de compilação para *\< platform \>* \(conjunto de ferramentas de plataforma \= '*\< versão \>*'\) não pode ser encontrado. Criar usando o *\< versão \>* ferramentas de compilação, instale o *\< platform \>* ferramentas de compilação.  Como alternativa, você pode atualizar para o Visual Studio tools atual selecionando o menu Project ou com o botão direito na solução e selecionando "Redirecionar a solução".
