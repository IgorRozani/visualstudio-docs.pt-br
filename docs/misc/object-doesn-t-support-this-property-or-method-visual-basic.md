# Objeto n&#227;o suporta essa propriedade ou m&#233;todo (Visual Basic)
Você especificou um método ou propriedade não existe para esse objeto de automação.  
  
### Para corrigir este erro  
  
-   Leia a documentação do objeto para obter mais informações sobre o objeto e verificar a ortografia de propriedades e métodos.
