# Estendendo theTask lista
O [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] **lista de tarefas** permite que os usuários adicionam tarefas de programação personalizadas, exibir os comentários de tarefa que vinculam às linhas no código e criarem e exibir categorias personalizadas para mensagens de tarefa.  
  
 Tarefas são tratadas por meio de um serviço denominado <xref:Microsoft.VisualStudio.Shell.Interop.SVsTaskList>, que implementa <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskList> e <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskList2>. Usar básicas **lista de tarefas** funcionalidade, você deve criar um provedor de tarefa com a implementação de <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskProvider>.  
  
 Registrar seu provedor de tarefa com o <xref:Microsoft.VisualStudio.Shell.Interop.SVsTaskList> serviço chamando <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskList.RegisterTaskProvider%2A>, que retorna um valor de cookie que deve ser usado para identificar exclusivamente o provedor de tarefas em todas as transações subsequentes.  
  
 Cada implementação de provedor de tarefa é responsável por manter uma lista interna de tarefas. O provedor de tarefa pode chamar <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskList.RefreshTasks%2A> em **lista de tarefas** para atualizar a lista exibida de tarefas. Quando isso ocorre:  
  
1.  O serviço chama de volta para o provedor de tarefas, usando o <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskProvider.EnumTaskItems%2A> método.  
  
2.  A implementação do provedor de tarefa de <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskProvider.EnumTaskItems%2A> retorna um <xref:Microsoft.VisualStudio.Shell.Interop.IVsEnumTaskItems> objeto.  
  
3.  O <xref:Microsoft.VisualStudio.Shell.Interop.IVsEnumTaskItems> objeto itera em uma coleção de <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskItem> objetos.  
  
4.  O <xref:Microsoft.VisualStudio.Shell.Interop.SVsTaskList> atualizações de serviço, em seguida, o **lista de tarefas** exibir.  
  
 Funcionalidade adicional está disponível. O <xref:Microsoft.VisualStudio.Shell.Interop.IVsTaskProvider3> interface permite que cada provedor de tarefas fornecer seu próprio conjunto personalizado de colunas.
