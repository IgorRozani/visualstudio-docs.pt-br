# Os servi&#231;os de depura&#231;&#227;o do .NET Framework n&#227;o foi instalados corretamente. A causa mais prov&#225;vel &#233; que mscordbi n&#227;o est&#225; registrado corretamente. Clique em Ajuda para obter mais informa&#231;&#245;es sobre como reparar sua instala&#231;&#227;o do .NET Framework.
Esse erro geralmente ocorre quando o Visual Studio não pode localizar o arquivo mscordbi, que é instalado com os serviços de depuração Common Language Runtime.  
  
### Para corrigir este erro  
  
-   Repare a instalação do .NET Framework em seu computador.
