# N&#250;mero de &#237;ndices &#233; menor que o n&#250;mero de dimens&#245;es da matriz indexada
O número de índices usado para acessar um elemento de matriz deve ser exatamente o mesmo que a classificação da matriz, ou seja, o número de dimensões declarada para ela.  
  
 **ID do erro:** BC30105  
  
### Para corrigir este erro  
  
-   Adicione subscrições à referência de matriz até que o número total de subscrições é igual a classificação da matriz.
