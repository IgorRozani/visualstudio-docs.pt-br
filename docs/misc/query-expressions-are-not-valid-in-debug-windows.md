# Express&#245;es de consulta n&#227;o s&#227;o v&#225;lidas em janelas de depura&#231;&#227;o
Uma consulta LINQ foi inserida no **inspeção** ou **imediato** durante uma sessão de depuração. Consultas não estão disponíveis em janelas de depuração.  
  
 **ID do erro:** BC36609
