# Nenhuma fonte de entrada especificado
Durante a compilação, nenhuma fonte de entrada foi especificado.  
  
 **ID do erro:** BC2008  
  
### Para corrigir este erro  
  
-   Certifique\-se de que incluir um arquivo de código\-fonte.
