# N&#227;o &#233; poss&#237;vel remover a &#250;ltima sa&#237;da gerada
A última saída gerada conhecida não foi removida do disco e\/ou do projeto.  
  
 Se uma ferramenta personalizada retornou uma falha, a última saída gerada \(se houver\) é removida do disco e do projeto.  
  
 Isso pode acontecer se a saída gerada é bloqueada no disco.  
  
 **Para corrigir este erro**  
  
-   Feche e reinicie o [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
