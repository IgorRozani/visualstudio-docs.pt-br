# PRJ0038 de erro de compila&#231;&#227;o de projeto
Falha ao localizar o diretório raiz do IIS. O IIS deve ser instalado para que a ferramenta de implantação da Web funcione.  
  
 Implantação da Web requer o IIS no computador.
