# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ServiceModel.MessageHeaderException
Um <xref:System.ServiceModel.MessageHeaderException> exceção é lançada quando as expectativas relacionadas aos cabeçalhos de uma mensagem SOAP não são atendidas quando a mensagem é processada.
