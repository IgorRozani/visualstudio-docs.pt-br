# Este sistema n&#227;o tem suporte para a localidade especificada
Você está tentando usar o `VbStrConv` enumeração, mas seu sistema não dá suporte para a localidade especificada.  
  
### Para corrigir este erro  
  
1.  Atualize a máquina para oferecer suporte à localidade, ou execute o aplicativo em uma máquina que dá suporte a ela.
