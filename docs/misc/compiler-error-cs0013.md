# CS0013 de erro do compilador
Erro inesperado ao gravar metadados no arquivo 'ModelStore.dll' \- ' nenhum espaço lógico deixado para criar mais cadeias de caracteres do usuário.  
  
 O .NET Framework common language runtime não emitir metadados. Verifique se que o caminho está correto e se o disco não está cheio. Se o problema persistir, você terá que reparar ou reinstalar [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] e\/ou o .NET Framework.
