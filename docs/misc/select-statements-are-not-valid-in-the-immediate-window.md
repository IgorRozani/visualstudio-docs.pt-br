# Instru&#231;&#245;es &#39;Select&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Select` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30718  
  
### Para corrigir este erro  
  
-   Remover `Select` instruções do código de depuração.
