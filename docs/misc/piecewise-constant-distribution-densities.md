# piecewise_constant_distribution::densities
Retorna as densidades armazenadas para a distribuição.  
  
## Sintaxe  
  
```  
std::vector<double> densities() const;  
```  
  
## Comentários  
 A função de membro retorna as densidades armazenadas, calculadas a partir de parâmetros de peso.  
  
## Requisitos  
 **Cabeçalho:** \< random \>  
  
 **Namespace:** std
