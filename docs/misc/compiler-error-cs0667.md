# CS0667 de erro do compilador
O recurso 'recurso inválido' está obsoleto. Em vez disso, use 'recurso válido' '.  
  
 O recurso que você está tentando usar agora foi substituído. Atualize seu código para usar o recurso válido em vez disso.
