# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IdentityModel.Selectors.UntrustedRecipientException
Um <xref:System.IdentityModel.Selectors.UntrustedRecipientException> exceção é lançada quando um usuário examina as informações de uma entidade que está solicitando um token e decide não confiar na entidade. A usuário clica em **Desabilitar Windows CardSpace** na interface de usuário do CardSpace.
