# Op&#231;&#245;es de comando, o Assistente de pacote do Visual Studio
Esta etapa permite que você defina o nome e ID de comando para um comando que será exibido no Visual Studio **ferramentas** menu.  
  
 **Nome do comando**  
 Essa caixa de texto será exibido como:  
  
-   O nome do comando.  
  
-   A dica de ferramenta do botão de comando na barra de ferramentas.  
  
 **ID de comando**  
 -   Essa caixa de texto define a ID de comando para o comando. Essa ID é uma constante que é usada no código gerado para se referir a este item de menu.  
  
 **Cancelar**  
 -   Cancelar e parar o assistente.  
  
 **Anterior**  
 -   Move para a página anterior do assistente.  
  
 **Concluir**  
 -   Cria o VSPackage.  
  
 **Avançar**  
 -   Move para a próxima página do assistente.
