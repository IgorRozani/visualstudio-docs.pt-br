# N&#227;o &#233; poss&#237;vel adicionar um item ao projeto quando h&#225; v&#225;rios itens selecionados, ou nenhum, no Explorador de Projeto.
Esse erro ocorre geralmente quando nenhum item foi selecionado no Solution Explorer ou mais de um item foi selecionada.  
  
### Para corrigir este erro  
  
1.  Selecionar somente um item no Solution Explorer.
