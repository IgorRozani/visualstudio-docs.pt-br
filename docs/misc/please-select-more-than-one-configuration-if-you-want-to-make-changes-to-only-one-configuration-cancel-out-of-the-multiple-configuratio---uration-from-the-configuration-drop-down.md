# Selecione mais de uma configura&#231;&#227;o. Se voc&#234; quiser fazer altera&#231;&#245;es em apenas uma configura&#231;&#227;o, cancele a caixa de di&#225;logo v&#225;rias configura&#231;&#245;es e selecione a lista suspensa de configura&#231;&#227;o.
Esse erro geralmente ocorre quando uma configuração única solução foi selecionada no **várias configurações** caixa de diálogo.  
  
### Para corrigir este erro  
  
1.  Selecione configurações adicionais de solução.  
  
     – ou –  
  
2.  Se você quiser atualizar uma única configuração, escolha Cancelar e selecione a configuração da solução correta na lista a configuração de **Propriedades da solução** caixa de diálogo.
