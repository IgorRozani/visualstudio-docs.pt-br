# N&#227;o &#233; poss&#237;vel criar o arquivo &lt; nome_do_arquivo &gt;.
Esse erro geralmente ocorre quando uma tentativa de criar um relatório da Web de comentário do código, mas o usuário atual não tem acesso de gravação.  
  
### Para corrigir este erro  
  
1.  Verifique se você tem acesso de gravação.  
  
2.  Verifique se há espaço suficiente para salvar as páginas.
