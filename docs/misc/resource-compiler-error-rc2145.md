# RC2145 de erro do compilador de recurso
ID de idioma primário muito grande  
  
 A ID do idioma primário deve ser uma ID de idioma hexadecimal válido. Consulte [idiomas e cadeias de caracteres de país\/região](/visual-cpp/c-runtime-library/locale-names-languages-and-country-region-strings) no *referência da biblioteca de tempo de execução* para obter uma lista de IDs de idioma válido.
