# Erro MSB3112 (MSBuild)
**MSB3112: Dois ou mais assemblies têm a mesma identidade '\< assembly \>'.**  
  
 Esse aviso é gerado quando mais de um dos assemblies referenciados têm a mesma identidade.  
  
### Para corrigir este erro  
  
-   Remova um dos assemblies referenciados do projeto.
