# CS0690 de erro do compilador
Arquivo de entrada 'arquivo' contém metadados inválidos.  
  
 É possível abrir o arquivo de metadados, mas devido a algum problema de localização, os metadados foi corrompido. Isso é semelhante ao erro [CS0009](../misc/compiler-error-cs0009.md), exceto que você é capaz de abrir o arquivo de metadados.  
  
 Um PE \(executáveis de processo\) é um arquivo executável.
