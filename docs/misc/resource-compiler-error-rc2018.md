# RC2018 de erro do compilador de recurso
caractere desconhecido 'hexnumber'  
  
 O caractere ASCII correspondente ao número hexadecimal especificado aparece no arquivo de origem, mas é um caractere inválido.  
  
 Esse erro pode ser causado por corrupção de arquivo de origem.
