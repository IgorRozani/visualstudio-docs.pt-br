# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Net.NetworkInformation.PingException
A exceção que é lançada quando um <xref:System.Net.NetworkInformation.Ping.Send%2A> ou <xref:System.Net.NetworkInformation.Ping.SendAsync%2A> método chama um método que lança uma exceção. Por exemplo, um servidor pode não oferecer suporte a ping ou um firewall pode estar bloqueando a operação.
