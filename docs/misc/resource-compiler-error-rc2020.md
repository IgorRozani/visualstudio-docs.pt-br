# RC2020 de erro do compilador de recurso
número de dígitos inválidos para a base de base  
  
 Especificado `digit` não é um dígito válido para a base especificada por `radix`. O dígito ou a base pode estar incorreta.  
  
 Dígitos octais devem ser números de 0 a 7, e dígitos hexadecimais devem ser números de 0 a 9 ou letras de à E.
