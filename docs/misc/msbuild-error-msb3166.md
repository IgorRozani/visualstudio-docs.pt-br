# Erro MSB3166 (MSBuild)
**MSB3166: Não foi possível localizar o arquivo necessário '\< arquivo \>' para '\< pacote \>' do item.**  
  
 Esse erro é gerado quando um contrato de licença de usuário final \(EULA\) ou o arquivo de verificação externo está ausente. Verifique se que o arquivo existe no disco.  
  
### Para corrigir este erro  
  
-   Reinstalar o SDK do Visual Studio  
  
     \- ou \-  
  
-   Copie os arquivos necessários para o local apropriado.
