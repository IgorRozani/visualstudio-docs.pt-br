# RC1106 de Erro Fatal de compilador de recurso
opção inválida: opção  
  
 A opção não é uma opção de compilador de recurso válida.  
  
 As opções válidas são:  
  
 \/r  
  
 Emitir arquivo. res  
  
 \/v  
  
 Detalhado \(mensagens de progresso de impressão\)  
  
 \/d  
  
 Definir um símbolo  
  
 \/FO  
  
 Renomeie o arquivo. res  
  
 \/l  
  
 ID de idioma padrão em hexadecimal  
  
 \/I  
  
 Adicionar um caminho para **incluir** pesquisas  
  
 \/x  
  
 Ignorar **incluir** variável de ambiente  
  
 \/c  
  
 Definir uma página de código usada pela conversão de NLS
