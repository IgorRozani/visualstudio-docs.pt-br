# Valor da propriedade n&#227;o &#233; v&#225;lido. (Visual Studio) (1)
Esse erro geralmente ocorre quando um valor inválido inserido para uma propriedade. Por exemplo, se a propriedade é uma propriedade booleana com valores válidos de TRUE e FALSE, o valor TALVEZ seria inválido para a propriedade.  
  
### Para corrigir este erro  
  
1.  Insira um valor válido para a propriedade.  
  
    > [!TIP]
    >  Pesquise a documentação online para o nome da propriedade descobrir quais valores são válidos.
