# A lista para exame deve conter pastas v&#225;lidas ou arquivos v&#225;lidos, separados por ponto-e-v&#237;rgula.
Esse erro geralmente ocorre quando o diretório ou os arquivos listados no **Examinar** lista contêm caracteres que não são válidos.  
  
### Para corrigir este erro  
  
1.  Use o **Examinar** caixa de diálogo para procurar o nome do diretório ou arquivo para pesquisa em vez de digitar o nome de arquivo ou caminho. Você pode acessar o **Examinar** caixa de diálogo clicando no botão de reticências ao lado de **Examinar** lista.
