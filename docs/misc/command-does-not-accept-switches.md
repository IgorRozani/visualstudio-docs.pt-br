# O comando n&#227;o aceita op&#231;&#245;es.
Esse erro geralmente ocorre quando um comutador foi fornecido para um comando que reconhece apenas argumentos.  
  
### Para corrigir este erro  
  
1.  Revise os requisitos de sintaxe para o comando. Para obter mais informações, consulte [comandos do Visual Studio com argumentos](../ide/reference/visual-studio-commands.md).
