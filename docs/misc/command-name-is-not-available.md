# Comando &lt; nome &gt; n&#227;o est&#225; dispon&#237;vel.
Esse erro geralmente ocorre quando um comando é inserido no **comando** janela ou o **Localizar\/comando** caixa que não é válida, com base no estado atual do Visual Studio. Por exemplo, o `File.CloseSolution` comando não é válido se uma solução não estiver aberta no Visual Studio.  
  
### Para corrigir este erro  
  
1.  Escolha um comando diferente.  
  
     – ou –  
  
     Faça uma seleção diferente para o comando para operar.
