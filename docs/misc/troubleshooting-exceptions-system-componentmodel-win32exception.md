# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ComponentModel.Win32Exception
A exceção que é lançada para um código de erro do Win32.  
  
## Comentários  
 Códigos de erro Win32 são convertidos de suas representações numéricas em uma mensagem do sistema quando elas são exibidas. Use o <xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> propriedade para acessar a representação numérica do código de erro.
