# &#39;&lt; typename &gt;&#39; n&#227;o pode ser usado como um atributo porque est&#225; declarado como &#39;MustInherit&#39;
Classes de atributos personalizados não podem ser declarados como `MustInherit`.  
  
 **ID do erro:** BC31506  
  
### Para corrigir este erro  
  
-   Remover o `MustInherit` modificador de atributos personalizados.
