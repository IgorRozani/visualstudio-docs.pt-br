# N&#227;o &#233; poss&#237;vel calcular o n&#250;mero de per&#237;odos usando os argumentos fornecidos
Uma chamada para o `NPer`função não contém todos os argumentos necessários.  
  
### Para corrigir este erro  
  
-   Verifique se o `Rate`, `Prnt` e `PV` valores estão incluídos na chamada da função.
