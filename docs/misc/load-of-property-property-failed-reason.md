# Falha no carregamento da propriedade &#39;propriedade&#39;. &lt; motivo &gt;
Um valor de uma propriedade está fora do intervalo.  
  
 Esse erro provavelmente é causado editando manualmente o arquivo de projeto.  
  
 **Para corrigir este erro**  
  
-   Edite o arquivo de projeto novamente e corrigir o erro manualmente.  
  
     As propriedades afetadas serão definidas como o valor padrão, o valor da propriedade quando o projeto foi criado.
