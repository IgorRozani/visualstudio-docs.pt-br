# O atributo &#39;&lt; attributename &gt;&#39; n&#227;o &#233; v&#225;lido: &lt; erro &gt;
O common language runtime não aceita o atributo no contexto atual.  
  
 **ID do erro:** BC30127  
  
### Para corrigir este erro  
  
1.  Remova o atributo da declaração ou substituí\-lo por um atributo que é válido para o contexto.  
  
2.  Examine a mensagem de erro e tomar as devidas providências.
