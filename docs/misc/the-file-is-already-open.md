# O arquivo j&#225; est&#225; aberto
Foi feita uma tentativa de abrir um arquivo que já está aberto.  
  
### Para corrigir este erro  
  
-   Feche o arquivo antes de tentar abri\-lo.  
  
     \- ou \-  
  
-   Remova a `FileOpen` instrução no seu código.
