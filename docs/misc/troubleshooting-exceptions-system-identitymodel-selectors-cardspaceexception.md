# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IdentityModel.Selectors.CardSpaceException
Um <xref:System.IdentityModel.Selectors.CardSpaceException> exceção é lançada quando um ou mais exceções ocorrerem no nível de serviço do CardSpace. É a causa do erro no log de eventos.  
  
## Comentários  
 As condições que causam essa exceção incluem a ocorrência de um erro enquanto um repositório ou razão está sendo carregado.
