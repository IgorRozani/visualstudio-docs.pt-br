# PRJ0012 de erro de compila&#231;&#227;o de projeto
Recurso do sistema pode estar criticamente baixo. Não é possível criar um thread necessário para iniciar uma compilação.  
  
 Se os recursos do sistema estão baixos:  
  
-   Feche alguns aplicativos em execução.  
  
-   Reinicie o Visual Studio para liberar recursos.  
  
-   Reinicialização.
