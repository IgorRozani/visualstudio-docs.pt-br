# Especificadores s&#227;o v&#225;lidos somente no in&#237;cio de uma declara&#231;&#227;o
Palavras\-chave especificando acessibilidade e outras designações devem aparecer antes da palavra\-chave de declaração, como `Dim` ou `Function`.  
  
 **ID do erro:** BC30181  
  
### Para corrigir este erro  
  
-   Mova todos os especificadores entre a lista de atributos \(se houver\) e a palavra\-chave de declaração.
