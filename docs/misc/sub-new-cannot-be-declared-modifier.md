# &#39;Sub New&#39; n&#227;o pode ser declarado &#39;&lt; modificador &gt;&#39;
Você tentou declarar um `Sub New` com um modificador, como `Overrides`, que é inválido.  
  
 **ID do erro:** BC30364  
  
### Para corrigir este erro  
  
-   Remova o modificador.
