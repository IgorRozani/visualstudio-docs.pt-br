# Erro ao configurar a op&#231;&#227;o de manifesto de assembly: &lt; mensagem de erro &gt;
O [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador chama o vinculador de Assembly \(Al.exe, também conhecido como Alink\) para gerar um assembly com um manifesto. O vinculador relatou um erro ao localizar um arquivo de chave ou o nome da chave para assinar o assembly.  
  
 **ID do erro:** BC30139  
  
### Para corrigir este erro  
  
1.  Examine a mensagem de erro entre aspas e consulte a página chamada [Al.exe Tool Errors and Warnings](http://msdn.microsoft.com/pt-br/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) para obter mais explicações e conselhos.  
  
2.  Se o erro persistir, colete informações sobre as circunstâncias e notifique o Microsoft Product Support Services.
