# PRJ0051 de erro de compila&#231;&#227;o de projeto
Falha ao copiar 'arquivo' para o diretório de destino.  Verifique se que o arquivo de destino não está em uso.  
  
 O sistema de compilação não conseguiu copiar um arquivo.  
  
 Certifique\-se de que o arquivo não está aberto em seu computador ou em uso por outro programa.
