# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: Microsoft.VisualStudio.Tools.Applications.Runtime.UpdateFailedException
A exceção que é lançada quando um manifesto de implantação referencia um manifesto de aplicativo que é uma versão diferente do manifesto do aplicativo no documento, mas o manifesto do aplicativo no documento não pode ser atualizado.
