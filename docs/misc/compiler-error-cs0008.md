# CS0008 de erro do compilador
Erro inesperado ao ler metadados do arquivo 'arquivo', 'description'  
  
 Uma DLL foi aberta com êxito para recuperar metadados, mas estiver corrompida, que não foi possível ler dados dela. Para obter mais informações, consulte [\/reference \(Import Metadata\)](/dotnet/csharp/language-reference/compiler-options/reference-compiler-option).
