# &#39;Variant&#39; n&#227;o &#233; um tipo suportado; Use o tipo &#39;Object&#39; em vez disso
O `Object` substitui o tipo de dados `Variant`.  
  
 **ID do erro:** BC30804  
  
### Para corrigir este erro  
  
-   Declarar o item como `Object`.
