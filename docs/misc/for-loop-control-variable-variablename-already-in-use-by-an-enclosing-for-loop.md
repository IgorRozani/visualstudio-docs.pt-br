# Para a vari&#225;vel de controle de loop &#39;&lt; variablename &gt;&#39; j&#225; est&#225; em uso por um loop For delimitador
Aninhados `For` todos os loops devem usar variáveis de controle exclusivo.  
  
 **ID do erro:** BC30069  
  
### Para corrigir este erro  
  
-   Altere um dos nomes de variável de controle.
