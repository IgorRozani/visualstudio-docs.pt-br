# A cadeia de caracteres cont&#233;m muitas express&#245;es marcadas.
Esse erro geralmente ocorre quando mais de nove expressões marcadas foram especificadas na cadeia de pesquisa.  
  
### Para corrigir este erro  
  
1.  Reduza o número de expressões marcadas especificado.
