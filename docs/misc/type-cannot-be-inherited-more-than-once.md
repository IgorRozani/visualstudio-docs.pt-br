# &#39;&lt; type &gt;&#39; n&#227;o pode ser herdado mais de uma vez
Ter duplicado um `Inherits` instrução.  
  
 **ID do erro:** BC30584  
  
### Para corrigir este erro  
  
-   Remova a `Inherits` instrução.
