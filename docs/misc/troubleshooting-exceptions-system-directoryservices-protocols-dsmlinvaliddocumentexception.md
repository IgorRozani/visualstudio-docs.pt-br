# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.DirectoryServices.Protocols.DsmlInvalidDocumentException
A exceção que é lançada quando um documento de solicitação ou resposta DSML \(linguagem de marcação de serviços de diretório\) não está bem formado XML, ou não pode ser validada com o esquema DSMLV2.
