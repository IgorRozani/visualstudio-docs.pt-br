# Vari&#225;vel &#39;&lt; variablename &gt;&#39; &#233; usada antes de ele foi atribu&#237;do um valor (erro do Visual Basic)
Uma variável está sendo usada antes de ele foi atribuído um valor, resultando em um <xref:System.NullReferenceException>.  
  
 **ID do erro:** BC42109  
  
### Para corrigir este erro  
  
-   Verifique se que a variável foi atribuída um valor.
