# CXX0042 de erro do avaliador de express&#227;o
número de linha inválido  
  
 Este erro nunca deve ocorrer.  
  
 Observe as circunstâncias do erro e notificar a Microsoft Corporation. Você pode relatar o erro on\-line em [http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/) ou você pode chamar o Microsoft Product Support Services. Para obter mais informações, clique em **suporte técnico** sobre o **Ajuda** menus no Visual C\+\+.  
  
 Esse erro é idêntico ao CAN0042.
