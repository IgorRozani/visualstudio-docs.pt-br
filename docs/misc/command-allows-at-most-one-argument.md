# O comando permite no m&#225;ximo um argumento.
Esse erro geralmente ocorre quando há muitos argumentos foram inseridos para um comando que aceita um argumento ou nenhum argumento.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
