# N&#227;o foi poss&#237;vel copiar o item da Caixa de Ferramentas para outra guia.
Esse erro geralmente ocorre quando o item está corrompido ou a máquina ficou sem memória.  
  
### Para corrigir este erro  
  
1.  Feche outros aplicativos e, em seguida, tente mover o item novamente.  
  
     – ou –  
  
2.  Selecione um item diferente para mover.
