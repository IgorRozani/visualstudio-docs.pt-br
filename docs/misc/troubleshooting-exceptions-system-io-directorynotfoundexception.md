# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IO.DirectoryNotFoundException
Um <xref:System.IO.DirectoryNotFoundException> exceção é lançada quando parte de um caminho de arquivo ou um diretório não foi encontrado.  
  
## Dicas associadas  
 **Verifique se o diretório aparece no local especificado.**  
 Verifique se o diretório existe no local especificado.  
  
 **Ao usar caminhos relativos, verifique se que o diretório atual está correto.**  
 O caminho pode estar incorreto se você está assumindo um diretório atual incorreto.
