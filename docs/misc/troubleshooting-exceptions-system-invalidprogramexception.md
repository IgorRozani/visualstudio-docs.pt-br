# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.InvalidProgramException
A exceção que é gerada quando um programa contém código MSIL \(Microsoft intermediate language\) ou metadados. Isso geralmente indica um erro no compilador que gerou o programa.
