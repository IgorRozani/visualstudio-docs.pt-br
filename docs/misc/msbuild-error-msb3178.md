# Erro MSB3178 (MSBuild)
**MSB3178: O Assembly '\< assembly \>' foi especificado incorretamente como um arquivo.**  
  
 Esse aviso é gerado durante a geração de manifesto do aplicativo quando o processo de compilação detecta que uma referência de arquivo é realmente um assembly \(gerenciado ou nativo\).  
  
### Para corrigir este erro  
  
-   Para obter mais informações sobre a causa e as possíveis resoluções de erro MSBuild, consulte [Recursos adicionais](../msbuild/additional-msbuild-resources.md).
