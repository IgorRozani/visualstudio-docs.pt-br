# Forne&#231;a um t&#237;tulo para a ferramenta.
Esse erro geralmente ocorre quando o campo de **Título** dentro da caixa de diálogo **Ferramentas Externas** foi deixado em branco.  
  
### Para corrigir este erro  
  
1.  Digite um nome para a nova ferramenta externa na caixa de texto **Título** .
