# A ferramenta personalizada &#39;ferramenta personalizada&#39; falhou. &lt; motivo &gt;
A ferramenta personalizada não foi executada com êxito.  
  
 Se você encontrar um erro MSDataSetGenerator ao atualizar projetos que contêm conjuntos de dados de N camadas, você deve executar novamente a ferramenta personalizada depois que todos os projetos são atualizados.  
  
 Falha da ferramenta personalizada 'MSDataSetGenerator'. O projeto especificado no atributo DataSetProject em \< nome do conjunto de dados \> deve usar uma versão do .NET Framework que é igual ou posterior do projeto atual.  
  
### Para corrigir os erros MSDataSetGenerator em conjuntos de dados de N camadas  
  
-   Clique no arquivo. xsd no Solution Explorer e, em seguida, clique em **Executar ferramenta personalizada**.
