# Argumento desconhecido para &#39;:&#39; operador. Express&#227;o Regular completa necess&#225;ria na cadeia de caracteres de pesquisa.
Esse erro geralmente ocorre durante a pesquisa ou substituir quando caracteres curingas ou expressões regulares são usados em uma cadeia de caracteres de pesquisa. Os dois\-pontos do operador \(`:`\) foi inserido, mas o argumento seguido não era um argumento válido.  
  
> [!NOTE]
>  Argumentos para os dois\-pontos \(`:`\) operador diferenciam maiúsculas de minúsculas.  
  
### Para corrigir este erro  
  
1.  Examine a sintaxe de expressão regular correta localizada no tópico "Expressões regulares".  
  
2.  Verifique se as maiúsculas\/minúsculas corretas do argumento foi usada.  
  
3.  Se você estiver usando um Editor de método de entrada \(IME\), certifique\-se de que o argumento não é fornecido usando caracteres de largura total.
