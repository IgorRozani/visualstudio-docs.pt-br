# O arquivo de projeto n&#227;o possui a sub-se&#231;&#227;o &#39;sub-se&#231;&#227;o&#39; da se&#231;&#227;o &#39;se&#231;&#227;o&#39;.
A subseção \(subseção\) é incluir e seção \(seção\) é arquivos.  
  
 Esse erro provavelmente é causado editando manualmente o arquivo de projeto.  
  
 **Para corrigir este erro**  
  
-   Trabalhar em **Show All Files** modo \(clicando nesse botão na barra de ferramentas do Gerenciador de soluções\) e incluir arquivos relevantes.  
  
     Se você receber esse erro, o projeto será aberto, mas nenhum arquivo será adicionado ao projeto.
