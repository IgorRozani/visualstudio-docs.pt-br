# O projeto &#39;&lt; projectname &gt;&#39; faz uma refer&#234;ncia indireta ao assembly &#39;&lt; assemblyname &gt;&#39; que cont&#233;m &#39;&lt; classname &gt;&#39;
O projeto '\< projectname \>' faz uma referência indireta ao assembly '\< assemblyname \>' que contém '\< classname \>'. Adicione uma referência a '\< assemblyname \>' ao seu projeto.  
  
 Este projeto contém membros que dependem de um assembly que não está contido nas referências ao projeto.  
  
 **ID do erro:** BC31515  
  
### Para corrigir este erro  
  
-   Adicione uma referência ao assembly **referências** no **Solution Explorer** e, em seguida, clicando em **Adicionar referência**.
