# RC1207 de Erro Fatal de compilador de recurso
página de código padrão é inválida  
  
 A página de código especificada pelo **RCCODEPAGE** variável de ambiente não é válida. Consulte [páginas de código](/visual-cpp/c-runtime-library/code-pages) no *referência da biblioteca de tempo de execução* para obter mais informações.
