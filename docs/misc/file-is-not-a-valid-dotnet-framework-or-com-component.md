# O arquivo n&#227;o &#233; um componente .NET Framework ou COM v&#225;lido.
Você está tentando exibir um arquivo no pesquisador de objetos que não contém um objeto válido.  Somente [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)] e componentes COM podem ser exibidos no pesquisador de objetos.  
  
### Para corrigir este erro  
  
1.  Se você selecionou o arquivo incorreto, selecione o arquivo e tente corretos novamente.  
  
2.  Se você selecionou equivocadamente o pesquisador de objetos na caixa de diálogo **Abrir com** , escolha um programa diferente para abrir o arquivo com.
