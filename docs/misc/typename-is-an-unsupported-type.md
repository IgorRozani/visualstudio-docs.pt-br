# &#39;&lt; typename &gt;&#39; &#233; um tipo sem suporte
O tipo especificado não pode ser representado por [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
 **ID do erro:** BC30649  
  
### Para corrigir este erro  
  
-   Use um tipo com suporte.  
  
-   Desenvolva um novo tipo.
