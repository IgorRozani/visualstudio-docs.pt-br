# Retornar ao Visual Studio para entrar no IDE com sua nova conta da Microsoft!
Você criou com êxito uma Account da Microsoft. Retornar ao Visual Studio e logon do boas\-vindas do assistente durante a primeira inicialização ou do canto superior direito do IDE qualquer tempo.  
  
 Entrar no IDE para começar a usar seus créditos do Azure, publicar o código para um repositório Git particular, as configurações de sincronização e desbloquear o IDE.[Mais](../ide/signing-in-to-visual-studio.md) sobre todos os serviços de desenvolvedor disponíveis.
