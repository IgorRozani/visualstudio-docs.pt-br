# CA0056
Um arquivo de projeto tem uma versão incorreta da ferramenta de análise.  
  
 Este aviso ocorre se FxCopCmd.exe é invocado manualmente com um arquivo de projeto que tem uma versão de ferramenta de análise de código incorreto.
