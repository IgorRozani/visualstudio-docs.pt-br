# O projeto &#39;projeto&#39; n&#227;o pode ser referenciado. &lt;reason&gt;
Não pôde ser feita uma referência de projeto ao projeto para um projeto específico.  
  
 Referência um projeto cria um arquivo. exe é o motivo mais comum para esse erro.  Este erro também indica que um projeto referencia um projeto cuja saída não é um assembly \(por exemplo, um aplicativo C\+\+ nativo\).
