# Erro MSB3183 (MSBuild)
**MSB3183: Referência '\< assembly \>' é um assembly de interoperabilidade que requer confiança total.**  
  
 Esse aviso é gerado quando um assembly de interoperabilidade é encontrado entre as referências ao gerar o manifesto para um aplicativo parcialmente confiável.
