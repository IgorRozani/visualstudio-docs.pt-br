# Erro MSB4132 (MSBuild)
**MSB4132: A versão das ferramentas "' \< version \>'" não é reconhecida.**  
  
 [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] foi possível encontrar um conjunto de ferramentas que corresponde ao valor especificado de `ToolsVersion`.  
  
### Para corrigir este erro  
  
-   Especifique um valor válido para `ToolsVersion` na marca do projeto ou na linha de comando ao usar o MSBuild **\/ToolsVersion.** alternar.
