# Argumento n&#227;o especificado para o par&#226;metro &#39;&lt; parametername &gt;&#39;
Um argumento para um parâmetro necessário não foi fornecido.  
  
 **ID do erro:** BC36569  
  
### Para corrigir este erro  
  
-   Forneça um argumento para o parâmetro especificado.
