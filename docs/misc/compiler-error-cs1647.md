# CS1647 de erro do compilador
Uma expressão é muito longa ou complexa para ser compilada próximo a 'code'  
  
 Ocorreu um estouro de pilha no compilador seu código de processamento. Para resolver esse erro, simplificar o seu código. Se seu código for válido, entre em contato com [suporte ao produto](http://msdn.microsoft.com/pt-br/77e75b8b-817d-45bf-9c38-458930d873b4).
