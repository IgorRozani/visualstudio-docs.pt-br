# Um delimitador n&#227;o pode ser nada ou uma cadeia de caracteres vazia
O `TextFieldParser` não pode ler o arquivo porque o `Delimiters` está definida como `Nothing` ou está vazio `String` \(""\).  
  
### Para corrigir este erro  
  
-   Forneça um valor válido para `Delimiters`.
