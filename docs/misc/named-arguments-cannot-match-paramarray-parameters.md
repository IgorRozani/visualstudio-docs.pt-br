# Argumentos nomeados n&#227;o correspondem aos par&#226;metros ParamArray
Matrizes de parâmetros devem ser passados por valor.  
  
### Para corrigir este erro  
  
1.  Passar os argumentos `ByVal`, em vez de nomeá\-los.
