# Par&#226;metros opcionais n&#227;o podem ter tipos de estrutura
Parâmetros opcionais não podem ser tipos de estrutura.  
  
 **ID do erro:** BC31405  
  
### Para corrigir este erro  
  
1.  Declare parâmetros de tipo de estrutura conforme necessário.  
  
2.  Declare um membro de classe estática para tomar o lugar de uma estrutura.
