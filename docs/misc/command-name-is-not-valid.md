# Comando &lt; nome &gt; n&#227;o &#233; v&#225;lido.
Esse erro geralmente ocorre quando um comando ou alias de um comando não reconhecido pelo Visual Studio foi inserido para o **Localizar\/comando**caixa ou **comando** janela. Se um alias foi inserido, o nome completo do comando é exibido em vez do nome do alias na mensagem de erro.  
  
### Para corrigir este erro  
  
1.  Digite novamente o nome do comando ou escolha um nome de comando válido.
