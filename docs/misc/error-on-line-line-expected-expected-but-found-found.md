# Erro na linha &lt; linha &gt;. Esperado &#39;esperado&#39;, mas &#39; localizado &#39;localizado
Isso é um erro de análise ao ler o arquivo de projeto XML geral. Esse erro pode ocorrer por arquivos de projeto \(.vbproj\/.csproj\), bem como arquivos de configurações por usuário \(.vbproj.user\/.csproj.user\). Mais informações serão fornecidas pelo sistema de projeto nos parâmetros `expected` e `found`.  
  
 Esse erro é provavelmente causado ao editar o arquivo de projeto \(ou usuário\) à mão.  
  
 **Para corrigir este erro**  
  
-   Edite o arquivo de projeto novamente e restaurá\-lo a uma condição satisfatória.  
  
     Dependendo da gravidade do erro, não é possível abrir o projeto.
