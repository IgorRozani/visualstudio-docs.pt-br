# &#39;&lt; mathfunction1 &gt;&#39; n&#227;o est&#225; declarado.
'\< mathfunction1 \>' não está declarado. Função foi movido para a classe System. Math e agora é chamada de '\< mathfunction2 \>'.  
  
 Várias funções que foram intrínsecas a [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] em versões anteriores foram movidas para o <xref:System.Math?displayProperty=fullName> namespace. Isso torna a funcionalidade mais disponível para todas as linguagens de programação.  
  
 **ID do erro:** BC30819  
  
### Para corrigir este erro  
  
-   Use os métodos declarados em <xref:System.Math?displayProperty=fullName>.
