# CS1639 de erro do compilador
A assinatura de classe de wrapper gerenciado coclass 'assinatura' para interface ' de ' não é uma assinatura de nome de classe válida  
  
 Esse erro ocorre se os metadados para a interface são inválido. Isso nunca deve ocorrer se a interface foi gerada usando c\# ou um idioma com suporte do .NET. Verifique com o fornecedor da interface ou verificar se o assembly de interface foi gerado com êxito.
