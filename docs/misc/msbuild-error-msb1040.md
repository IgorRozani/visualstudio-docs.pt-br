# Erro MSB1040 (MSBuild)
**MSB1040: ToolsVersion é inválido.**  
  
 O valor de `ToolsVersion` especificado com a opção de **\/ToolsVersion** é inválido.  
  
### Para corrigir este erro  
  
-   Especificar um valor válido de `ToolsVersion` quando você cria usando a opção de **\/ToolsVersion** na linha de comando.
