# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.ServiceModel.Persistence.InstanceLockException
O <xref:System.ServiceModel.Persistence.InstanceLockException> exceção destina para uso por classes que herdam de <xref:System.ServiceModel.Persistence.LockingPersistenceProvider> quando a operação não pode ser executada devido ao estado do bloqueio de instância.
