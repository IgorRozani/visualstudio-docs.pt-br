# O aplicativo n&#227;o registrou uma f&#225;brica de classes para identifica&#231;&#227;o de classe &lt; n&#250;mero &gt;.
Um objeto COM que implementa a interface IClassFactory do aplicativo não foi registrado. Este é um erro de sistema operacional.  
  
### Para corrigir este erro  
  
1.  Reinstalar o Visual Studio ou o aplicativo afetado e tente novamente.
