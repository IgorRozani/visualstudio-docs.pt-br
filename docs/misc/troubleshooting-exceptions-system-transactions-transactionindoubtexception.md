# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Transactions.TransactionInDoubtException
A exceção que é lançada quando uma operação é tentada em uma transação que está em dúvida.  
  
## Comentários  
 Uma transação está em dúvida quando um Gerenciador de recursos registrado não pode ser contatado.
