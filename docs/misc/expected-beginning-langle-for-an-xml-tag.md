# Abertura esperada &#39; &lt;&#39; para uma marca XML
A marca XML para um literal XML tem início obrigatório ' \<' caracteres.  
  
 **ID do erro:** BC31165  
  
### Para corrigir este erro  
  
-   Adicionar a ' \<' caractere para o início da marca XML para o literal XML.
