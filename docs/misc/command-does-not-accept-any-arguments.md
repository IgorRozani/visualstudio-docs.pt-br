# O comando n&#227;o aceita nenhum argumento.
Esse erro geralmente ocorre quando um argumento foi inserido para um comando que só aceita comutadores.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
