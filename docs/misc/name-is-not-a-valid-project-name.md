# &lt; nome &gt; n&#227;o &#233; um nome de projeto v&#225;lido.
Esse erro geralmente ocorre quando o `File.NewProject` comando for emitido o **comando** janela ou **Localizar\/comando** caixa e um nome de projeto incorreto for inserido para *projectname*.  
  
### Para corrigir este erro  
  
1.  Verifique se o nome do projeto não contém espaços extras e não é uma palavra reservada como NUL, CON ou AUX  
  
     – ou –  
  
     Digite novamente o comando, omitindo o nome do projeto.
