# Erro MSB1015 (MSBuild)
**MSBuild não é executado nesta versão do sistema operacional. Somente há suporte no Windows 2000, Windows XP e versões posteriores.**  
  
 [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] Não há suporte nesta versão do sistema operacional.  
  
### Para corrigir este erro  
  
1.  Instalar e executar [!INCLUDE[vstecmsbuild](../extensibility/internals/includes/vstecmsbuild_md.md)] em uma das versões com suporte do sistema operacional.
