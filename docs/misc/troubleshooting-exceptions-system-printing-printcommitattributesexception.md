# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Printing.PrintCommitAttributesException
Um <xref:System.Printing.PrintCommitAttributesException> exceção é lançada quando uma condição de erro impede que os atributos sejam confirmados por um <xref:System.Printing.PrintSystemObject> para o real do computador, impressora ou dispositivo que representa o objeto.
