# O arquivo &lt; nome &gt; n&#227;o pode ser registrado como um componente COM.
O arquivo não pode ser adicionado às referências do projeto. Esse erro ocorre quando um arquivo foi corrompido, não contém bibliotecas de tipos, usa um arquivo incompatível tipos ou quando o sistema não tem recursos suficientes para concluir a operação.  
  
### Para corrigir este erro  
  
1.  Verifique se o arquivo contém uma biblioteca de tipo válido e tente novamente.  
  
     – ou –  
  
     Reinstale o produto que instalou o arquivo.  
  
    > [!NOTE]
    >  O produto que instalou o arquivo pode não ser o Visual Studio.
