# A op&#231;&#227;o &lt; nome &gt; requer um valor.
Esse erro geralmente ocorre quando uma opção for especificada sem um valor obrigatório.  
  
### Para corrigir este erro  
  
1.  Digite novamente o comando e fornecer um valor para a opção.  
  
     – ou –  
  
     Digite novamente o comando e omitir a opção.
