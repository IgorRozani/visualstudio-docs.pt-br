# &#39;Implements&#39; n&#227;o &#233; v&#225;lidos em m&#243;dulos
Um `Implements` instrução ou cláusula ocorre dentro de um módulo. Módulos não podem implementar interfaces.  
  
 **ID do erro:** BC30231  
  
### Para corrigir este erro  
  
-   Remover o `Implements` instrução ou cláusula ou redigite o módulo como uma classe.
