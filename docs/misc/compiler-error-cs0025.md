# CS0025 de erro do compilador
Arquivo de biblioteca padrão 'arquivo' não pôde ser encontrado  
  
 Não foi encontrado um arquivo que é necessário pelo compilador. Verifique se o caminho está correto e se o arquivo existe.  
  
 Se o arquivo for um [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] arquivo do sistema, talvez seja necessário reparar sua [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] instalação, ou reinstalá\-lo completamente.
