# RC1023 de Erro Fatal de compilador de recurso
não é possível abrir o arquivo de origem 'filename'  
  
 O arquivo especificado seja não existe, não pôde ser aberto ou não foi encontrado.  
  
 Certifique\-se de que as configurações de ambiente são válidas e se o caminho correto para o arquivo é especificado.  
  
 Se esse erro for exibida sem uma mensagem de erro, o compilador de recurso ficou sem identificadores de arquivos.
