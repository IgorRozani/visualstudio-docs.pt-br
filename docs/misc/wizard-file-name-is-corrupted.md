# O arquivo &lt; nome &gt; do assistente est&#225; corrompido.
O arquivo. vsz do assistente não tem o formato correto. Esse erro ocorre quando um arquivo. vsz está corrompido ou se o arquivo for de um produto diferente do Visual Studio.  
  
### Para corrigir este erro  
  
1.  Verifique se o assistente é um assistente do Visual Studio.  
  
     – ou –  
  
     Reinstale o Visual Studio para substituir o arquivo corrompido.
