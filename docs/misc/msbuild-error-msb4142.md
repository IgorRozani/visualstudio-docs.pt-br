# Erro MSB4142 (MSBuild)
**MSB4142: MSBuildToolsPath não é o mesmo que MSBuildBinPath para o ToolsVersion “\<x.x\>”.**  
  
 A definição do kit de ferramentas especifica valores diferentes para `MSBuildBinPath` e `MSBuildToolsPath`.  
  
### Para corrigir este erro  
  
-   Certifique\-se de que os valores para `MSBuildBinPath` e `MSBuildToolsPath` são os mesmos na definição do kit de ferramentas.
