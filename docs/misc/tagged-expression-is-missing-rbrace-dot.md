# &#39;}&#39; est&#225; ausente na express&#227;o marcada.
Esse erro geralmente ocorre quando uma chave de abertura \(`{`\) foi especificado mas a chave de fechamento \(`}`\) foi omitido em uma cadeia de caracteres de localização.  
  
### Para corrigir este erro  
  
1.  Para pesquisar o caractere literal `{`, use `\{`.  
  
2.  Para marcar uma expressão, digite ausentes `}`.
