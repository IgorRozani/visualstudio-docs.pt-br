# CS1559 de erro do compilador
Não é possível usar 'object' para o método Main porque é importado  
  
 Uma classe inválida foi especificada para o [\/principal](/dotnet/csharp/language-reference/compiler-options/main-compiler-option) opção de compilador; a classe não pode ser usada como um local para o [principal](/dotnet/csharp/programming-guide/main-and-command-args/main-and-command-line-arguments) método.
