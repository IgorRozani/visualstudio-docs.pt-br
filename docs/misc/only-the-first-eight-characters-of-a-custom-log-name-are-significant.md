# Somente os oito primeiros caracteres de um nome de log personalizado s&#227;o significativos
Ao verificar os nomes de log de eventos de exclusividade, somente os oito primeiros caracteres são considerados. Um conflito pode resultar de logs de eventos que compartilham seus primeiros oito caracteres.  
  
### Para corrigir este erro  
  
-   Nomeie o log de eventos nos quais os oito primeiros caracteres são exclusivos.
