# Erro MSB1026 (MSBuild)
**Arquivo de esquema não existe.**  
  
 O arquivo de esquema não existe no local especificado.  
  
### Para corrigir este erro  
  
1.  Verifique se o arquivo de esquema existe no diretório especificado, se você tiver especificado o local absoluto ou relativo do arquivo do esquema.
