# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IO.FileFormatException
Um <xref:System.IO.FileFormatException> exceção é lançada quando um arquivo de entrada ou um fluxo de dados que deve estar de acordo com uma determinada especificação de formato de arquivo está malformado.
