# Erro MSB3122 (MSBuild)
**Msb3122 \(\): O uso de associações de arquivo requer confiança total.**  
  
 Publicar um aplicativo que configura as associações de arquivo requer que o aplicativo seja um aplicativo de confiança total.  
  
### Para corrigir este erro  
  
-   Habilitar configurações de segurança do ClickOnce e configurar o aplicativo para um aplicativo de confiança total. Para obter mais informações, consulte [Como habilitar configurações de segurança do ClickOnce](../deployment/how-to-enable-clickonce-security-settings.md).
