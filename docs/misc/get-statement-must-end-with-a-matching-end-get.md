# Instru&#231;&#227;o &#39;Get&#39; deve finalizar com &#39;End Get&#39; correspondente
`Get` procedimentos de propriedade devem terminar com um `End Get` instrução.  
  
 **ID do erro:** BC30631  
  
### Para corrigir este erro  
  
-   Verifique se o `Get` procedimento de propriedade termina com um `End Get` Construir.
