# Comutador &lt; nome &gt; n&#227;o permite que um valor.
Esse erro geralmente ocorre quando um comutador que não permite que um valor é atribuído um valor.  
  
### Para corrigir este erro  
  
1.  Digite novamente o comando, omitindo um valor para a opção.
