# O comando requer pelo menos um argumento.
Esse erro geralmente ocorre quando foram inseridos sem argumentos para um comando que requer pelo menos um argumento.  
  
### Para corrigir este erro  
  
1.  Verifique a documentação para obter a sintaxe correta para o comando e tente novamente.
