# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IO.FileNotFoundException
Um <xref:System.IO.FileNotFoundException> exceção é lançada quando é feita uma tentativa de acessar um arquivo ou diretório não existe no disco.  
  
## Dicas associadas  
 **Verifique se o arquivo existe no local especificado.**  
 Se o arquivo não existir, essa exceção é lançada. Para obter mais informações, consulte <xref:Microsoft.VisualBasic.FileIO.FileSystem.FileExists%2A>.  
  
 **Ao usar caminhos relativos, verifique se que o diretório atual está correto.**  
 Se você estiver usando o diretório atual incorretamente, caminhos relativos serão incorretos também.
