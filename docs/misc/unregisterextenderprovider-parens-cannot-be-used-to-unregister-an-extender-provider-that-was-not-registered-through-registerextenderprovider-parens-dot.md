# UnregisterExtenderProvider() n&#227;o pode ser usado para cancelar o registro de um provedor do extensor que n&#227;o tenha sido registrado por meio de RegisterExtenderProvider().
Provedores de registro não precisam ser registrados e não registrado explicitamente.
