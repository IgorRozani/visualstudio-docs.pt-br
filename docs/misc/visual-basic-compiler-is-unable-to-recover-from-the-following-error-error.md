# Compilador do Visual Basic &#233; capaz de recuperar o seguinte erro: &lt; erro &gt;
O [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador é capaz de recuperar o seguinte erro: \<`error`\>.  
  
 Alguns recursos IDE foram desabilitados.  
  
 Ocorreu um erro interno no [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador.  
  
 **ID do erro:** BC31021  
  
### Para corrigir este erro  
  
1.  Salve seu trabalho e reinicie o [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
2.  Se o erro persistir, reinstale [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
3.  Se o erro persistir após a reinstalação, notifique o Microsoft Product Support Services.
