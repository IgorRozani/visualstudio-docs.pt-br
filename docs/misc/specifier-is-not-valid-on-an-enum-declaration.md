# &#39;&lt; especificador &gt;&#39; n&#227;o &#233; v&#225;lido em uma declara&#231;&#227;o de Enum
Você usou um especificador que não é válido em um `Enum` declaração.  
  
 **ID do erro:** BC30396  
  
### Para corrigir este erro  
  
1.  Remova o especificador.
