# Foi passado um cookie inv&#225;lido para UnregisterExtenderProvider().
Você especificou um cookie que não é mais válido.  
  
### Para corrigir este erro  
  
1.  Retorne o mesmo valor de cookie que você tenha retornado da chamada RegisterExtenderProvider \(\).
