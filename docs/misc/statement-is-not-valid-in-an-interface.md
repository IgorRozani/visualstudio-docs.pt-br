# Instru&#231;&#227;o n&#227;o &#233; v&#225;lida em uma interface
Uma instrução inválida foi encontrada na interface.  
  
 **ID do erro:** BC31041  
  
### Para corrigir este erro  
  
-   Verifique se que instruções na interface definem elementos, mas não fornecem implementação.
