# RC2126 de erro do compilador de recurso
cadeia de caracteres de menu esperado  
  
 Cada **MENUITEM** e **pop\-up** instrução deve conter um *texto* campo, que é uma cadeia de caracteres entre aspas duplas que especifica o nome do menu pop\-up ou de item de menu. Um **MENUITEM SEPARADOR** instrução não exige nenhuma cadeia de caracteres entre aspas.
