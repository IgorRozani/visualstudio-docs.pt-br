# Par&#226;metro j&#225; declarado com nome &#39;&lt; argumentname &gt;&#39;
Uma declaração de procedimento define dois argumentos com o mesmo nome. Todos os nomes de argumento em uma declaração de procedimento devem ser exclusivos.  
  
 **ID do erro:** BC30237  
  
### Para corrigir este erro  
  
-   Altere o nome de um dos argumentos.
