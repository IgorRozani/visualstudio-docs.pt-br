# O arquivo est&#225; em uso e n&#227;o pode ser substitu&#237;do.
O arquivo atual não pode ser salvo.  Esse erro geralmente ocorre quando você tenta salvar um arquivo usando o nome de um arquivo aberto também ao mesmo tempo.  
  
### Para corrigir este erro  
  
1.  Feche o arquivo com o mesmo nome e tente novamente.  
  
     \- ou \-  
  
     Renomeie o arquivo e salve o.
