# Op&#231;&#227;o n&#227;o reconhecida &lt; optionname &gt;; ignorado
Foi especificada uma opção que não é válido na linha de comando.  
  
 **ID do erro:** BC2007  
  
### Para corrigir este erro  
  
-   Verificar opções especificadas para encontrar aquela que não é válida.
