# Erro MSB4138 (MSBuild)
**MSB4138: os dados não\-cadeia de caracteres foram especificados no local “'\<key\>'” do Registro.**  
  
 O local do Registro especificado em uma configuração personalizada do kit de ferramentas é inválido.  
  
### Para corrigir este erro  
  
-   Certifique\-se de que a definição do kit de ferramentas está correta e que a chave de Registro especificada existe.
