# RC2132 de erro do compilador de recurso
VALOR esperado, bloquear ou, terminar a palavra\-chave  
  
 Um bloco um **versão** recursos não terminam corretamente ou um novo bloco não iniciar corretamente. Isso resulta em uma **versão** instrução que não seja um bloco válido.
