# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.MulticastNotSupportedException
A exceção que é lançada quando há uma tentativa de combinar duas instâncias de um tipo delegado não combinável. Uma exceção não é lançada se um dos operandos for uma referência nula \(`Nothing` no Visual Basic\).
