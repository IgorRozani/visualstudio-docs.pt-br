# Instru&#231;&#245;es &#39;Exit&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Exit` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30713  
  
### Para corrigir este erro  
  
-   Remover `Exit` instruções do código de depuração.
