# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.IdentityModel.Tokens.SecurityTokenValidationException
Um <xref:System.IdentityModel.Tokens.SecurityTokenValidationException> exceção é lançada quando um token de segurança recebido não é válido. A exceção é lançada normalmente pelo <xref:System.IdentityModel.Selectors.SecurityTokenAuthenticator.ValidateToken%2A> método.
