# Erro MSB3162 (MSBuild)
**MSB3162: O Item '\< pacote \>' não foi possível localizar item dependente '\< pacote \>'.**  
  
 Esse erro ocorre quando um pacote de bootstrapper depende de outro, mas o manifesto do pré\-requisito bootstrapper dependente não pode ser encontrado.
