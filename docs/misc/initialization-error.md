# Erro de inicializa&#231;&#227;o
Ocorreu um erro ao inicializar o programa. Isso pode ser resultado de cancelar uma instalação enquanto ele ainda está em andamento.  
  
 **ID do erro:** BC2000  
  
### Para corrigir este erro  
  
-   Tente a instalação novamente.
