# piecewise_constant_distribution::intervals
Retorna os intervalos armazenados da distribuição.  
  
## Sintaxe  
  
```  
std::vector<RealType> intervals() const;  
```  
  
## Comentários  
 A função de membro retorna os intervalos armazenados.
