# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
O <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> exceção é lançada quando um diretório do segmento está ausente da estrutura de diretório de pipeline.  
  
## Comentários  
 Quando um suplemento é ativado, seu pipeline para o host é construído. O sistema suplemento exige que os segmentos de pipeline sejam colocados em uma estrutura de diretório especificado com nomes de diretório especificado, conforme descrito em [requisitos de desenvolvimento de Pipeline](http://msdn.microsoft.com/pt-br/ef9fa986-e80b-43e1-868b-247f4c1d9da5).
