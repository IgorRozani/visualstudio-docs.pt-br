# Refer&#234;ncia necess&#225;ria para o m&#243;dulo &#39;&lt; modulename &gt;&#39; contendo a classe base &#39;&lt; classname &gt;&#39;
Referência necessária para o módulo '\< modulename \>' contendo a classe base '\< classname \>'. Adicione uma ao seu projeto.  
  
 A classe é definida em um módulo que não é referenciado diretamente em seu projeto. O [!INCLUDE[vbprvb](../code-quality/includes/vbprvb_md.md)] compilador requer uma referência para evitar ambigüidade caso a classe é definida em mais de um módulo.  
  
 **ID do erro:** BC30008  
  
### Para corrigir este erro  
  
-   Inclua o nome do módulo sem referência em suas referências do projeto.
