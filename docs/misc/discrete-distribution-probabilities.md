# discrete_distribution::probabilities
Retorna as probabilidades armazenadas para a distribuição.  
  
## Sintaxe  
  
```  
std::vector<double> probabilities() const;  
```  
  
## Comentários  
 Essa função membro retorna os valores de probabilidade armazenado.

 **Cabeçalho:** \< random \>  
  
 **Namespace:** std
