# Exce&#231;&#245;es de solu&#231;&#227;o de problemas: System.Threading.ThreadStartException
A exceção que é lançada quando ocorre uma falha em um thread gerenciado depois do início do thread do sistema operacional subjacente, mas antes que o thread está pronto para executar o código do usuário.
