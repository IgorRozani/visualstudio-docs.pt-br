# Erro MSB3176 (MSBuild)
**MSB3176: Especificada a versão mínima necessária é maior do que a atual versão de publicação. Especifique uma versão menor ou igual a atual versão de publicação.**  
  
 Esse erro é gerado quando a versão mínima necessária do aplicativo é maior que a versão atual do manifesto de implantação.  
  
### Para corrigir este erro  
  
-   Especifique uma versão mínima necessária menor ou igual a atual versão de publicação.
