# &#39;Optional&#39; e &#39;ParamArray&#39; n&#227;o podem ser combinados
O `Optional` e `ParamArray` modificadores são mutuamente exclusivos.  
  
 **ID do erro:** BC30642  
  
### Para corrigir este erro  
  
-   Especifique o `ParamArray` ou `Optional`, mas não ambos.
