# CS2018 de erro do compilador
Não é possível localizar mensagens de arquivo 'cscmsgs.dll'  
  
 O arquivo. dll que contém o erro e mensagens de aviso do compilador não foi encontrado. Esse arquivo deve estar presente no mesmo diretório que os outros arquivos de suporte do compilador.
