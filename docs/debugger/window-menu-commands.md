# <a name="window-menu-commands"></a>Comandos do menu Janela
O **janela** menu fornece comandos para gerenciar o windows Spy + +.  
  
 **Em cascata**  
 Reorganiza todos os windows em um padrão em cascata.  
  
 **Lado a lado horizontalmente**  
 Reorganize todos os windows em um bloco horizontal padrão.  
  
 **Lado a lado verticalmente**  
 Reorganiza todos os windows em um padrão de bloco vertical.  
  
 **Organizar ícones**  
 Reorganiza os ícones de exibição, se o windows tiver sido minimizado.  
  
 **Fechar / fechar tudo**  
 Fecha uma ou todas as janelas no Spy + +.  
  
 **Atualizar**  
 Atualiza a exibição selecionada.  
  
 **Janela**  
 Permite a seleção de uma janela específica.
