# Caixa de di&#225;logo Pesquisa de Janela
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use essa caixa de diálogo para localizar e selecionar o nó de uma janela específica no  [De exibição do Windows](../debugger/windows-view.md).  Para exibir esta caixa de diálogo, mova o foco para a janela de visualização do Windows.  Em seguida, escolha  **Janela Localizar** partir do  **pesquisa** menu.  
  
 Você pode usar o  **Ferramenta de localização de** selecionar uma janela, ou você pode ser inserido o identificador de janela ou identificar a janela por classe e a legenda.  
  
 As configurações a seguir estão disponíveis:  
  
 **Ferramenta de localização**  
 A ferramenta arrastar e soltar, usada para selecionar uma janela.  
  
 **Ocultar Spy \+ \+**  
 Remove a janela principal do Spy \+ \+ e deixa a  **WindowSearch** caixa de diálogo visível em relação os outros aplicativos.  
  
 **Alça**  
 A identificação exclusiva da janela para pesquisar.  
  
 **Legenda**  
 O texto na legenda da janela para pesquisar.  
  
 **Classe**  
 A classe da janela para pesquisar.  
  
 **Direção da pesquisa para cima ou para baixo**  
 A direção inicial da pesquisa.
