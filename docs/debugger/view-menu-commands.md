# Comandos do menu Exibir
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

O  **Exibir** menu fornece comandos para controlar o Spy \+ \+ interface do usuário.  
  
 **Fonte**  
 Abre o  **fonte** caixa de diálogo, onde você pode selecionar uma fonte para uso na exibição Spy \+ \+ atual.  
  
 **Barra de ferramentas**  
 Oculta ou exibe a barra de ferramentas.  
  
 **Barra de status**  
 Oculta ou exibe a barra de Status.  
  
 **Propriedades**  
 Abre o  **Propriedades** caixa de diálogo para o modo ativo.
