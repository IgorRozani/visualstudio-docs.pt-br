# Prepara&#231;&#227;o de depura&#231;&#227;o: Servi&#231;os Windows
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Um serviço do Windows é um programa executado em segundo plano no Microsoft Windows.  Os exemplos incluem o serviço de Telnet e o serviço de tempo do Windows, que atualiza o relógio visível do computador.  Um serviço do Windows não pode ser executados no [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]; ele deve ser executado no contexto do Gerenciador de Controle de Serviços.  Para obter mais informações, consulte [Criando serviços do Windows](../Topic/How%20to:%20Create%20Windows%20Services.md), [Depurando aplicativos de serviços do Windows](../Topic/How%20to:%20Debug%20Windows%20Service%20Applications.md) e [Aplicativos de serviços do Windows](../Topic/Developing%20Windows%20Service%20Applications.md).
