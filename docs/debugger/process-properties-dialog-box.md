# Caixa de di&#225;logo Propriedades do Processo
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use essa caixa de diálogo para obter mais informações sobre um processo específico.  Para exibir esta caixa de diálogo, mova o foco para um  [Processos de modo de exibição](../debugger/processes-view.md) janela.  Selecione qualquer nó do processo na árvore e escolha  **Propriedades** da  **Exibir** menu.  
  
 Em sistemas de operacionais Windows 98 e Windows 95, somente uma versão modificada da  **Geral** guia é exibida.
