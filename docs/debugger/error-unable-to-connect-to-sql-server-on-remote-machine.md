# Erro: n&#227;o foi poss&#237;vel se conectar ao SQL Server na m&#225;quina remota
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Não é possível conectar\-se ao SQL Server no computador remoto *name*.  Acesso negado.  Verifique se você instalou o depurador remoto no computador remoto.  Se o computador remoto não estiver em um domínio ou se o Visual Studio for executado em uma conta local, o computador remoto deverá ter uma conta com o mesmo nome de usuário e senha que a conta local.  
  
### Para corrigir este erro  
  
-   Consulte [Depuração remota](../debugger/remote-debugging.md).
