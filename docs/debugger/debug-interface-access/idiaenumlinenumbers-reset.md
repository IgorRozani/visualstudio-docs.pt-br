# IDiaEnumLineNumbers::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Redefine uma seqüência de enumeração para o início.  
  
## Sintaxe  
  
```cpp#  
HRESULT Reset();  
```  
  
## Valor de retorno  
 Returns `S_OK`.
