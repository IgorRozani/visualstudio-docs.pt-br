# <a name="idiasymbolgetisstatic"></a>IDiaSymbol::get_isStatic
Recupera um sinalizador que especifica se a camada de função ou conversão tiver sido marcada como estática.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT get_isStatic(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pFlag`  
 [out] Retorna `TRUE` se a camada de função ou conversão tiver sido marcada como estático; caso contrário, retornará `FALSE`.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retorna `S_OK`; caso contrário, retorna `S_FALSE` ou um código de erro.  
  
> [!NOTE]
>  Um valor de retorno `S_FALSE` significa que a propriedade não está disponível para o símbolo.
