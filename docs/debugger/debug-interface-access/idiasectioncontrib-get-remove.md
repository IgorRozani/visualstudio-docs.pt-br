# <a name="idiasectioncontribgetremove"></a>IDiaSectionContrib::get_remove
Recupera um sinalizador que indica se a seção foi removida antes de ela se torna parte da imagem na memória.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT get_remove (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pRetVal`  
 [out] Retorna `TRUE` se a seção não deve ser adicionado à imagem de memória; caso contrário, retornará `FALSE`.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retornará `S_OK`. Retorna `S_FALSE` se não há suporte para essa propriedade. Caso contrário, retornará um código de erro.
