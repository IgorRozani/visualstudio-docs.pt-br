# <a name="idiasymbolgethasmanagedcode"></a>IDiaSymbol::get_hasManagedCode
Recupera um sinalizador que indica se o módulo contém o código gerenciado.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT get_hasManagedCode(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pFlag`  
 [out] Retorna `TRUE` se o módulo contém o código gerenciado; caso contrário, retornará `FALSE`, o código é um código não gerenciado.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retorna `S_OK`; caso contrário, retorna `S_FALSE` ou um código de erro.  
  
> [!NOTE]
>  Um valor de retorno `S_FALSE` significa que a propriedade não está disponível para o símbolo.  
  
## <a name="remarks"></a>Comentários  
 Essa propriedade está disponível na `SymTagCompilandDetails` tipo de símbolo (consulte [CompilandDetails](../../debugger/debug-interface-access/compilanddetails.md)).
