# <a name="idiasymbolgetlocationtype"></a>IDiaSymbol::get_locationType
Recupera o tipo de local de um símbolo de dados.  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
HRESULT get_locationType (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pRetVal`  
 [out] Retorna um valor da [enumeração LocationType](../../debugger/debug-interface-access/locationtype.md) enumeração que especifica o tipo de local de um símbolo de dados, como `static` ou `local`.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retorna `S_OK`; caso contrário, retorna `S_FALSE` ou um código de erro.  
  
> [!NOTE]
>  Um valor de retorno `S_FALSE` significa que a propriedade não está disponível para o símbolo.
