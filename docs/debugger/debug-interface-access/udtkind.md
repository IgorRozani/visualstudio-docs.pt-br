# <a name="udtkind"></a>UdtKind
Descreve a variedade de tipo definido pelo usuário (UDT).  
  
## <a name="syntax"></a>Sintaxe  
  
```C++  
enum UdtKind {   
   UdtStruct,  
   UdtClass,  
   UdtUnion,  
   UdtInterface  
};  
```  
  
## <a name="elements"></a>Elementos  
 UdtStruct  
 UDT é uma estrutura.  
  
 UdtClass  
 UDT é uma classe.  
  
 UdtUnion  
 É uma união de UDT.  
  
 UdtInterface  
 UDT é uma interface.  
  
## <a name="remarks"></a>Comentários  
 Os valores nesta enumeração são retornados pelo [: Get_udtkind](../../debugger/debug-interface-access/idiasymbol-get-udtkind.md) método.  
  
## <a name="requirements"></a>Requisitos  
 Cabeçalho: cvconst.h
