# Caixa de di&#225;logo Pesquisa de Processo
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use essa caixa de diálogo para localizar e selecionar o nó para um processo específico em  [Processos de modo de exibição](../debugger/processes-view.md).  Para exibir esta caixa de diálogo, mova o foco para um  **processos** janela de exibição.  Em seguida, escolha  **O processo de encontrar** da  **pesquisa** menu.  
  
 As configurações a seguir estão disponíveis:  
  
 **Process**  
 A identificação do processo para procurar.  
  
 **Module**  
 A seqüência de módulo para pesquisar.  
  
 **Direção da pesquisa para cima ou para baixo**  
 A direção inicial da pesquisa.
