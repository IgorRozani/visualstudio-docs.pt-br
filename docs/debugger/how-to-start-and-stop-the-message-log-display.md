# Como iniciar e parar a exibi&#231;&#227;o do log de mensagem
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Quando um  [Modo de exibição de mensagens](../debugger/messages-view.md) janela estiver ativa, um  **Iniciar registro** ou  **Parar registro** escolha aparecerá sobre o Spy \+ \+  **mensagens** menu e o correspondente  **Iniciar registro** ou  **Parar registro** botão da barra de ferramentas se torna ativo.  
  
### Para iniciar a exibição de log de mensagem  
  
1.  Do  **mensagens** menu, escolha  **Iniciar registro**.  
  
### Para interromper a exibição de log de mensagem  
  
1.  Do  **mensagens** menu, escolha  **Parar registro**.
