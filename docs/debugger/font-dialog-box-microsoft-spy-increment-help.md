# Caixa de di&#225;logo Fonte (Ajuda do Microsoft Spy++)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use essa caixa de diálogo para configurar a fonte de caracteres usada em um  [Modos de exibição Spy \+ \+](../debugger/spy-increment-views.md) janela.  Para exibir esta caixa de diálogo, expanda o  **Exibir** menu e escolha  **fonte**.  
  
 As configurações a seguir estão disponíveis:  
  
 **Fonte**  
 A fonte de caracteres para a janela Spy \+ \+ modos de exibição que tem foco.  
  
 **Estilo da fonte**  
 O estilo da fonte para a janela Spy \+ \+ modos de exibição que tem foco.  
  
 **Size** \(Tamanho\)  
 O tamanho da fonte para a janela Spy \+ \+ modos de exibição que tem foco.  
  
 **Salvar a fonte como padrão**  
 Salva as configurações de fonte atual como suas preferências ao abrir o Spy \+ \+ modos de exibição.
