# Como alterar fontes
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Você pode alterar a fonte, estilo da fonte e tamanho da fonte para o windows Spy \+ \+.  
  
### Para alterar as opções de fonte  
  
1.  Do  **Exibir** menu, escolha  **fonte**.  
  
2.  Escolha uma fonte, estilo da fonte e tamanho da fonte.  
  
3.  Clique em **OK**.  
  
 Selecionando  **Salvar fonte como padrão** fará com que todas as futuras Spy \+ \+ janelas para usá\-la.
