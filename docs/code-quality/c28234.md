# C28234
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28234 de aviso: a expressão de \_At\_ não se aplica à função atual  
  
 Esse aviso indicar que o valor de uma expressão de `_At_` não identifica um objeto acessível.
