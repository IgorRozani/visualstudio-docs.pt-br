# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28228: Parâmetro de anotação: não é possível usar tipo em anotações  
  
 Esse aviso indica que um parâmetro é do tipo que não tem suporte.  As anotações podem usar apenas um conjunto limitado de tipos como parâmetros.  Isso não deve ser possível se as macros padrão estão sendo usados para anotações; esse aviso não é esperado que seja considerado em uso típico.
