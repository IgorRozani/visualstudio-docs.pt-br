# <a name="c28224"></a>C28224
Aviso C28224: anotação requer parâmetros

 Esse aviso indica que a anotação nomeada é usada sem parâmetros, e pelo menos um parâmetro é necessário. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
