# <a name="how-to-generate-code-metrics-data"></a>Como gerar dados de métricas do código
Você pode optar por gerar resultados de métricas de código para uma solução inteira ou um projeto selecionado.  
  
 **Requisitos**  
  
 Visual Studio Enterprise  
  
### <a name="to-generate-code-metrics-results-for-an-entire-solution"></a>Para gerar resultados de métricas de código para uma solução inteira  
  
-   Clique em **menu analisar**e, em seguida, clique em **CalculateCode métricas para solução**.  
  
     \- ou -  
  
-   Em **Solution Explorer**, clique com botão direito a solução e, em seguida, clique em **CalculateCode métricas**.  
  
     \- ou -  
  
-   No **resultados de métricas de código** janela, clique no **CalculateCode métricas para solução** botão.  
  
     Os resultados são gerados e **resultados de métricas de código** janela é exibida.  
  
### <a name="to-generate-code-metrics-results-for-one-or-more-selected-projects"></a>Para gerar resultados de métricas de código para um ou mais projetos selecionados  
  
1.  Em **Solution Explorer**, selecione um ou mais projetos.  
  
2.  Clique em **analisar**e, em seguida, clique em **gerar as métricas de código para projetos selecionado (s)**.  
  
     \- ou -  
  
     Clique em suas seleções e, em seguida, clique em **CalculateCode métricas**.  
  
 Os resultados são gerados e **resultados de métricas de código** janela é exibida.  
  
### <a name="to-view-the-results-details"></a>Para exibir os detalhes de resultados  
  
-   Expanda a árvore no **hierarquia** coluna.
