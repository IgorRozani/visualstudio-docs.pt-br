# C28246
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28246: A anotação para a função '\<nome\>' \- parâmetro '\<parâmetro\>' não corresponde ao tipo do parâmetro  
  
 Um operador de \_\_deref foi aplicada a um tipo de ponteiro não ao criar uma anotação.
