# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28211: Nome de contexto estático esperado para SAL\_context  
  
 Esse aviso indicar que o operando à anotação de `_Static_context_` deve ser o nome de um uso de contexto definido.  Esse aviso nunca deve ocorrer no uso normal.
