# <a name="c28211"></a>C28211
Aviso C28211: nome do contexto estático esperado para SAL_context  
  
 Esse aviso indica que o operando para o `_Static_context_` anotação deve ser o nome de um contexto definido de ferramenta. Esse aviso nunca deve ocorrer no uso normal.
