# <a name="ca0055"></a>CA0055
Não foi possível carregar o arquivo:\<caminho >  
  
 A ferramenta de análise de código não pôde carregar o arquivo de destino foi especificado para análise. Isso ocorre se o arquivo não foi encontrado ou o acesso ao arquivo foi negado. Normalmente, isso ocorre se o usuário optou por substituir a propriedade do assembly de entrada em [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], ou invocado fxcopcmd.exe manualmente.
