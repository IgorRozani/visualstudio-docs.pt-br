# C1253
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C1253 de aviso: Não é possível carregar o arquivo de modelo.  
  
 A ferramenta de Análise de Código relata este aviso quando há um erro interno no arquivo de modelo, não no código que está sendo usado.
