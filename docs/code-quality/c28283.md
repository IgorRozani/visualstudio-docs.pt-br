# <a name="c28283"></a>C28283
Aviso C28283: para símbolo, a especificação de tamanho especificado ainda não é suportada  
  
 O aviso indica que a anotação sentinel"especificação de tamanho" recebeu um valor diferente de zero. Essencialmente, o chamador tentou dizer que a cadeia de caracteres é terminada por um caractere diferente de zero binário.
