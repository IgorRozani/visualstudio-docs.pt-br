# C28232
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28232 de aviso: \_Pre\_, \_Post\_, ou \_Deref\_ não aplicado a qualquer anotação  
  
 Esse aviso indica que `_Pre_`, `_Post_`, o operador ou de `_Deref_` aparecem em uma expressão de anotação sem uma anotação funcional subsequente; o modificador foi ignorado, mas isso indica uma anotação incorretamente codificada.
