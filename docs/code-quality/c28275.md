# <a name="c28275"></a>C28275
Aviso C28275: O parâmetro para _Macro_value\_ é nulo

 Esse aviso indica que há um erro interno no arquivo de modelo, não no código que está sendo analisado. O *macroValue* função foi chamada sem um parâmetro.
