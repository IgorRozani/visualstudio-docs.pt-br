# C6298
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C6298: usando uma cadeia de caracteres \<ponteiro\> como um argumento modificável de cadeia de caracteres: isso tentará gravar na memória somente leitura estática e causará falhas aleatórias  
  
 Esse aviso indica o uso de uma cadeia de caracteres constante como um argumento para uma função que pode alterar o conteúdo dessa cadeia de caracteres.  Como o compilador atribui cadeias de caracteres constantes na memória somente leitura, estática todas as tentativas de alterar violações de acesso da causa de falhas e aleatórios.  
  
 Isso pode ser evitado para armazenar a cadeia de caracteres constante em uma matriz local e usando a matriz como o argumento para a função.  
  
## Exemplo  
 O código de exemplo a seguir gera este aviso:  
  
```  
#include <windows.h>  
#include <stdio.h>  
  
void f()  
{  
    STARTUPINFO si;  
    PROCESS_INFORMATION pi;  
  
    ZeroMemory( &si, sizeof( si ) );  
    si.cb = sizeof( si );  
    ZeroMemory( &pi, sizeof( pi ) );  
    if( !CreateProcess(NULL,  
                      "\"c:\\Windows\\system32\\calc.exe\"",  
                      NULL,               
                      NULL,                
                      FALSE,              
                      0,                   
                      NULL,               
                      NULL,                
                      &si,                
                      &pi ) )   
    {  
        puts( "CreateProcess failed." );  
        return;  
    }  
  
    // Wait until child process exits.  
    WaitForSingleObject( pi.hProcess, INFINITE );  
  
    // Close process and thread handles.   
    CloseHandle( pi.hProcess );  
    CloseHandle( pi.hThread );  
}  
  
```  
  
 Para corrigir esse aviso, use o seguinte código de exemplo:  
  
```  
#include <windows.h>  
#include <stdio.h>  
  
void f( )  
{  
    STARTUPINFO si;  
    PROCESS_INFORMATION pi;  
  
    char szCmdLine[] = "\"c:\\Windows\\system32\\calc.exe\"";  
    ZeroMemory( &si, sizeof( si ) );  
    si.cb = sizeof( si );  
    ZeroMemory( &pi, sizeof( pi ) );  
  
    if( !CreateProcess(NULL,  
                      szCmdLine,  
                      NULL,               
                      NULL,                
                      FALSE,              
                      0,                   
                      NULL,               
                      NULL,                
                      &si,                
                      &pi ) )   
    {  
        puts( "CreateProcess failed." );  
        return;  
    }  
  
    // Wait until child process exits.  
    WaitForSingleObject( pi.hProcess, INFINITE );  
  
    // Close process and thread handles.   
    CloseHandle( pi.hProcess );  
    CloseHandle( pi.hThread );  
}  
```
