# <a name="c28280"></a>C28280
Aviso C28280: para símbolo, um 'fim' foi encontrado sem um correspondente 'Início'  
  
 A linguagem de anotação suporta um início e fim (`{` e `}` em C) construção e o emparelhamento ficou desbalanceada. Essa situação pode ser evitada se as macros são usadas.
