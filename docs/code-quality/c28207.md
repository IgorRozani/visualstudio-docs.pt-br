# C28207
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28207: \<expressão\>: O operando da esquerda é uma struct, utilize .  
  
 Este aviso é relatado quando um ponto de desreferência de struct \(.\) é esperado.
