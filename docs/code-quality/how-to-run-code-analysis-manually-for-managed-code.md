# Como executar a an&#225;lise de c&#243;digo manualmente para c&#243;digo gerenciado
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

A ferramenta de análise de código fornecem informações sobre defeitos possíveis em seu código\-fonte.  Você pode executar a análise de código automaticamente a cada construção de um projeto de código, e você também pode executar a análise de código manualmente.  As regras que são verificadas quando a análise de código é executada são especificadas na página do código de páginas de propriedades do projeto.  Para obter mais informações, consulte [Como configurar a análise de código para um projeto de código gerenciado](../code-quality/how-to-configure-code-analysis-for-a-managed-code-project.md)  
  
### Para executar manualmente a análise de código  
  
1.  Em **Gerenciador de Soluções**, clique no projeto.  
  
2.  No menu de **Analisar** , clique **Executar análise de código sobre** *Project Name*.
