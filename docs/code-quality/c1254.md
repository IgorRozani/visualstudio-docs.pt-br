# C1254
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C1254: Incompatibilidade de versão de plug\-in: a versão não coincide com a versão do driver de PREfast  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno com a versão de plug\-in, não no código que está sendo analisado.
