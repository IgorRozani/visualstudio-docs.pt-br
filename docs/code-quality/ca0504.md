# <a name="ca0504"></a>CA0504
O diretório de regra foi ignorado porque ele não existe  
  
 Aviso CA504 é gerado quando um diretório especificado no **CodeAnalysisRuleDirectories** não foi possível localizar uma propriedade em um arquivo de configuração do projeto ou compilação. Para resolver esse problema, insira um caminho de diretório válido.
