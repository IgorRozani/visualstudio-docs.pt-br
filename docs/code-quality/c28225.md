# <a name="c28225"></a>C28225
Aviso C28225: não localizou o número correto de parâmetros necessários na anotação  
  
 Esse aviso indica que a anotação nomeada é usada com o número incorreto de parâmetros. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
