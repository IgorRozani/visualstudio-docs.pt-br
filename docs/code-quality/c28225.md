# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28225: Não encontrado o número correto de parâmetros necessários na anotação  
  
 Esse aviso indica que a anotação nomeada é usada com o número incorreto de parâmetros.  Isso não deve ser possível se as macros padrão estão sendo usados para anotações; esse aviso não é esperado que seja considerado em uso típico.
