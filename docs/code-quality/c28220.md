# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28220: Expressão Integer esperada para anotação do parâmetro na anotação  
  
 Esse aviso indica que um parâmetro para uma anotação é esperado ser uma expressão de inteiro, e qualquer outro tipo foi encontrado.  Isso geralmente indica uma macro incorretamente codificado da anotação.
