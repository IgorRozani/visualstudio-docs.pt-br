# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28289 de aviso: Para a função: ReadableTo ou WritableTo não tinha as especificações associado como um parâmetro.  
  
 A ferramenta de análise de código relata esse aviso quando a anotação da função\/parâmetro miscoded como observada.
