# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28212: Expressão de ponteiro esperada para anotação  
  
 Esse aviso indica que o parâmetro numerado a uma anotação \(não a função que está sendo anotada\) é esperado que seja um ponteiro ou um tipo de matriz, mas qualquer outro tipo foi encontrado.  A anotação precisa ser corrigida.
