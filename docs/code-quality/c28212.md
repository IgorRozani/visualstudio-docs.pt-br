# <a name="c28212"></a>C28212
Aviso C28212: expressão de ponteiro esperada para anotação  
  
 Esse aviso indica que o parâmetro numerado em uma anotação (não a função que está sendo anotada) deve ser um ponteiro ou um tipo de matriz, mas algum outro tipo foi encontrado. A anotação deve ser corrigido.
