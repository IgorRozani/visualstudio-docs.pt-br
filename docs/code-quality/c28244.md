# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

aviso C28244: A anotação da função tem uma anotação de parâmetro\/externa não analisável  
  
 Isso não deve estar possível se macros de padrão estão sendo usados para anotações; esse aviso não é esperado que seja considerado em uso típico.
