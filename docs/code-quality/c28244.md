# <a name="c28244"></a>C28244
Aviso C28244: A anotação para função tem uma anotação de parâmetro/externa não analisável  
  
 No momento não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
