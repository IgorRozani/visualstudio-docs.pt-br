# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28217 de aviso: Para a função, o número de parâmetros para a anotação não corresponde ao localizado no arquivo  
  
 As anotações na linha atual e na linha da mensagem forem inconsistentes.  Isso não deve ser possível se as macros padrão estão sendo usados para anotações; esse aviso não é esperado que seja considerado no uso normal.
