# <a name="c28241"></a>C28241
Aviso C28241: A anotação para função no parâmetro não é reconhecida

 Um nome de anotação não reconhecido foi usado. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
