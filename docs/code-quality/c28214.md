# <a name="c28214"></a>C28214
Aviso C28214: nomes de parâmetro de atributo devem ser p1... p9

 Esse aviso indica que, quando você construir uma declaração de anotação, os nomes de parâmetro são limitados a p1... p9. Esse aviso nunca deve ocorrer no uso normal.
