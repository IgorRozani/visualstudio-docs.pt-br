# <a name="c28160"></a>C28160
Aviso C28160: anotação de erro  
  
 Esse aviso é relatado quando um `__drv_error` anotação foi encontrada. Esta anotação é usada para sinalizar práticas de codificação que devem ser corrigidas e podem ser usadas com um `__drv_when` anotação para indicar a combinações específicas de parâmetros.
