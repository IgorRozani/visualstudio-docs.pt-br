# C28310
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

aviso 28310: A anotação \<annotation\_name\> em \<function\> \<parameter\> não tem versão de SAL.  
  
 Todas as anotações de SAL usadas no código\-fonte têm uma versão de anotação aplicada pelo uso de SAL\_name.  Isso precisa ser corrigido na definição de macro.  
  
 Esse aviso é relatado apenas uma vez pela declaração.  Inspecione o restante da declaração para uma SAL mais obsoleta.
