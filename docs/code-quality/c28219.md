# <a name="c28219"></a>C28219
Aviso C28219: membro de enumeração esperada para anotação do parâmetro na anotação

 Um parâmetro em uma anotação deve ser um membro do nomeado `enum` tipo e algum outro símbolo foi encontrado; usar um membro que `enum` tipo. Isso geralmente indica uma macro de anotação codificado incorretamente.
