# <a name="c28221"></a>C28221
Aviso C28221: esperada para o parâmetro na anotação de expressão de cadeia de caracteres

 Esse aviso indica que um parâmetro em uma anotação deve ser uma cadeia de caracteres e outro tipo foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente e não deve ser visto no uso normal.
