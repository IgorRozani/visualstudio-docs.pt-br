# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28221: Expressão String esperada para o parâmetro na anotação  
  
 Este aviso indica que se espera que um parâmetro de uma anotação seja uma cadeia de caracteres, e que algum outro tipo foi encontrado.  Isso geralmente indica uma macro de anotação mal\-codificada e não deve ocorrer em uso normal.
