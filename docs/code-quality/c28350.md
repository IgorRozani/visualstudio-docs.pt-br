# C28350
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28350: A anotação \<anotação\> descreve uma situação que não é condicionalmente aplicável.  
  
 Esse aviso é gerado geralmente quando uma anotação é aplicada onde o tipo C\/C\+\+ está sendo inspecionado.
