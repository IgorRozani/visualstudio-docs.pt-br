# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28290: A anotação para a função contém mais Externals do que o número real de parâmetros  
  
 A ferramenta de análise de código relata esse aviso quando a anotação para a função Externals contém mais do que o número real de parâmetros.
