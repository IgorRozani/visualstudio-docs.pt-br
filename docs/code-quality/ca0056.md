# <a name="ca0056"></a>CA0056
Um arquivo de projeto possui uma versão incorreta da ferramenta de análise.  
  
 Este aviso ocorre se FxCopCmd.exe é invocada manualmente, com um arquivo de projeto que tem uma versão de ferramenta de análise de código incorreto.
