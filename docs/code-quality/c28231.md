# C28231
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28231: A anotação só é válida em matriz  
  
 Esse aviso indica que um argumento para uma anotação deve ser uma matriz, e qualquer outro tipo foi encontrado.
