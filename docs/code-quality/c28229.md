# <a name="c28229"></a>C28229
Aviso C28229: anotação não oferece suporte a parâmetros

 Esse aviso indica que uma anotação foi usada com um parâmetro quando a anotação é declarada sem parâmetros. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
