# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28230: O tipo de parâmetro não tem nenhum membro.  
  
 Esse aviso indica que um argumento para uma anotação tentar acessar `struct`, `class`, ou `union` e o membro nomeada não existe.
