# <a name="c28230"></a>C28230
Aviso C28230: O tipo de parâmetro não tem nenhum membro.  
  
 Esse aviso indica que um argumento para uma anotação tenta acessar um `struct`, `class`, ou `union` e o membro nomeado não existe.
