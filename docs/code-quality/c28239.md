# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28239 de aviso: A anotação no parâmetro tem ambos, avaliar o stringValue, ou o longValue; e paramn\=xxx.  Use somente paramn\=xxx  
  
 Um erro interno ocorreu no arquivo de modelo do PREfast.  Esse aviso não deve ocorrer no uso típico.
