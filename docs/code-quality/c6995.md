# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C6995 de aviso: Falha ao salvar o arquivo de log de XML  
  
 Esse aviso indicar que a ferramenta de análise de código não pode criar o log de defeito, que é a saída da análise de código.  
  
 Esse erro pode indicar um erro de disco ou indicar que você não tem permissão para criar um arquivo no diretório especificado.
