# <a name="c6995"></a>C6995
Aviso C6995: Falha ao salvar arquivo de Log XML  
  
 Esse aviso indica que a ferramenta de análise de código não é possível criar o log de defeito, que é a saída da análise de código.  
  
 Esse erro pode indicar um erro de disco ou indicar que você não tem permissão para criar um arquivo no diretório especificado.
