# <a name="c28301"></a>C28301
Aviso C28301: não há anotações para a primeira declaração de \<função >.\< note1 > consulte \<filename > (\<linha >). \<2 >  
  
 Esse aviso é relatado quando anotações não encontradas durante a primeira declaração de uma determinada função.
