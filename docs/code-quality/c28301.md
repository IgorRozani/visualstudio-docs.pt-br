# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28301: Nenhuma anotação para a primeira declaração \<função\>.\<nota\> Consulte \<nome do arquivo\>\(\<linha\>\). \<nota2\>  
  
 Esse aviso é relatado quando as anotações não foram encontradas na primeira declaração de uma função específica.
