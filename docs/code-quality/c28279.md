# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28279: Para o símbolo, foi encontrado um 'begin' sem um 'end' correspondente  
  
 O idioma da anotação oferece suporte a um início e uma extremidade \(`{` e `}` C\) na construção, e o emparelhamento atingiram desequilibrado.  Esta situação pode ser evitado se as macros são usados.
