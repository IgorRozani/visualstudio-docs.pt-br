# <a name="ca0503"></a>CA0503
A propriedade foi preterida. Use a propriedade de substituição  
  
 CA0503 de aviso é gerado quando uma propriedade de análise de código obsoleto contém um valor no arquivo de configuração de compilação ou de projeto. Para resolver este aviso, use a propriedade especificada que substitui-lo.
