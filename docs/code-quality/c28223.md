# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

C28223 de aviso: Não encontrou o token\/identificador esperados da anotação, parâmetro  
  
 Esse aviso indica que um parâmetro para uma anotação é esperado que seja um identificador, e qualquer outro símbolo foi encontrado.  Isso geralmente indica uma macro incorretamente codificado da anotação.  O uso de palavras\-chave c ou C\+\+ 2.0 nessa posição provocará este erro.
