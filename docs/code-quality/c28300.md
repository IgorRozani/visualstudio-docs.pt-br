# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28300: \<parameter\_name\>: Operandos da expressão de tipos incompatíveis para o operador \<operator\_name\>  
  
 Esse aviso é relatado quando o operando em um parâmetro não é compatível com o operador.
