# C28206
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28206: \<expressão\> : O operando da esquerda aponta para um struct, utilize `->`  
  
 Esse aviso é relatado quando o ponteiro da estrutura cancelará a notação `->` foi esperado.
