# C6705
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C6705: Erro de anotação esperava \<número\_esperado\> argumentos para a anotação \<parâmetro\> foram encontrados \<número\_real\>.  
  
 Este erro é reportado quando há um erro nas anotações.
