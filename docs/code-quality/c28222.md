# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

o aviso 28222: \_Yes\_, \_No\_, ou \_Maybe\_ esperado para a anotação  
  
 Esse aviso indica que um parâmetro para uma anotação é esperado ser um dos símbolos `_Yes_`, `_No_`, ou `_Maybe_`, e qualquer outro símbolo foi encontrado.  Isso geralmente indica uma macro incorretamente codificado da anotação.
