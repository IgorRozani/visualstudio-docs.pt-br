# <a name="c28222"></a>C28222
Aviso 28222: Sim\_, _No\_, ou _Maybe\_ esperada para anotação  
  
 Esse aviso indica que um parâmetro em uma anotação deve ser um dos símbolos `_Yes_`, `_No_`, ou `_Maybe_`, e alguns outros símbolo foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente.
