# C6702
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C6702: O valor não é um valor de cadeia de caracteres: \<cadeia de caracteres\>  
  
 Este erro é reportado quando há um erro nas anotações.
