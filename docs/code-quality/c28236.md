# <a name="c28236"></a>C28236
Aviso C28236: A anotação não pode ser usada em uma expressão

 Esse aviso indica que foi feita uma tentativa para usar uma função declarada com uma anotação em um contexto de expressão. Isso não deve ser possível se as macros padrão estão sendo usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
