# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28286: Para a função, erro de sintaxe perto do final  
  
 A ferramenta de análise de código relata esse aviso quando um erro provável é encontrado no arquivo de modelo.  Alguns erros do arquivo de origem também podem causar esses erros.
