# <a name="c28286"></a>C28286
Aviso C28286: para a função, erro de sintaxe perto do fim

 A ferramenta de análise de código informa este aviso quando um erro provável é encontrado no arquivo de modelo. Alguns erros de arquivo de origem também podem causar esses erros.
