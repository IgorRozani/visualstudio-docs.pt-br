# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28204: \<função\> : Apenas uma dessa sobrecarga e aquela descrita em \<nome de arquivo\>\(\<linha\>\) são anotadas para o \<paramname\>: ambos ou nenhum devem ser anotados.  
  
 Este erro é reportado quando há um erro nas anotações.
