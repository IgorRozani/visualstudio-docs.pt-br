# <a name="c28204"></a>C28204
Aviso C28204: \<função >: somente um essa sobrecarga e a \<filename > (\<linha >) são anotados para \<paramname >: ambos ou nenhum deles deve ser anotado.  
  
 Esse aviso é relatado quando há um erro nas anotações.
