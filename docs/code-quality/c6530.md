# <a name="c6530"></a>C6530
Aviso 6530: não reconhecido de estilo de cadeia de caracteres de formato \<nome >

> [!NOTE]
>  Este aviso ocorre somente no código que está usando uma versão preterida da linguagem de anotação do código-fonte (SAL). É recomendável que a porta de seu código para usar a versão mais recente de SAL. Para obter mais informações, consulte [usando anotações de SAL para reduzir defeitos de código C/C++](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md).

 Esse aviso indica que a propriedade FormatString está usando um valor diferente de scanf ou printf. Para corrigir esse aviso, revise o código e use um valor válido para a propriedade de estilo.

## <a name="example"></a>Exemplo
 O código a seguir gera este aviso devido a um erro de digitação no `Style` propriedade:

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_FormatString(Style="printfd")] char *px);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([FormatString(Style="printfd")] char *px);

```

 Para corrigir esse aviso, use um estilo válido, conforme mostrado no código a seguir:

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
void f([SA_FormatString(Style="printf")] char *px);

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
void f([FormatString(Style="printf")] char *px);
```
