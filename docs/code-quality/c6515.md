# <a name="c6515"></a>C6515
Aviso C6515 - anotação inválida: \<nome > propriedade só pode ser usada em valores de tipo de ponteiro  
  
 Esse aviso indica que uma propriedade para uso em ponteiros foi aplicada a um tipo de ponteiro não. Para obter uma lista de propriedades de anotação, consulte [propriedades de anotação](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Exemplo  
 O código a seguir gera este aviso:  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 Para corrigir esse aviso, use o seguinte código:  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
