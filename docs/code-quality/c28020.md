# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28020: A expressão \<expr\> não é verdadeira nesta chamada  
  
 Esse aviso é relatado quando a expressão de \_Satisfies\_ listada não for verdadeira.  Geralmente isso indica um parâmetro incorreto.  
  
 Se isso ocorrer em uma declaração de função, as anotações indica uma condição impossível.
