# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Aviso C28213: A anotação `_Use_decl_annotations_` deve ser usada para fazer referência, sem modificação, a uma declaração anterior. \<why\>  
  
 Esse aviso é relatado quando uma instrução anterior for referenciada sem a anotação necessária de `_Use_decl_annotations_` .
