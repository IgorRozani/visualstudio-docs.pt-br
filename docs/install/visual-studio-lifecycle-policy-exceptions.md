# Exce&#231;&#245;es de pol&#237;tica de ciclo de vida do Visual Studio
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual Studio inclui um conjunto de compiladores, linguagens, tempos de execução, ambientes e outros recursos que permitem o desenvolvimento para várias plataformas da Microsoft. Como uma conveniência para nossos clientes do Visual Studio, Visual Studio poderá instalar determinados SDKs da Microsoft e outros componentes da Microsoft direcionados e compatíveis com essas plataformas da Microsoft. Esses componentes podem ser licenciados e suportados sob seus próprios termos e políticas.
