# Como localizar a chave do produto do Visual Studio
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Talvez você queira localizar a chave do produto para [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] quando você estiver trabalhando com o atendimento ao cliente, atualizando de uma versão de avaliação ou atualizando seus próprios registros.  
  
> [!NOTE]
>  Para localizar a chave do produto para [!INCLUDE[esprtfs](../code-quality/includes/esprtfs_md.md)], siga as etapas no [Localizar ou alterar a chave do produto para o Team Foundation Server](../Topic/Locate%20or%20Change%20the%20Product%20Key%20for%20Team%20Foundation%20Server.md).  
  
### Para localizar a chave do produto para o Visual Studio  
  
-   Procure a chave do produto em um destes locais:  
  
    -   Na mídia de instalação \(se aplicável\).  
  
    -   Sobre o [as assinaturas do MSDN](http://go.microsoft.com/fwlink/?LinkId=206363) página do site da Microsoft.  
  
    -   Documentação ou outros comunicados do fornecedor de quem você adquiriu o software.
