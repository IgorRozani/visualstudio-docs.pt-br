# <a name="visual-studio-architecture-tooling-guidance"></a>Orientação de ferramentas de arquitetura do Visual Studio
Este guia de ferramentas de arquitetura é um pacote para download de orientação prática sobre o uso de ferramentas de modelagem do Visual Studio 2010. Ele apresenta uma visão geral de como usar as ferramentas de modelagem de maneira integrada e aborda os cenários mais comuns em que elas são usadas.
