# Como: abrir a janela de fontes de dados
A janela **Fontes de Dados** exibe as fontes de dados no projeto.  É possível usar a janela **Fontes de Dados** para criar uma interface de usuário dos controles de associação de dados arrastando itens da janela para formulários no projeto.  Consulte [Associar controles dos Windows Forms a dados no Visual Studio](../data-tools/bind-windows-forms-controls-to-data-in-visual-studio.md).  
  
 É possível popular a janela **Fontes de Dados** executando\-se o Assistente de **Configuração da Fonte de Dados**.  Para obter mais informações, consulte [Assistente para Configuração da Fonte de Dados](../data-tools/media/data-source-configuration-wizard.png).  
  
### Para abrir a janela Fontes de Dados  
  
-   Na barra de menus, escolha **Exibir**, **Outras Janelas**, **Fontes de Dados** \(ou selecione as teclas Shift\+Alt\+D\).  
  
    > [!IMPORTANT]
    >  Só será possível adicionar uma fonte de dados a um projeto se der suporte à criação e ao trabalho com fontes de dados.  Por exemplo, não é possível abrir a janela **Fontes de Dados** em um projeto de um aplicativo da Windows Store.
