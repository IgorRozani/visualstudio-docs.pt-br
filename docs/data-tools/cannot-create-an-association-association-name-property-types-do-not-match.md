# <a name="cannot-create-an-association-ltassociation-namegt---property-types-do-not-match"></a>Não é possível criar uma associação &lt;nome da associação&gt; -tipos de propriedade não coincidem
Não é possível criar uma associação \<nome da associação >-tipos de propriedade não coincidem. Propriedades não têm tipos correspondentes: \<nomes de propriedade >.  
  
 Associações definidas pelo selecionado **propriedades de associação** no **Editor de associação** caixa de diálogo. As propriedades em cada lado de associação devem ser do mesmo tipo de dados.  
  
 As propriedades listadas na mensagem não têm os mesmos tipos de dados.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Examine a mensagem e observe as propriedades chamadas na mensagem.  
  
2.  Clique em **Okey** para ignorar a caixa de diálogo.  
  
3.  Inspecione o **propriedades de associação** e selecione Propriedades do mesmo tipo de dados.  
  
4.  Clique em **OK**.
