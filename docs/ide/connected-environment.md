# <a name="connected-environment"></a>Ambiente conectado
O Visual Studio é um ambiente conectado:  

- Você pode entrar Visual Studio para habilitar o acesso a todos os serviços associados à sua conta de usuário e também a todas as contas vinculadas a essa conta. As configurações da conta usam perfis móveis sempre que você usa o Visual Studio.  
- O Visual Studio informa quando um componente da instalação, como um SDK, tem uma atualização disponível.  
- Você pode baixar extensões e pacotes do NuGet.  
- E muito mais...
