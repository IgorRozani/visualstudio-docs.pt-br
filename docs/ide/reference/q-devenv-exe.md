# <a name="-devenvexe"></a>/? (devenv.exe)
Exibe uma caixa de mensagem listando todas as opções `devenv`, com uma breve descrição de cada uma.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
devenv /?  
```
