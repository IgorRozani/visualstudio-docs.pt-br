# <a name="globalize-applications"></a>Globalizar aplicativos

Globalização é o processo de projetar e desenvolver um produto de software que funciona para várias culturas. O artigo a seguir se aplica a páginas do Windows Forms e do Web Forms.

 [Classes específicas da cultura para formulários do Windows e formulários Web](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) Aborda classes que formatam datas, hora, números, moeda e outras informações de acordo com a configuração de cultura.
