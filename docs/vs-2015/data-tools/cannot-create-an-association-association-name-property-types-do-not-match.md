# <a name="cannot-create-an-association-ltassociation-namegt---property-types-do-not-match"></a>Não é possível criar uma associação &lt;nome da associação&gt; -tipos de propriedade não coincidem
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [não é possível criar uma associação &lt;nome da associação&gt; -tipos de propriedade não coincidem](https://docs.microsoft.com/visualstudio/data-tools/cannot-create-an-association-association-name-property-types-do-not-match).  
  
  
Não é possível criar uma associação \<nome da associação >-tipos de propriedade não correspondem. Propriedades não têm tipos de correspondência: \<nomes de propriedade >.  
  
 Associações são definidas por selecionado **propriedades de associação** na **Editor de associação** caixa de diálogo. As propriedades em cada lado de associação devem ser do mesmo tipo de dados.  
  
 As propriedades listadas na mensagem não têm os mesmos tipos de dados.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Examine a mensagem e observe as propriedades chamadas na mensagem.  
  
2.  Clique em **Okey** para descartar a caixa de diálogo.  
  
3.  Inspecione o **propriedades de associação** e selecione Propriedades do mesmo tipo de dados.  
  
4.  Clique em **OK**.
