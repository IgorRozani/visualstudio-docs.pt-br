# <a name="c28022"></a>C28022
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28022](https://docs.microsoft.com/visualstudio/code-quality/c28022).  
  
Aviso C28022: as classes de função \<classlist1 > sobre esta função não coincidem com as classes de função \<classlist2 > no typedef usado para defini-lo.  
  
 Esse aviso é relatado quando ocorre um erro nas anotações. A definição de tipo e a própria função têm `_Function_class_` anotações, mas eles não coincidem. Se ambos forem usados deve corresponder.
