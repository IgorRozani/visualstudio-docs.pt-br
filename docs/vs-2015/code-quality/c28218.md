# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28218](https://docs.microsoft.com/visualstudio/code-quality/c28218).  
  
Aviso C28218: para parâmetro de função, o parâmetro de anotação não corresponde ao encontrado no arquivo  
  
 As anotações na linha atual e na linha da mensagem são inconsistentes. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
