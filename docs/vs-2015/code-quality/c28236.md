[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28236](https://docs.microsoft.com/visualstudio/code-quality/c28236).  
  
Aviso C28236: A anotação não pode ser usada em uma expressão  
  
 Este aviso indica que foi feita uma tentativa para usar uma função declarada como uma anotação em um contexto de expressão. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
