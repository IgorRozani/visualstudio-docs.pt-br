# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28021](https://docs.microsoft.com/visualstudio/code-quality/c28021).  
  
Aviso C28021: O parâmetro \<param > que está sendo anotado com \<anno > deve ser um ponteiro  
  
 Esse aviso é relatado quando o objeto que está sendo anotado não é um tipo de ponteiro. Essa anotação não pode ser usada com `void` ou tipos integrais.
