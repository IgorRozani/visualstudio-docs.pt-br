# <a name="c28286"></a>C28286
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28286](https://docs.microsoft.com/visualstudio/code-quality/c28286).  
  
Aviso C28286: para a função, erro de sintaxe perto do fim  
  
 A ferramenta de análise de código relata esse aviso quando um erro provável é encontrado no arquivo de modelo. Alguns erros de arquivo de código-fonte também podem causar esses erros.
