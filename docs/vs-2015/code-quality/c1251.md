# <a name="c1251"></a>C1251
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C1251](https://docs.microsoft.com/visualstudio/code-quality/c1251).  
  
Aviso C1251: não é possível carregar modelos.  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno no arquivo de modelo, não no código que está sendo analisado.
