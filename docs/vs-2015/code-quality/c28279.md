# <a name="c28279"></a>C28279
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28279: para símbolo, um 'begin' foi encontrado sem um 'end' correspondente  
  
 A linguagem de anotação dá suporte a uma begin e end (`{` e `}` em C) constructo e o emparelhamento ficou desbalanceada. Essa situação pode ser evitada se as macros são usadas.
