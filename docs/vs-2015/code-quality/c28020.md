# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28020](https://docs.microsoft.com/visualstudio/code-quality/c28020).  
  
Aviso C28020: A expressão \<expr > não é verdadeira nesta chamada  
  
 Esse aviso é relatado quando o _Satisfies\_ expressão listado não é verdadeira. Frequentemente, isso indica um parâmetro incorreto.  
  
 Se isso ocorrer em uma declaração de função, as anotações indicam uma condição de impossível.
