[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28226](https://docs.microsoft.com/visualstudio/code-quality/c28226).  
  
Aviso C28226: Anotação não pode ser também um PrimOp (na declaração atual)  
  
 Este aviso indica que a anotação nomeada está sendo declarada como um PrimOp e também foi declarada anteriormente como uma anotação de normal. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso normal.
