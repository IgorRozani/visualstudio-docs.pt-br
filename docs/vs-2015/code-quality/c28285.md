# <a name="c28285"></a>C28285
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28285: para a função, erro de sintaxe no parâmetro  
  
 A ferramenta de análise de código relata esse aviso quando um erro provável é encontrado no arquivo de modelo. Alguns erros de arquivo de código-fonte também podem causar esses erros.
