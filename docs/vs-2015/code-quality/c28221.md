# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28221: expressão String esperada para o parâmetro na anotação  
  
 Este aviso indica que um parâmetro em uma anotação deve ser uma cadeia de caracteres, e algum outro tipo foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente e não deve ser visto no uso normal.
