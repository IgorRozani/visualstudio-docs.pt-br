[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aviso C28219: membro de enumeração esperada para anotação do parâmetro na anotação  
  
 Um parâmetro em uma anotação deve ser um membro nomeado `enum` foi encontrado o tipo e alguns outros símbolos; usar um membro que `enum` tipo. Isso geralmente indica uma macro de anotação codificado incorretamente.
