# <a name="c28300"></a>C28300
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28300](https://docs.microsoft.com/visualstudio/code-quality/c28300).  
  
Aviso C28300: < parameter_name >: operandos de expressão de tipos incompatíveis para o operador < operator_name >  
  
 Esse aviso é relatado quando os tipos de operando em um parâmetro não são compatíveis com o operador.
