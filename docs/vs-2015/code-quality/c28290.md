# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28290](https://docs.microsoft.com/visualstudio/code-quality/c28290).  
  
Aviso C28290: a anotação para função contém mais Externals que o número real de parâmetros  
  
 A ferramenta de análise de código relata esse aviso quando a anotação para a função contém mais Externals do que o número real de parâmetros.
