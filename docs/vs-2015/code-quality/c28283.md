# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28283](https://docs.microsoft.com/visualstudio/code-quality/c28283).  
  
Aviso C28283: para símbolo, a especificação de tamanho especificado ainda não é suportada  
  
 O aviso indica que a anotação de sentinela"especificação de tamanho" recebeu um valor diferente de zero. Essencialmente, o chamador tentou dizer que a cadeia de caracteres é terminada por um caractere diferente de zero binário.
