# <a name="c28275"></a>C28275
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28275](https://docs.microsoft.com/visualstudio/code-quality/c28275).  
  
Aviso C28275: O parâmetro para macro_value _\_ é nulo  
  
 Este aviso indica que há um erro interno no arquivo de modelo, não no código que está sendo analisado. O *macroValue* função foi chamada sem parâmetros.
