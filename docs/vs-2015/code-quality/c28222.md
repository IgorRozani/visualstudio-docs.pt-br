# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28222](https://docs.microsoft.com/visualstudio/code-quality/c28222).  
  
Aviso 28222: Sim\_, já _\_, ou Maybe\_ esperada para anotação  
  
 Este aviso indica que um parâmetro em uma anotação deve ser um dos símbolos `_Yes_`, `_No_`, ou `_Maybe_`, e alguns outros símbolos foi encontrado. Isso geralmente indica uma macro de anotação codificado incorretamente.
