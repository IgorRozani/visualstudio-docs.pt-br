# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28230](https://docs.microsoft.com/visualstudio/code-quality/c28230).  
  
Aviso C28230: O tipo de parâmetro não tem nenhum membro.  
  
 Este aviso indica que um argumento para uma anotação tenta acessar um `struct`, `class`, ou `union` e o membro nomeado não existe.
