# <a name="c28211"></a>C28211
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28211](https://docs.microsoft.com/visualstudio/code-quality/c28211).  
  
Aviso C28211: nome do contexto estático esperado para SAL_context  
  
 Este aviso indica que o operando para o `_Static_context_` anotação deve ser o nome de um contexto definido de ferramenta. Esse aviso nunca deve ocorrer em uso normal.
