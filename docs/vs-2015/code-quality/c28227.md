# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28227](https://docs.microsoft.com/visualstudio/code-quality/c28227).  
  
Aviso C28227: Anotação não pode ser também um PrimOp (consulte a declaração anterior)  
  
 Este aviso indica que a anotação nomeada está sendo declarada como uma anotação comum e também foi declarada anteriormente como um PrimOp. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
