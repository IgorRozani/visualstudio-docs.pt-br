# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225).  
  
Aviso C28225: não encontrou o número correto de parâmetros necessários na anotação  
  
 Este aviso indica que a anotação nomeada é usada com o número incorreto de parâmetros. Essa não é possível se as macros padrão usadas para anotações; Esse aviso não é esperado para ser visto no uso típico.
