# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
Aviso C28216: A _Check_return\_ anotação só se aplica a pós-condições para o parâmetro de função específica.  
  
 O `_Check_return_` anotação foi aplicada em um contexto diferente de `__post`; ele pode ser necessário um `__post` (ou `__drv_out`) modificador ou ele pode ser colocado incorretamente.
