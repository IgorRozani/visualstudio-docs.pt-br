[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C1252](https://docs.microsoft.com/visualstudio/code-quality/c1252).  
  
Aviso C1252: dependência Circular ou ausente entre plug-ins: requer GUID  
  
 A ferramenta de análise de código relata esse aviso quando há um erro interno com dependências de plug-in, não no código que está sendo analisado.
