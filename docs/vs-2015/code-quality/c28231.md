# <a name="c28231"></a>C28231
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28231](https://docs.microsoft.com/visualstudio/code-quality/c28231).  
  
Aviso C28231: anotação só é válida em matriz  
  
 Este aviso indica que um argumento para uma anotação deve ser uma matriz, e algum outro tipo foi encontrado.
