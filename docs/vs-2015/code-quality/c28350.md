# <a name="c28350"></a>C28350
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [C28350](https://docs.microsoft.com/visualstudio/code-quality/c28350).  
  
Aviso C28350: A anotação \<anotação > descreve uma situação que não é aplicável condicionalmente.  
  
 Normalmente, esse aviso é gerado quando uma anotação é aplicada em que o tipo de C/C++ está sendo inspecionado.
