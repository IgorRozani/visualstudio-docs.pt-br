# <a name="spanspan-destructor"></a>Destruidor span::~span
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [span:: ~ span destruidor](https://docs.microsoft.com/visualstudio/profiling/span-tilde-span-destructor).  
  
Destrói o objeto `span` e libera seus recursos.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
~span();  
```  
  
## <a name="requirements"></a>Requisitos  
 **Cabeçalho:** cvmarkersobj.h  
  
 **Namespace:** Concurrency::diagnostic
