# <a name="tier-interaction-views"></a>Exibições de interação da camada
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [exibições de interação de camada](https://docs.microsoft.com/visualstudio/profiling/tier-interaction-views).  
  
A criação de perfil de interação de camada fornece informações adicionais sobre os tempos de execução de funções em aplicativos de várias camadas que se comunicam com os bancos de dados por meio de [!INCLUDE[vstecado](../includes/vstecado-md.md)]. Os dados são coletados apenas para chamadas de função síncronas. É possível incluir dados de interação de camada em todos os métodos de criação de perfil.
