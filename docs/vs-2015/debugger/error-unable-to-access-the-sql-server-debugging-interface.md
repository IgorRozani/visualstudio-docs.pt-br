# <a name="error-unable-to-access-the-sql-server-debugging-interface"></a>Erro: não é possível acessar a interface de depuração do SQL Server
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [erro: não é possível acessar a interface de depuração do SQL Server](https://docs.microsoft.com/visualstudio/debugger/error-unable-to-access-the-sql-server-debugging-interface).  
  
Essa mensagem ocorre quando você tenta depurar o SQL que em um computador onde a depuração do SQL não está habilitada.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Ver [Configurando a depuração SQL](http://msdn.microsoft.com/en-us/3db09e68-edcc-42de-9c22-4e97cfd55ab3).
