# <a name="idiasymbolgetcallingconvention"></a>IDiaSymbol::get_callingConvention
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [idiasymbol:: Get_callingconvention](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-callingconvention).  
  
Retorna um indicador de uma convenção de chamada de métodos.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp#  
HRESULT get_callingConvention (   
   DWORD* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pRetVal`  
 [out] Retorna um valor da [enumeração CV_call_e](../../debugger/debug-interface-access/cv-call-e.md) convenção de chamada de enumeração que especifica um método.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retornará `S_OK`; caso contrário, retorna `S_FALSE` ou um código de erro.  
  
> [!NOTE]
>  Um valor de retorno `S_FALSE` significa que a propriedade não está disponível para o símbolo.
