# <a name="idiaenumsymbolsreset"></a>IDiaEnumSymbols::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [idiaenumsymbols:: Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumsymbols-reset).  
  
Redefine uma sequência de enumeração para o início.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp#  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valor de retorno  
 Retorna `S_OK`.
