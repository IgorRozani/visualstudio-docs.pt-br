# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Erro: a depuração de modo misto só é suportada durante o uso do Microsoft .NET Framework 2.0 ou superior
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [erro: depuração de modo misto é suportado apenas quando o Microsoft .NET Framework 2.0 ou superior](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-is-supported-only-when-using-microsoft-dotnet-framework-2-0-or-greater).  
  
Para depurar código nativo e gerenciado misto, você deverá ter o [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] versão 2.0, 3.0. 3.5 ou 4.0. Depuração de modo misto com versões anteriores do [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] não tem suporte.  
  
### <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Atualizar o [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] para a versão 2.0, 3.0, 3.5 ou 4.0.
