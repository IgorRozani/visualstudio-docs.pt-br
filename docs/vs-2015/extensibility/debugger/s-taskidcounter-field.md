# <a name="staskidcounter-field"></a>Campo s_taskIdCounter
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [campo s_taskIdCounter](https://docs.microsoft.com/visualstudio/extensibility/debugger/s-taskidcounter-field).  
  
O próximo identificador disponível para um <xref:System.Threading.Tasks.Task> objeto.  
  
 **Namespace:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **Assembly:** mscorlib (em mscorlib. dll)  
  
 Porque você não pode acessar esse membro interno do .NET Framework, a sintaxe a seguir é fornecida em comum Intermediate Language (CIL).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
.field static assembly int32 s_taskIdCounter  
```
