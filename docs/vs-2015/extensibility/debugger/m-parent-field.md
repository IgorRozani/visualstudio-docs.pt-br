# <a name="mparent-field"></a>Campo m_Parent
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [campo m_parent](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-parent-field).  
  
O campo de suporte para o <xref:System.Threading.Tasks.Task?displayProperty=fullName> propriedade pai.  
  
 **Namespace:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **Assembly:** mscorlib (em mscorlib. dll)  
  
 Porque você não pode acessar esse membro interno do .NET Framework, a sintaxe a seguir é fornecida em comum Intermediate Language (CIL).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
