# <a name="idebugprocess2getprocessid"></a>IDebugProcess2::GetProcessId
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [IDebugProcess2::GetProcessId](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugprocess2-getprocessid).  
  
Obtém o GUID para este processo.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp#  
HRESULT GetProcessId(  
   GUID* pguidProcessId  
);  
```  
  
```csharp  
int GetProcessId(  
   out Guid pguidProcessId  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pguidProcessId`  
 [out] Retorna o GUID para este processo.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retornará `S_OK`; caso contrário, retorna um código de erro.  
  
## <a name="remarks"></a>Comentários  
 O identificador global exclusivo (GUID) identifica esse processo de todos os outros processos em execução no sistema.
