# <a name="project-properties-reference"></a>Referência de propriedades de projeto
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [referência de propriedades do projeto](https://docs.microsoft.com/visualstudio/ide/reference/project-properties-reference).  
  
  
Saiba mais sobre como configurar e personalizar as propriedades do projeto.
