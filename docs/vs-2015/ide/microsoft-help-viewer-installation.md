# <a name="microsoft-help-viewer-installation"></a>Instalação do Microsoft Help Viewer
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

A versão mais recente deste tópico pode ser encontrada em [instalação do Visualizador de Ajuda do Microsoft](https://docs.microsoft.com/visualstudio/ide/microsoft-help-viewer-installation).  
  
Vários produtos de exibem a Ajuda no Microsoft Help Viewer. As instruções de instalação variam com base no produto que instalou o Help Viewer.  
  
## <a name="locating-setup-information"></a>Localizando informações de instalação  
 Para obter informações específicas para seu produto, consulte o arquivo Leiame.
