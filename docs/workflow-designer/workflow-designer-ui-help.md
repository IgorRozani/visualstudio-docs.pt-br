# <a name="workflow-designer-ui-help"></a>Ajuda de Designer de Fluxo de Trabalho interface de usuário

Esta seção contém os tópicos de ajuda sensível ao contexto do Designer de fluxo de trabalho do Windows. Ao trabalhar no Visual Studio 2010, você pode acessar esses tópicos da interface do usuário, pressionando **F1** quando uma caixa de diálogo do Designer de fluxo de trabalho é aberta.

Esta seção também contém vários tópicos preocupados com navegação e mensagens de erro.
