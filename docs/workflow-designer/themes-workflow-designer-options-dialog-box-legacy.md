# Themes, Workflow Designer, Options Dialog Box (Legacy)
Este tópico descreve como usar a caixa de diálogo **Temas, Designer de Fluxo de Trabalho, opções** em [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)]herdado.  Use [!INCLUDE[wfd2](../workflow-designer/includes/wfd2_md.md)] herdado quando você precisa definir como alvo [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] ou [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)].  
  
 Um tema define o plano de fundo e de primeiro plano as cores, estilos, ícones, e outros elementos visuais de um fluxo de trabalho.  Você pode salvar temas para reutilização por outros fluxos de trabalho.  Você pode selecionar temas personalizados padrão e salvos usando a caixa de diálogo **Temas, Designer de Fluxo de Trabalho, opções** .  Para abrir esta caixa de diálogo, clique **Selecione tema personalizado** no menu de **Fluxo de Trabalho** .  Ou, clique com o botão direito do mouse na superfície de design de fluxo de trabalho e clique **Selecione o tema personalizado**.  
  
 A tabela a seguir descreve os elementos de \(UI\) de interface de usuário da caixa de diálogo **Temas, Designer de Fluxo de Trabalho, opções** .  
  
|Elemento da Interface do Usuário|Descrição|  
|--------------------------------------|---------------|  
|**Novo**|Clique para abrir [Theme Configuration Dialog Box \(Legacy\)](../workflow-designer/theme-configuration-dialog-box-legacy.md) para criar um novo tema.|  
|**Modifique…**|Clique para abrir [Theme Configuration Dialog Box \(Legacy\)](../workflow-designer/theme-configuration-dialog-box-legacy.md) para alterar o tema selecionado.|  
|**Procure**|Clique para procurar um arquivo de tema de fluxo de trabalho, que usa uma extensão de nome de arquivo .wtm.|  
|**Selecione um tema**|Título de coluna do nome do tema de lista dos temas.  Selecione um tema da lista, clique em **Modificar** para editar, ou clique em **OK** para usar o tema selecionado para o fluxo de trabalho.|  
|**Descrição**|Título de coluna de descrição de tema de lista dos temas.|  
|**Local**|Título de coluna do local de tema de lista dos temas.|
