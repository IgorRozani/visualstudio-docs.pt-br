# Developing Workflow Applications Targeting the .NET 3.0 or .NET 3.5 Framework
[!INCLUDE[vs2010](../modeling/includes/vs2010_md.md)] fornece [!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] para criar aplicativos de [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] que direcionam [!INCLUDE[netfx40_long](../workflow-designer/includes/netfx40_long_md.md)].  O designer usa a interface do usuário e de [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] para a compilação e a depuração de aplicativos gráficos de fluxo de trabalho.
