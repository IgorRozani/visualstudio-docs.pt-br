# Control Flow Activity Designers
[!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] inclui um número o sistema forneceu as atividades que você pode usar para construir os fluxos de trabalho.  Esta seção contém sistema forneceu as atividades usadas para controlar o fluxo em um fluxo de trabalho.  Os tópicos a seguir descrevem essas atividades e fornecem orientação sobre como usá\-los.  
  
## Nesta seção  
 [DoWhile](../workflow-designer/dowhile-activity-designer.md)  
 Executa a atividade contida no corpo pelo menos uma vez, até que uma condição especificada for avaliada como **true**.  
  
 [ForEach\<T\>](http://msdn.microsoft.com/pt-br/a680cddd-2760-497a-b27b-c023fcbc6f33)  
 Executa a atividade contida no corpo para cada item em uma coleção especificada.  
  
 [If](../workflow-designer/if-activity-designer.md)  
 Avalia uma condição e executa uma atividade dependendo dos resultados da avaliação.  
  
 [Parallel](../workflow-designer/parallel-activity-designer.md)  
 Executa uma coleção de filhos atividades simultaneamente.  
  
 [ParallelForEach\<T\>](../workflow-designer/parallelforeach-t-activity-designer.md)  
 Enumera os elementos de uma coleção e executa uma declaração inserido para cada elemento da coleção paralelamente  
  
 [Pick](../workflow-designer/pick-activity-designer.md)  
 Executa um dos vários ramificações em resposta a um evento que fornece o controle de fluxo baseado.  
  
 [PickBranch](../workflow-designer/pickbranch-activity-designer.md)  
 Fornece um caminho potencial de execução dentro de uma atividade de <xref:System.Activities.Statements.Pick> .  
  
 [Sequence](../workflow-designer/sequence-activity-designer.md)  
 Contém uma coleção ordenada de atividades filhos que executam em ordem.  
  
 [Switch\<T\>](http://msdn.microsoft.com/pt-br/ce1aa634-c4db-4475-a1c8-a88478a57212)  
 Avalia uma expressão especificada e executa a atividade de uma coleção de atividades cuja chave associado corresponde ao valor obtido de avaliação.  
  
 [While](../workflow-designer/while-activity-designer.md)  
 Executa a atividade contida no corpo enquanto uma condição especificada avaliar a **true**.
