# <a name="error-handling-activity-designers"></a>Designer de atividade de tratamento de erro

O designer de atividade de tratamento de erros é usado para gerenciar o tratamento de erros no Designer de fluxo de trabalho do Windows. Os tópicos contidos nesta seção fornecem orientação sobre como usar a atividade de Designer de fluxo de trabalho e designers de modelo.
