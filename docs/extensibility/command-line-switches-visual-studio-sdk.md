# <a name="command-line-switches-visual-studio-sdk"></a>Opções de linha de comando (SDK do Visual Studio)

Os VSPackages também pode disponibilizar seus próprios switches na linha de comando Devenv. Os tópicos a seguir fornecem informações sobre como usar as opções de linha de comando no IDE.
