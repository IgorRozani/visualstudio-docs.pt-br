# <a name="reference-visual-studio-debugging-apis"></a>Referência (Visual Studio APIs de depuração)
As referências do Visual Studio SDK depuração descrevem o modelo básico para a arquitetura de tempo de execução e a funcionalidade do depurador e seus componentes. A documentação faz suposições menor número possível sobre os componentes e interfaces descreve, exceto o comportamento define por meio de interfaces.  
  
 O objetivo da seção de referência é fornecer suporte a diversas um conjunto de sistemas operacionais, linguagens e tempos de execução possível.
