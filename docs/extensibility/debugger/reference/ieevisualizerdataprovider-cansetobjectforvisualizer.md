# <a name="ieevisualizerdataprovidercansetobjectforvisualizer"></a>IEEVisualizerDataProvider::CanSetObjectForVisualizer
Este método determina se o visualizador pode ter o objeto de dados representa atualizado.  
  
## <a name="syntax"></a>Sintaxe  
  
```cpp  
HRESULT CanSetObjectForVisualizer(  
   BOOL* b  
);  
```  
  
```c#  
int CanSetObjectForVisualizer(  
   out int b  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `b`  
 [out] Diferente de zero (`TRUE`) se o objeto sobre o visualizador pode ser atualizado, zero (`FALSE`) se não for.  
  
## <a name="return-value"></a>Valor de retorno  
 Se for bem-sucedido, retornará `S_OK`; caso contrário, retorna um código de erro.  
  
## <a name="remarks"></a>Comentários  
 Um objeto pode não ser alterável se ele está associado a memória somente leitura, por exemplo.
