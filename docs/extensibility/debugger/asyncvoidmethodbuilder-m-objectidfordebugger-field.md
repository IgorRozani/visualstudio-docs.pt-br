# <a name="asyncvoidmethodbuildermobjectidfordebugger-field"></a>Campo AsyncVoidMethodBuilder.m_objectIdForDebugger
Representa o objeto de inicialização ociosa usado pelo depurador para identificar exclusivamente este construtor.  
  
 **Namespace:**<xref:System.Runtime.CompilerServices?displayProperty=fullName></xref:System.Runtime.CompilerServices?displayProperty=fullName>  
  
 **Assembly:** mscorlib (em mscorlib. dll)  
  
 Porque você não pode acessar esse membro interno do .NET Framework, a sintaxe a seguir é fornecida idioma intermediário comum (CIL).  
  
## <a name="syntax"></a>Sintaxe  
  
```  
.field private object m_objectIdForDebugger  
```
