# <a name="document-position"></a>Posição do documento
Em [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] depuração, um **de posição de documento**:  
  
-   Fornece uma abstração de uma posição em um arquivo de origem como conhecidos para o IDE. Na maioria dos idiomas hoje, uma posição de documento pode ser pensada como uma posição em um arquivo de origem.  
  
-   Descreve uma posição em um documento de origem para um mecanismo de depuração.  
  
-   É implementado por um [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) interface.
