O delegado que representa o código para executar no <xref:System.Threading.Tasks.Task>objeto.</xref:System.Threading.Tasks.Task>  
  
 **Namespace:**<xref:System.Threading.Tasks?displayProperty=fullName></xref:System.Threading.Tasks?displayProperty=fullName>
