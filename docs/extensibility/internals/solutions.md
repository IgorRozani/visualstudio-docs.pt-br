# <a name="solutions"></a>SoluçõesD;
Uma solução é um agrupamento de um ou mais projetos que trabalham juntos para criar um aplicativo. As projeto e informações de status que pertencem à solução são armazenados em dois arquivos de solução, um arquivo de solução com base em texto (. sln) e um arquivo de opção (. suo) solução binário do usuário.
