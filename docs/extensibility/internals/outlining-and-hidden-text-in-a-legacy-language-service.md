# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>Estrutura de tópicos e ocultado o texto em um serviço de linguagem herdado
Estrutura de tópicos torna possível recolher uma sequência de linhas de texto em uma única linha. Por exemplo, em C++ todos os métodos podem ser recolhidos em uma única linha, mostrando apenas a assinatura do método. Texto oculto é a sequência de linhas de texto que pode ser mostrado ou ocultado.
