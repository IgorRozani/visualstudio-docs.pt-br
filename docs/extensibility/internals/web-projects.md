# <a name="web-projects"></a>Projetos Web
Um sistema de projeto da Web é um sistema de projeto que cria projetos da Web. Projetos da Web por sua vez criar aplicativos Web. Você pode programar uma página da Web usando linguagens de programação tradicionais, como [!INCLUDE[vbprvb](../../code-quality/includes/vbprvb_md.md)] ou [!INCLUDE[csprcs](../../data-tools/includes/csprcs_md.md)], para coletar e processar informações de um usuário, armazene-o em um banco de dados e assim por diante. Páginas da Web com código associado são chamadas de smart páginas da Web.
