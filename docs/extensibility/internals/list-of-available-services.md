# <a name="list-of-available-services"></a>Lista de serviços disponíveis
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]e os seguintes serviços de suporte do SDK do Visual Studio. Alguns pacotes oferecem seus próprios serviços que não estão listados aqui — por exemplo, serviços de linguagem não têm um único serviço GUID. Você deve usar o nome do idioma para localizar o GUID do serviço de idioma no registro.  
  
 Use os GUIDs de serviço listados aqui ou obtido de outra origem (por exemplo, serviços de linguagem) para obter a interface primária ou interfaces mostrados com cada serviço.
