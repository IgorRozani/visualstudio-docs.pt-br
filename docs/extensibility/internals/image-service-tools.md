# <a name="image-service-tools"></a>Ferramentas de serviço de imagem
O SDK do VS inclui várias ferramentas para ajudar os desenvolvedores de extensão a usar o serviço de imagem do Visual Studio.  
  
-   O [de recursos de manifesto](../../extensibility/internals/manifest-from-resources.md) ferramenta usa uma lista de recursos de imagem (arquivos. png ou. XAML) e gera um arquivo de manifesto de imagem.  
  
-   O [manifesto para código](../../extensibility/internals/manifest-to-code.md) ferramenta usa um arquivo de manifesto de imagem e gera um arquivo de conteúdo adicional a ser usado em arquivos C++, c#, VB ou. VSCT.  
  
-   O [Visualizador da biblioteca de imagens](../../extensibility/internals/image-library-viewer.md) pode carregar, editar e pesquisar os manifestos de imagem.
