# <a name="extending-properties-and-the-property-window"></a>Estendendo propriedades e a janela de propriedades
O **propriedades** janela é usada para exibir e editar as propriedades de tempo de design e os eventos dos objetos selecionados.  
  
 Quando você seleciona um objeto em [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], as propriedades públicas do objeto são exibidos no **propriedades** janela. Para obter mais informações, consulte [visão geral da janela propriedades](../extensibility/internals/properties-window-overview.md).
