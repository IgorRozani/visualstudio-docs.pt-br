# <a name="extensibility-in-visual-studio"></a>Extensibilidade do Visual Studio
O Visual Studio fornece muitas opções de extensibilidade. Você pode criar suas próprias SDKs, use o Windows, Windows Phone e SDKs do Azure (que são instalados como parte do Visual Studio) e estender o próprio Visual Studio.  
  
## <a name="extend-visual-studio"></a>Estenda o Visual Studio  
 Você pode usar o SDK do Visual Studio para estender a praticamente qualquer parte do Visual Studio: comandos, menus, windows, editores e projetos. Para obter mais informações, consulte [Visual Studio SDK](../extensibility/visual-studio-sdk.md).  
  
## <a name="create-your-own-sdks"></a>Criar suas próprias SDKs  
 Descubra como criar, empacotar e implantar suas próprias SDKs da plataforma e extensão: [criando um Kit de desenvolvimento de Software](../extensibility/creating-a-software-development-kit.md).  
  
## <a name="roslyn-extensibility"></a>Extensibilidade de Roslyn  
 Saiba como usar o Roslyn para extensibilidade: [extensibilidade .NET Compiler Platform ("Roslyn")](../extensibility/dotnet-compiler-platform-roslyn-extensibility.md).
