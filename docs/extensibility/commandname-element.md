# <a name="commandname-element"></a>Elemento CommandName
O `CommandName` elemento Especifica o texto que aparece na categoria de teclado no **opções** caixa de diálogo e no **comandos** lista no **personalizar** caixa de diálogo.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## <a name="attributes-and-elements"></a>Atributos e elementos  
 As seções a seguir descrevem atributos, elementos filho e elementos pai.  
  
### <a name="attributes"></a>Atributos  
 Nenhum.  
  
### <a name="child-elements"></a>Elementos filho  
 nenhuma.  
  
### <a name="parent-elements"></a>Elementos pai  
  
|Elemento|Descrição|  
|-------------|-----------------|  
|[Elemento de cadeias de caracteres](../extensibility/strings-element.md)|Agrupa elementos de texto, como `ButtonText` e `CommandName`.|
